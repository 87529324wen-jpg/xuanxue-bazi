## 分析流程（7步法）

```
第1步: 排盘
  输入: 出生年月日时(公历/农历) + 性别 + 出生地
  输出: 四柱八字 + 纳音 + 生肖
  工具: sxtwl库(寿星万年历, 权威)

第2步: 太岁检查
  检查四柱地支与流年地支的关系
  值太岁/冲太岁/刑太岁/害太岁/破太岁
  严重度排序: 冲太岁 > 值太岁+自刑 > 三刑 > 害 > 破

第3步: 神煞
  天乙贵人/文昌/将星/华盖/驿马/羊刃/桃花/十干禄
  /天德/月德/空亡/魁罡
  注意: 神煞之间有交互关系(将星×华盖等)

第4步: 日主强弱
  月令旺相休囚死 + 帮身/克泄耗力量对比
  判断: 身旺/身弱/中和

第5步: 用神喜忌
  身旺 → 克泄耗为用(官杀/财星/食伤)
  身弱 → 生扶为用(印星/比劫)
  调候用神: 穷通宝鉴——按日主天干+月份查用神
  (如: 丙火生冬月需壬甲, 甲木生春月需丙癸等)

第6步: 十神关系分析
  这是最关键的一步!
  不是只列十神定义, 要看十神之间的组合关系:
  - 贪财坏印 / 印化杀 / 官印相生
  - 官杀混杂 / 食神制杀 / 伤官见官
  - 劫财夺财 / 食伤生财 / 财官相生
  - 比肩重重 / 印绶太重
  每个关系引用经典原文

第7步: 大运+流年
  大运排列 + 起运年龄
  流年与原局天干地支作用
  大运+流年叠加效应
  逐月分析
```

## 技术要点（踩坑记录）

### 排盘必须用 sxtwl
- 系统Python: `/usr/bin/python3` (3.9.6)
- 安装: `pip install sxtwl` (系统pip, 不是sandbox)
- 不能用JDN公式自己算日柱, 基准日期容易错
- sxtwl是寿星万年历, 最权威的中文万年历库
- API: `sxtwl.fromSolar(y,m,d)`, `sxtwl.fromLunar(y,m,d,False)`
- `day.getYearGZ(False)` / `day.getMonthGZ()` / `day.getDayGZ()`
- `sxtwl.getShiGz(day_gan_idx, hour)` 获取时柱

### 太岁是流年分析第一步
- 第一版分析漏了太岁, 被用户指出"庚午生人今年刑太岁很严重"
- 检查顺序: 值太岁 > 冲太岁 > 自刑 > 三刑 > 害太岁 > 破太岁
- 年支犯太岁影响: 父母/根基/祖业/内心状态
- 日支犯太岁影响: 配偶/合作/内心

### 十神关系比单个十神重要
- 不是只列"偏财""七杀"这种标签
- 要分析组合: 贪财坏印/官杀混杂/劫财夺财/食神制杀/伤官见官/印化杀
- 每个组合有经典原文支撑
- 组合之间的矛盾是命局核心矛盾

### 太岁检查必须查自刑+值太岁叠加
- 午午/酉酉/辰辰/亥亥 既是值太岁也是自刑, 两者同时成立, 严重度加倍
- 用户指出"庚午生人今年刑太岁很严重"后才发现遗漏
- 太岁检查完整清单: 六冲/自刑/值太岁/三合/六合/三刑/六害/六破

## 合婚分析流程

无时辰也能做（三柱合婚），有时辰更准（四柱合婚）。

### 合婚检查清单（按优先级）

**第一层：日柱（最关键）**
- 日干关系: 查十神对应（男看女是什么十神，女看男是什么十神）
  - 正财/正官 = 最正的夫妻配（男正财=妻星，女正官=夫星）★★★★★
  - 偏财/七杀 = 偏缘，有吸引力但不稳定 ★★★
  - 比肩/劫财 = 同频型，平等但可能竞争 ★★★
  - 食神/伤官 = 有才华吸引但有挑战 ★★★
  - 印星 = 一方给另一方安全感 ★★★★
- 日支关系: 六合最佳（如卯戌合、午未合）→ 夫妻宫有情 ★★★★★
- 天干五合: 丙辛合/丁壬合/戊癸合/甲己合/乙庚合 → 核心本质互相吸引 ★★★★★

**第二层：年柱**
- 年柱六合（如子丑合）→ 家庭层面气场合
- 纳音关系: 同纳音=同类，相生=吉，相克=凶
- 生肖: 六合/三合为吉（鼠牛/虎猪/兔狗/龙鸡/蛇猴/马羊）

**第三层：月柱**
- 月柱天干五合（如甲己合）→ 事业/社会层面合
- 月支关系: 冲/害/破要标注

**第四层：时柱（有时辰才查）**
- 时柱关系 → 子女/晚年相处模式

**第五层：五行互补**
- 统计双方五行分布
- 一方缺的恰好是另一方多的 → 互补 ✓
- 双方都缺同一行（如都缺水）→ 两人都不"冷静"，容易一起燥 ✗

**第六层：跨柱深层关系**
- 三合局: 如男方有午+戌，女方有寅 → 寅午戌三合火局
- 六冲/六害/六破: 跨柱也要检查
- 伏吟: 一方某柱=另一方某柱 → 该领域共鸣或重复

### 合婚结论模板

```
表面合度: X柱有合/X柱有冲/X柱无关系
日干配对: 男X女X = XX/XX配（正配/偏配/比肩等）
五行互补: 她有X你缺✓ / 都缺X✗
深层关系: 三合/六冲/伏吟等
类型判断: 互补型/同频型/命定型/相处舒服型
风险点: XX
建议: XX
```

### 合婚经验（2026-04-21验证）

实测三对合婚对比:
- 三柱六合+五行互补但日干非正配 → "相处舒服型"，适配度高但缺命定感
- 四柱零合+五行不补+比肩关系 → "同频共鸣型"，来电但容易互烧
- 表面有破但日干丙辛合+正财正官配 → "命中注定型"，有摩擦但缘最深

**结论: 日干正配+日干五合 > 三柱六合 > 五行互补 > 表面无冲无合**
合的数量不如合的质量重要。一个丙辛合比三个六合更说明问题。

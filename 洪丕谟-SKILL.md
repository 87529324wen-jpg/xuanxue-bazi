## 八字排盘方法

**必须使用 sxtwl 库**（系统Python3已安装），不可用JDN手动推算（会出错）。

```python
import sxtwl
TIAN_GAN = ["甲","乙","丙","丁","戊","己","庚","辛","壬","癸"]
DI_ZHI = ["子","丑","寅","卯","辰","巳","午","未","申","酉","戌","亥"]

day = sxtwl.fromSolar(year, month, day)
yGz = day.getYearGZ(False)   # 以立春为界
mGz = day.getMonthGZ()
dGz = day.getDayGZ()
sGz = sxtwl.getShiGz(dGz.tg, hour)  # hour=19 for 戌时

# 四柱
年柱 = TIAN_GAN[yGz.tg] + DI_ZHI[yGz.dz]
月柱 = TIAN_GAN[mGz.tg] + DI_ZHI[mGz.dz]
日柱 = TIAN_GAN[dGz.tg] + DI_ZHI[dGz.dz]
时柱 = TIAN_GAN[sGz.tg] + DI_ZHI[sGz.dz]
日主 = TIAN_GAN[dGz.tg]
```

大运：阳男顺排，阴男逆排。从月柱起推。起运=生日到下一节气天数÷3（取整）。

真太阳时：北京时间-(当地经度-120°)×4分钟。哈尔滨126.6°差约26分钟。

## 排盘工具与方法

### 日柱计算
**必须使用 sxtwl 库**（寿星万年历），不要用JDN手动推算。
原因：JDN方法需要精确的参考日期，容易出错。sxtwl是权威中文万年历库。
```python
# /usr/bin/python3 环境下可用
import sxtwl
day = sxtwl.fromSolar(1996, 6, 18)
dGz = day.getDayGZ()
TIAN_GAN[dGz.tg] + DI_ZHI[dGz.dz]  # 日柱
```
已验证：sandbox Python无sxtwl，需用 `/usr/bin/python3`。

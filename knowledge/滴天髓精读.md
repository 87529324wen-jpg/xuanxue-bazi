# 滴天髓精读 — 哲学深度与推理心法

## 一、总纲

**"欲识三元万法宗，先观帝载与神功"**
- 三元 = 天元（天干）+ 地元（地支）+ 人元（藏干）
- 万法之宗 = 阴阳五行

**"戴天覆地人为贵，顺则吉兮凶则悖"**
- 顺 = 天干地支接续相生
- 悖 = 反克为害
- 天干衰弱地支生之 = 有情 = 吉
- 天干衰弱地支克之 = 无情 = 凶

## 二、十干性情（核心！）

### 阳干
**甲木参天，脱胎要火。春不容金，秋不容土。火炽乘龙，水宕骑虎。地润天和，植立千古。**
- 甲为纯阳之木，体本坚固，参天雄壮
- 春初木嫩气寒，得火而发荣；仲春旺极，宜泄其菁英
- 春不容金：衰金克旺木，木坚金缺
- 秋不容土：秋土虚薄，不能培木之根

**丙火猛烈，欺霜侮雪。能煅庚金，逢辛反怯。土众成慈，水猖显节。**
- 丙为纯阳之火，其势猛烈
- 能煅庚金：遇强暴而施克伐
- 逢辛反怯：合柔顺而寓和平
- 己土卑湿之体能收元阳之气；戊土高燥见丙火而焦坼

**戊土固重，既中且正。静翕动辟，万物司命。**
- 戊为阳土，高燥之体
- 水润物生，火燥物病
- 若在艮坤，怕冲宜静

**庚金刚健，刚又逢刚，势不两立。**
- 庚为阳金，秋天肃杀之气
- 用丁火锻炼方成器皿

**壬水通源，能泄金气。刚中之德，周流不滞。**
- 壬为阳水，刚中之德
- 能制暴烈之火

### 阴干
**乙木虽柔，刲羊解牛。怀丁抱丙，跨凤乘猴。虚湿之地，骑马亦忧。藤萝系甲，可春可秋。**
- 乙为阴木，柔韧之质
- 坐丑未制柔土（刲羊解牛）
- 有丙丁则虽生申酉月不畏（跨凤乘猴）
- 藤萝系甲 = 甲寅多见，弟从兄义，四季皆可

**丁火柔中，内性昭融。抱乙而孝，合壬而忠。旺而不烈，衰而不穷，如有嫡母，可秋可冬。**
- 丁为阴火，柔中之性
- 抱乙而孝 = 使辛金不伤乙木
- 合壬而忠 = 暗使戊土不伤壬水
- 有甲木则倚之不灭，焰至无穷

**己土卑湿，中正蓄藏。不愁木盛，不畏水狂。**
- 己为阴土，卑湿之体
- 能收元阳之气（与戊土高燥相反）

**辛金清淑，柔弱之质。最畏土多，乐水之盈。**
- 辛为阴金，珠玉之质
- 壬水刚中之德能制暴烈之火
- 辛金柔软之物，明作合而相亲，暗化水而相济

**癸水至弱，达于天津。得龙而运，功化斯神。**
- 癸为阴水，至阴之精
- 万物莫不由此而生，得此而茂

## 三、五阳五阴之别（从格判断核心）

**"五阳从气不从势，五阴从势无情义"**

- 阳干性刚健 → 不畏财煞，有恻隐之心 → **阳干不易从**
- 阴干性柔顺 → 见势忘义，有鄙吝之心 → **阴干易从**

### 实战意义
- 阳日主即使身弱，只要有一丝根气，就不轻易论从格
- 阴日主身弱无根，见旺势则从，从则彻底
- **"柔能克制刚，刚不能克柔"**

## 四、衰旺真机

### 旺相休囚
- **将来者进 = 相**（最佳状态）
- **进而当令 = 旺**（极盛，退反速）
- **功成者退 = 休**
- **退而无气 = 囚**

**"相妙于旺，旺则极盛之物，其退反速，相则方长"**

### 通根层级
- 长生禄旺 = 根之重者
- 墓库余气 = 根之轻者
- **"干多不如根重，理固然也"**
- 天干得一比肩，不如地支得一墓库
- 得二比肩，不如得一余气
- 得三比肩，不如得一长生禄旺

### 进气退气
- **甲木进气于丑月**（冬季将尽，春木将至）
- **庚金进气于未月**（夏季将尽，秋金将至）
- 进气者虽弱不弱，退气者虽旺不旺

## 五、中和之道

**"既识中和之正理，而于五行之妙，思过半矣"**

### 中和标准
- 五行和者，一世无灾
- 和 = 全者宜全，缺者宜缺，生者宜生，克者宜克
- 不是简单的"平衡"，而是"各得其所"

### 偏枯之害
- 火烈而性燥者 → 遇金水之激 → 灾祸
- 水奔而性柔者 → 当火土之神
- **"火燥而烈，只可用湿土润之"**（丑辰）
- 丙火午月 → 丑时最佳（湿土收烈火）

## 六、疾病判断

### 五行与五脏
- 木 → 肝/胆
- 火 → 心/小肠
- 土 → 脾/胃
- 金 → 肺/大肠
- 水 → 肾/膀胱

### 忌神入五脏
**"忌神入五脏而病凶"**
- 忌木入土 → 脾病
- 忌火入金 → 肺病
- 忌土入水 → 肾病
- 忌金入木 → 肝病
- 忌水入火 → 心病

### 客神游六经
**"客神游六经而灾小"**
- 客神 = 忌神浮于天干
- 阳而虚露，易制易化，为灾必小
- 犹病之在表，外感易于发散

**"两气合而成象，象不可破也"**

- 木火各半 / 金水各半 / 水木各半 等
- **最怕破象**：如木火成象见金水
- 一路澄清，必位高禄重；中途混乱，恐职弃家倾

**"中和纯粹，则有恻隐、辞让、诚实之情；若偏枯混浊，太过不及，则有是非、乖逆、骄傲之性"**

- 火烈性燥 → 争躁如风火，顺其性千金不惜，逆其性一芥中分
- 八字纯阴 → 一味趋炎附势，心存损人利己

## 核心推理模式

### 推理模式1：从格判断
1. 阳日主 → 有一丝根气就不从 → 即使极弱也论正格
2. 阴日主 → 无根见旺势则从 → 从则彻底
3. 从格条件：日主无根 + 旺势一方独大 + 无印比帮身

### 推理模式2：进气退气
1. 看日主在月令的进退气状态
2. 进气 = 虽不当令但力量正在增长 → 不弱
3. 退气 = 虽当令但力量正在衰退 → 不旺

### 推理模式3：疾病判断
1. 找忌神 → 看忌神在天干还是地支
2. 地支忌神 = 入五脏 → 病重
3. 天干忌神 = 游六经 → 病轻
4. 看忌神对应的五行 → 定脏腑

### 推理模式4：性格判断
1. 看日主五行 → 基本性格
2. 看五行是否中和 → 性格是否端正
3. 看偏枯混浊程度 → 性格缺陷程度

| 申月 | 甲木生申月，庚金司令，壬水为辅助庚，庚不离壬壬不离庚，二者配合主贵。 |
| 酉月 | 甲木生酉月，庚辛金旺，丁火制庚最要，壬水滋甲为辅，有丁无壬贵而不富。 |
| 戌月 | 甲木生戌月，戊土当权，先庚后丁，壬水佐之，庚透天干为贵。 |
| 亥月 | 甲木生亥月，水旺木浮，庚金发水之源，戊土止水为要，无戊则水泛木漂。 |
| 子月 | 甲木生子月，寒冬水旺，先丁火解冻，次庚壬辅用，无丁则甲木冻僵。 |
| 丑月 | 甲木生丑月，湿土严寒，先庚劈甲，后丁温暖，丑中辛癸己，庚透则贵。 |

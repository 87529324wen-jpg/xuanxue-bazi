## 二、十二宫含义

| 宫位 | 代表 | 断法要点 |
|------|------|----------|
| 命宫 | 一生总格局、性格、长相 | 最重要！命宫主星决定基本格局 |
| 兄弟宫 | 兄弟姐妹、母亲、合伙人 | 看兄弟缘分和助力 |
| 夫妻宫 | 配偶特征、婚姻质量 | 配偶长相/性格/缘分深浅 |
| 子女宫 | 子女数量/质量、性生活 | 子女缘分、生育能力 |
| 财帛宫 | 赚钱方式、财运好坏 | 财来的方式（正财/偏财/横财）|
| 疾厄宫 | 健康、疾病、意外 | 哪种病、何时发病 |
| 迁移宫 | 出外运、社会关系 | 外出吉凶、贵人小人 |
| 奴仆宫/交友宫 | 下属、朋友、人际关系 | 用人情况、被帮助还是被拖累 |
| 官禄宫/事业宫 | 事业、工作、学业 | 适合什么行业、事业高度 |
| 田宅宫 | 房产、家庭环境 | 有没有房、家庭氛围 |
| 福德宫 | 精神世界、兴趣爱好 | 内心快乐与否、精神追求 |
| 父母宫 | 父亲、长辈、上司 | 与父母缘分、遗传 |

## 四、排盘要点

### 安命宫
以出生月（农历）和出生时辰查：
- 正月起子，顺数到生月
- 再从生月宫起子时，逆数到生时
- 所落之宫即为命宫

### 安身宫
与命宫对冲之位。

### 定五行局
以命宫天干地支的纳音定局：
- 水二局、木三局、金四局、土五局、火六局

### 安主星
以命宫地支 + 五行局 → 查紫微星位置
然后按固定规则排布其他十三主星

## 五、格局判断

### 吉格
- **紫府同宫**：紫微天府同宫，大富大贵
- **机月同梁**：天机太阴天同天梁会齐，公职/幕僚
- **日月并明**：太阳太阴在旺地，光明正大
- **府相朝垣**：天府天相夹命宫，贵格
- **火贪/铃贪**：火星/铃星与贪狼同宫，暴发之格

### 凶格
- **巨门化忌**：口舌是非极重
- **廉贞化忌**：感情大灾
- **羊陀夹忌**：化忌被羊刃陀罗夹，困顿
- **命无正曜**：命宫无十四主星，格局不稳

## 使用说明

本模块用于排盘后快速查阅。排盘需先确定：
1. 出生年月日时（农历）
2. 性别
3. 安命宫 → 安身宫 → 定局 → 安主星 → 安辅星 → 安四化

具体排盘方法见《紫微斗数精成》第二章。

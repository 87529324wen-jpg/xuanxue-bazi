# 手相速查

> 来源：《麻衣神相》《神相全编》《相理衡真》
> 用途：看手相时快速查阅各部位含义
> 用法：用户发手掌照片后，按以下框架逐项分析

---

## 一、手相基础

### 看手原则
- 男看左手（先天），女看右手（先天）
- 实际应两手都看：左先天（遗传/本质），右后天（努力/变化）
- 手掌自然张开，不紧绷，光线充足
- 先看整体（大小/厚薄/色泽），再看主线，再看辅助线

### 手型分类（五行手）

| 手型 | 特征 | 性格 | 五行 |
|------|------|------|------|
| 木型手 | 手掌长方、手指修长、指节明显 | 理性、思考型、适合学术/技术 | 木 |
| 火型手 | 手掌方形、手指尖细、掌色红润 | 热情、行动力强、适合领导/销售 | 火 |
| 土型手 | 手掌方厚、手指粗短、掌肉厚实 | 务实、稳重、适合实业/管理 | 土 |
| 金型手 | 手掌方圆、手指匀称、掌色白润 | 果断、有魄力、适合金融/法律 | 金 |
| 水型手 | 手掌圆润、手指纤细、掌软如绵 | 灵活、善交际、适合艺术/传媒 | 水 |

### 手掌分区（九宫）

| 宫位 | 位置 | 代表 |
|------|------|------|
| 乾宫 | 掌底靠小指侧 | 父亲、天命、晚年运 |
| 坎宫 | 掌底正中 | 祖业、根基、肾/泌尿 |
| 艮宫 | 掌底靠拇指侧 | 母亲、田宅、脾胃 |
| 震宫 | 拇指根部 | 兄弟、肝胆、行动力 |
| 巽宫 | 食指根部 | 财运、事业、胆识 |
| 离宫 | 中指根部 | 功名、心脏、官禄 |
| 坤宫 | 无名指+小指根部 | 子女、福德、母亲 |
| 兑宫 | 小指根部下方 | 妻妾(男)/夫星(女)、口才 |
| 明堂 | 掌心正中 | 智慧、心境、健康总汇 |

## 二、五大主线

### 1. 生命线（地纹）
**位置**：从食指与拇指之间出发，沿拇指球弧形向下至手腕

**含义**：生命力、体质、健康、寿命趋势（非寿命长短）

| 特征 | 解读 |
|------|------|
| 深长清晰、弧度大 | 体质好、精力充沛、生命力强 |
| 浅短模糊 | 体质偏弱、易疲劳、需注意养生 |
| 弧度大包过拇指球 | 精力旺盛、抗压能力强 |
| 弧度小靠近拇指 | 精力有限、适合安稳生活 |
| 有岛纹（线中圆形） | 对应时段有健康问题 |
| 有断裂 | 对应时段有重大变故/手术/意外 |
| 断裂后有重叠 | 变故后能恢复 |
| 起端有分叉 | 早年生活变化多 |
| 末端有分叉 | 晚年可能搬迁或多处居住 |

### 2. 智慧线（人纹）
**位置**：从生命线起点附近出发，横穿掌心

**含义**：思维方式、智力、判断力、性格倾向

| 特征 | 解读 |
|------|------|
| 长而平直 | 理性、逻辑强、务实、适合理工/分析 |
| 长而弯（向月丘） | 想象力丰富、感性、适合艺术/创意 |
| 短而直 | 务实、直接、不绕弯子 |
| 末端分三叉 | 思维全面、多角度思考 |
| 末端上翘 | 有野心、追求更高目标 |
| 与生命线起点重合长 | 谨慎、缺乏安全感、做事犹豫 |
| 与生命线起点分离 | 独立、果断、有主见 |
| 有岛纹 | 对应时段思维混乱/迷茫 |
| 链状纹 | 注意力不集中、想法多但执行弱 |

### 3. 感情线（天纹）
**位置**：从小指下方出发，横穿掌心至食指或中指下方

**含义**：感情态度、婚姻运、心脏健康

| 特征 | 解读 |
|------|------|
| 长至食指下方 | 感情专一、重精神交流、择偶标准高 |
| 长至中指下方 | 感情务实、重现实条件、不太浪漫 |
| 长至食指与中指之间 | 感情平衡、既重感觉也重条件 |
| 弯曲上翘 | 热情、表达力强、感情丰富 |
| 平直 | 理性控制感情、不善表达 |
| 末端分叉 | 感情丰富、可能多段感情 |
| 末端分两叉（人字） | 感情收尾圆满 |
| 末端分三叉 | 感情生活多姿多彩 |
| 有岛纹 | 对应时段感情有困扰/第三者 |
| 链状纹 | 感情波折多、容易多情 |
| 感情线短 | 感情淡薄、不善表达爱意 |
| 感情线断裂 | 对应时段有重大感情变故 |
| 有羽状纹向下 | 在感情中付出多、容易受伤 |

### 4. 命运线（事业线/玉柱纹）
**位置**：从掌底向上延伸至中指根部（不一定人人都有）

**含义**：事业运、人生方向、社会地位

| 特征 | 解读 |
|------|------|
| 深直清晰 | 事业心强、方向明确、有成就 |
| 从生命线出发 | 靠自己奋斗、白手起家 |
| 从手腕出发 | 早年就有目标、事业起步早 |
| 从智慧线出发 | 中年后事业靠脑力发展 |
| 从感情线出发 | 事业受感情影响大、中年后才稳定 |
| 没有事业线 | 不代表没事业，可能自由职业/多方向发展 |
| 有中断 | 对应时段换行业/失业/转型 |
| 有岛纹 | 对应时段事业受阻 |
| 末端分叉 | 事业多方向发展/兼职/副业 |

### 5. 太阳线（成功线/六秀纹）
**位置**：从无名指根部向下延伸（不一定人人都有）

**含义**：名声、艺术天赋、成功运、受人欢迎程度

| 特征 | 解读 |
|------|------|
| 清晰深长 | 有名气、受人认可、有艺术/创作天赋 |
| 从太阳丘出发 | 天生有才华、容易被看见 |
| 没有此线 | 不代表不成功，只是低调型成功 |
| 与命运线平行 | 事业和名声同步发展 |
| 多条 | 多才多艺 |

## 三、辅助线

| 线名 | 位置 | 含义 |
|------|------|------|
| 婚姻线 | 小指下方、感情线上方（横短线） | 婚姻次数/质量/时间 |
| 健康线 | 从小指下方斜穿掌底至生命线方向 | 健康状况（有此线反而要注意健康） |
| 直觉线 | 从月丘弧形至小指下方 | 直觉力、灵感、第六感 |
| 金星环 | 中指与无名指之间的弧线 | 艺术感、审美能力 |
| 手腕线 | 手腕处横线 | 晚年运、生殖能力 |

### 婚姻线详解（女命尤重）

| 特征 | 解读 |
|------|------|
| 一条深长 | 一段刻骨铭心的感情/一次婚姻 |
| 一条浅短 | 感情淡、可能晚婚或不婚 |
| 两条 | 两段重要感情（不一定是两次结婚） |
| 多条 | 感情经历丰富 |
| 末端上翘 | 不婚主义/晚婚/分居 |
| 末端下弯碰到感情线 | 感情影响健康/因情伤身 |
| 有岛纹 | 婚姻有波折/对方有隐情 |
| 分叉 | 婚姻中可能有分居/异地 |

## 四、手相纹路吉凶

### 特殊纹路

| 纹路 | 特征 | 含义 |
|------|------|------|
| 十字纹 | 两线交叉成十字 | 凶纹，所在区域对应方面有波折 |
| 三角纹 | 三条线围成三角 | 吉纹，所在区域对应方面有成就 |
| 米字纹 | 多线交叉成米 | 吉纹，有意外好运 |
| 岛纹 | 线中出现圆形/椭圆 | 凶纹，对应时段有阻碍 |
| 星纹 | 多线交叉成星 | 大吉大凶视位置而定 |
| 格子纹 | 多线交叉成格 | 阻碍/困境 |
| 断裂纹 | 主线中断 | 对应时段有重大变故 |

### 手指特征

| 手指 | 对应 | 吉相 | 凶相 |
|------|------|------|------|
| 拇指 | 意志力/父母 | 指节匀称、拇指厚实有力 | 拇指短小软弱、过度后弯 |
| 食指 | 权力/进取心 | 长度适中、挺直有力 | 过长(自大)、过短(自卑) |
| 中指 | 命运/责任 | 最长且直 | 歪斜、过短 |
| 无名指 | 艺术/名声 | 与食指等长最佳 | 过长(好赌)、过短(无才) |
| 小指 | 口才/子女 | 长过无名指第一节 | 过短(口才差)、歪斜 |

### 指甲特征

| 特征 | 含义 |
|------|------|
| 甲色粉红 | 健康、气血好 |
| 甲色苍白 | 血虚、贫血 |
| 甲色偏紫 | 血液循环差 |
| 月牙（甲根白色弧） | 8-10指有月牙最佳，代表精力充沛 |
| 月牙过大 | 性急、血压偏高 |
| 无月牙 | 体质偏寒、精力不足 |
| 竖纹明显 | 体质偏弱、操心多 |
| 横纹（博氏线） | 对应时段有重大疾病/营养不良 |

## 五、分析框架（用户发照片后按此顺序分析）

1. **定手型**：五行手哪一类
2. **看整体**：手掌大小/厚薄/色泽/温度感
3. **生命线**：深浅/弧度/岛纹/断裂
4. **智慧线**：长短/弯曲度/起点
5. **感情线**：末端位置/分叉/岛纹
6. **命运线**：有无/深浅/起点
7. **太阳线**：有无/深浅
8. **婚姻线**：条数/深浅/末端
9. **特殊纹路**：十字/三角/岛纹/星纹
10. **手指+指甲**：长短/形态/月牙

每项标注：吉/凶/平 + 解读依据

---

## 六、注意事项

- 手相会变（3-5年有明显变化），不是一成不变的
- 左右手对比看：左手先天+右手后天=完整人生
- 手相是概率不是定论，心态和行动可以改变命运
- 不说"你一定会怎样"，说"有这个趋势，可以这样调整"
- 结合八字分析更准确，单独看手相只能看大方向

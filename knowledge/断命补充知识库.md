# 断命补充知识库 — 从三命通会+错题分析提取

## 一、外貌身材判断法

**出处：三命通会卷七《论性情相貌》**

### 核心规则
- **生旺者主长大，死绝者主矮小**（日主在月令的状态决定身高基础）
- **五行气足，体必丰肥**（五行力量充足→胖；不足→瘦）
- 有煞临则不在此限（煞星可以改变基本判断）

### 十干五行外貌速查

| 五行 | 旺相 | 休囚 | 死绝 |
|------|------|------|------|
| 木 | 骨格修长，手足纤腻，口尖髮美，面色青白 | 瘦长髮少，面黄 | 肌肉干燥，项长喉结 |
| 火 | 头小脚长，印堂窄眉浓，精神闪烁 | 黄瘦尖楞 | 面赤摇膝好动 |
| 土 | 背圆腰阔，鼻大口方面肥色黄，度量宽厚 | 面偏鼻低，声音重浊 | 颜色忧滞 |
| 金 | 面方白净，眉高眼深，鼻直耳红，声音清亮 | 身材瘦小 | 悭吝鄙啬 |
| 水 | 面黑光彩，语言清和 | 人物矮小 | 胆小无略 |

### 判断流程
1. 先看日主五行（决定基本骨架）
2. 看日主在月令的旺衰状态（旺=高大，死绝=矮小）
3. 看五行是否充足（充足=胖，不足=瘦）
4. 看是否有煞星改变（如七杀、伤官等可以改变基本判断）
5. 结合紫微斗数疾厄宫、命宫星曜交叉验证

### 典型错误案例（MingLi-Bench ftb_0009）
- 女命：1981-05-26 02:17 香港
- 八字：辛酉 癸巳 甲辰 乙丑
- 日主甲木，巳月休囚→木衰→偏瘦？但答案是"矮小肥胖"
- 原因：巳月火旺生土，土旺→"土旺相则背圆腰阔面肥"→肥胖
- **教训：不能只看日主五行，要看全局五行哪个最旺来判断身材**

## 二、疾病诊断法

**出处：三命通会卷七《论疾病先知五脏六腑所属干支》**

### 五脏六腑对应
- 甲→胆，乙→肝，丙→小肠，丁→心，戊→胃，己→脾
- 庚→大肠，辛→肺，壬→膀胱/三焦，癸→肾/包络

### 身体部位对应
- 甲→头，乙→顶，丙→肩，丁→心，戊→胁，己→腹
- 庚→脐轮，辛→股，壬→胫，癸→足
- 子→膀胱/耳，丑→脾，寅→胆/两手，卯→肝/十指
- 辰→皮肩胸，巳→面咽齿，午→眼目/精神，未→胃/脊梁
- 申→大肠/肺，酉→精血/小肠，戌→命门/腿踝，亥→头/肾

### 精神疾病判断（抑郁/癫狂/痴）

**关键信号：**
- "火中有土，少水制则心神恍惚" → 心神不宁/焦虑
- "木下藏金，无水收则腿足损伤" + 金克木 → 肝气郁结 → 抑郁
- "丙丁见壬癸亥子多者，内主心气疼痛，颠癎，舌强" → 精神类疾病
- "水死绝主肾气腰足...精神不安之疾" → 精神不安
- "火死绝主肠气结塞，惊悸健忘，精神不安之疾" → 精神问题
- "木死绝...喜怒颠倒，择饮择食之疾" → 情绪失控
- "土死绝...多思足虑，耳聋神浊，健忘少喜" → 思虑过度/抑郁

**抑郁判断要点：**
1. 木被金克严重 → 肝气郁结 → 情绪压抑
2. 火被水克严重 → 心火被灭 → 精神不安
3. 土死绝 → "多思足虑" → 思虑过度型抑郁
4. 印星太重（尤其偏印） → 想太多、钻牛角尖
5. 华盖重重 + 无贵人 → 孤独感强 → 易抑郁

### 长期用药判断
- "金水相生，生旺则气滞，死绝则精滑" → 慢性病
- 大运流年连续克泄日主 → 长期身体不好
- 疾厄宫有化忌 + 煞星多 → 需要长期调理

## 三、子女生育年份判断

**判断流程：**
1. 女命以食伤为子女星（食神=女儿，伤官=儿子）
2. 男命以官杀为子女星（正官=女儿，七杀=儿子）
3. 时柱为子女宫，看时柱十神和藏干
4. 生育年份信号：
   - 流年冲动时柱（子女宫被冲）→ 有生育之象
   - 流年食伤星出现/旺相 → 子女星到位
   - 流年合动子女宫 → 有子女缘
   - 大运走到子女星旺的阶段
5. 女命还要看：
   - 流年食伤透干 → 女儿/儿子星出现
   - 流年地支冲动日支（夫妻宫被冲） → 可能怀孕
   - 流年天干合日干 → 有合住之象

### 典型错误案例（MingLi-Bench ftb_0008）
- 女命：1981-05-26 02:17 香港
- 正确答案：2007年生女儿
- 模型选了2015年（错8年）
- **教训：子女生育年份不能只看子女宫被冲，还要看食伤星是否到位**

## 四、学业/外出求学判断

### 学业判断
- 印星旺 + 文昌 → 学业好
- 食伤旺 → 有才华但不一定读书好
- 官印相生 → 正统学历高
- 伤官见官 → 叛逆，可能中途退学

### 外出求学/迁移判断
- **驿马逢冲** → 有出行/迁移之象
- **年支冲流年地支** → 离开家乡
- **迁移宫有煞星** → 不宜远行（紫微）
- **迁移宫有吉星** → 适合外出发展
- 大运走到驿马位 → 有迁移机会

### 典型错误案例（MingLi-Bench ftb_0006）
- 女命：1981-05-26 02:17 香港
- 正确答案：2002年升学内地，2006年大学毕业
- 模型选了"预科后升本地大学"（错）
- **教训：要看迁移宫和驿马的组合，判断是否去外地**

## 五、结婚年份精确判断

### 核心信号（按优先级）
1. **流年合动夫妻宫（日支）** → 最强结婚信号
2. **流年正官/正财透干** → 配偶星出现
3. **流年合动日干** → 有合住之象
4. **大运走到正官/正财运** → 配偶运旺
5. **流年天喜/红鸾/桃花** → 感情信号

### 常见结婚年份特征
- 流年地支合日支 → 如日支卯，流年戌（卯戌合）
- 流年正官透干（女命）或正财透干（男命）
- 流年与日柱天合地合 → 如日柱甲子，流年己丑（甲己合+子丑合）
- 流年冲动夫妻宫但有合化解 → 先分后合

### 典型错误案例（MingLi-Bench ftb_0002）
- 男命：1974-04-28 16:40 USA
- 正确答案：2006年结婚
- 模型选了"到2022年为止，单身"（错）
- **教训：不能因为夫妻宫有煞星就判断不婚，要看大运流年是否有合动信号**

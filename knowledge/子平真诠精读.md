## 第1章 论十干十二支

### 原文金句

> “天地之间，一气而已。惟有动静，遂分阴阳。”
>
> “甲乙如官长，寅卯如该管地方。”

### 核心规则

1. **一气分阴阳** → 先有一气，再分动静阴阳、老少四象 → 学命必须先从“气机分化”看五行，不能把五行当死物。
2. **五行内含阴阳** → 每一行内部又各有阴阳，如木分甲乙 → 判断十神、根气、气质时，不能只看五行同类。
3. **阳干主气阴干主质** → 甲为木之生气，乙为木之形质 → 同属木，甲偏流行发动，乙偏凝成坚实。
4. **天干为天之气** → 甲乙行乎天，动而不居 → 看天干重在“流通、生化、透出”。
5. **地支为地之形** → 寅卯存乎地，止而不迁 → 看地支重在“根、位、月令、承载”。
6. **禄位以支定** → 甲禄于寅，乙禄于卯 → 禄根看地支，不因天干轮转而改变。
7. **月令以支为准** → 建寅月未必天干见甲，建卯月未必见乙 → 论月令先看支，不可拿月干代月令。
8. **土为冲气所结** → 土非独立孤行，而是木火金水冲气所结，寄旺四时 → 论土必须兼看季节与四行交会。
9. **气质不可俗解** → “甲大林、乙微苗”的机械比喻被作者直接否定 → 断命不可用形象化口诀替代阴阳气质分析。

### 格局成败

- **成**：先分清“干主气、支主形”“阳偏气、阴偏质”，后面才谈得上月令、通根、十神。
- **败**：把干支混看、把甲乙粗判成“大树小草”、把月令误作月干，后续取格几乎必偏。
- **救**：遇到判断混乱时，退回本章三层基础重排：先辨阴阳，再辨气质，再辨天干地支分工。

### 案例拆解

- 本章原文无完整四柱命例，以理论示例为主。
- 条件例1：甲乙对寅卯。作者不是单讲“木”，而是拆成天上之气与地下之形，说明日主、月令、根气不能混成一团。
- 条件例2：甲禄于寅、乙禄于卯。这里已经给出后文“通根比透干更实”的根本依据。

### 例外和易错点

- “甲乙为阳、寅卯为阴”是就“在天成象、在地成形”的统分法说，不是推翻前文“甲阳乙阴、寅阳卯阴”的分法。
- 甲旺于乙、乙坚于甲，讲的是“气”与“质”的差别，不是简单说谁强谁弱。
- 土寄四时，不等于土不重要，恰恰说明土要放进季节流转中看。

### 章际关联

- 本章是第2章“阴阳生克”的前提，不懂干支气质，就分不清官杀偏正。
- 本章也是第3章“阴阳生死”和第6章“得时不旺失时不弱”的根底，因为根气本来就落在“支承干气”这层。
- 与《断命流程指南》“先看月令、再看通根”的方法完全一致。

## 第2章 论阴阳生克

### 原文金句

> “是以四时之运，生与克同用，克与生同功。”
>
> “金同克木，而局有官煞也。”

### 核心规则

1. **生克并用** → 四时运转不能只有相生，必须同时有相克节制 → 克不是纯凶，往往正是成事之机。
2. **克为收敛之机** → 木盛于夏，杀于秋，秋杀使外泄之气内收 → 被克未必坏，常是使其成材。
3. **同五行不同十神** → 虽同是水生木、金克木，但因阴阳不同而分偏正 → 十神必须落实到阴阳，不可只按五行大类断。
4. **官杀须细分** → 金克木中，庚辛对甲乙的作用不一 → 官与杀不是同义词，吉凶层级差别很大。
5. **甲乙官杀异途** → 甲为木之生气，逢庚为杀、辛为官；乙为木之形质，反以庚为官、辛为杀 → 日主阴阳一换，官杀立刻反转。
6. **丙丁官杀异途** → 丙火、丁火对庚辛的克制方式不同 → 判断官杀不能脱离“气”与“质”的不同反应。
7. **印偏正可略，官杀不可混** → 作者明确说印绶偏正虽有不同，可先略看；官杀一善一恶，不可不细详 → 实战优先级上，先把官杀分清。

### 格局成败

- **成**：把相克看成“节而成器”，并把官杀分别清楚，后面才能立官格、杀格。
- **败**：只知生不知克，会把七杀全当凶神；只知克不分阴阳，会把官杀混杂当一回事。
- **救**：遇到“明明受克反而成格”的命，先回到“克与生同功”重看，很多贵局就在节制中成立。

### 案例拆解

- 本章原文无完整四柱命例，以十干关系示例为主。
- 条件例1：甲见庚辛。庚为肃杀之气，辛为金铁之质，所以甲以辛为官、庚为杀。
- 条件例2：乙见庚辛。乙为形质之木，和庚辛的关系反转，所以乙以庚为官、辛为杀。

### 例外和易错点

- “官杀不同”不是单看五行生克，而是看阴阳性质是否相投。
- 印偏印、正印虽然也有区别，但本章重心明确不在印，而在官杀的清浊分判。
- 被克并不必然凶，关键在“克得其节”还是“克之太过”。

### 章际关联

- 第4章“十干配合性情”讲合，第2章先把“原始生克”讲清，二者一顺一逆，必须合看。
- 第8章“论用神”中四吉四凶的顺逆用法，根子就在本章对生克善恶层次的区分。
- 与《十神关系速查》中的“官印相生、伤官见官、食神制杀”相互印证。

## 第3章 论阴阳生死

### 原文金句

> “阳主聚，以进为进，故主顺；阴主散，以退为退，故主逆。”
>
> “人之日主，不必生逢禄旺，即月令休囚，而年日時中，得长禄旺，便不为弱。”

### 核心规则

1. **阳顺阴逆** → 长生沐浴冠带等十二运，阳干顺行、阴干逆行 → 排长生十二运时先辨阴阳，错一步后面全错。
2. **阳生处即阴死处** → 阳干所生之地，往往就是对应阴干所死之地 → 阴阳生死互换是自然流转，不可硬套同一套表意。
3. **气与质的生死不同** → 甲生于亥死于午，乙则生于午死于亥 → 同一五行之阴阳两面，其盛衰判断并不相同。
4. **十二运是细分过程** → 长生到养并非玄词，而是气由生到衰、由绝到续的连续分段 → 看旺衰要理解过程，不只背名目。
5. **日主有根不怕休囚** → 月令虽休囚，但年时日支得长生、禄、旺，仍可不弱 → 强弱不能只看提纲。
6. **墓库也是根** → 阳干逢库，多作有根看 → 库不是一定要冲开才算根。
7. **阳长生力大** → 阳干长生有力，阴干长生虽不如阳干，但亦不弱 → 阴长生不能当废根看。
8. **阳库可用阴库多虚** → 阳干逢库多有根，阴干逢库常无用 → 阴阳大小不同，不能平推。
9. **“投库必冲”是俗论** → 作者直斥时产投库必冲之说为谬 → 见墓库先看是否藏根，不可先起冲库念头。

### 格局成败

- **成**：认清“月令只是大环境，根气仍可补足”，才能正确判断身强身弱。
- **败**：把阴阳十二运混排、把墓库一概当死地、把“投库”一概当需冲开，最容易误判根基。
- **救**：强弱判断失准时，回到三处重查：月令、通根、阴阳顺逆。

### 案例拆解

- 本章原文无完整四柱命例，以长生墓库条件例为主。
- 条件例1：甲木。亥为长生，午为死地，体现“阳主聚、顺行而进”。
- 条件例2：乙木。午为长生，亥为死地，体现“阴主散、逆行而退”。

### 例外和易错点

- 阴长生“不甚有力”，不是“无力”。
- 阳干逢库多论根，阴干逢库多不受用，不能把“有库”一律当“有根”。
- 十二运名目是比喻，不是字面人生故事，不可按词义硬套吉凶。

### 章际关联

- 本章是第6章“得时不旺失时不弱”的理论展开，后者等于把本章根气思想用于实战判断。
- 第8章“论用神”里扶抑法，本质也建立在本章强弱辨法上。
- 与《断命流程指南》附录A“十干生旺死绝表”完全对应。

## 第4章 论十干配合性情

### 原文金句

> “合化之义，以十干阴阳相配而成。”
>
> “用神之情，不向日主，或日主之情，不向用神，皆非美朕也。”

### 核心规则

1. **十干相合有化序** → 甲己土、乙庚金、丙辛水、丁壬木、戊癸火 → 合化次序依五行相生而来，不可乱配。
2. **合不仅看化，更看向背** → 有配合就有“向我”或“背我” → 同样是合，有的助格，有的夺用。
3. **喜神被他合去即失用** → 官被丙合、印被戊合、财被别甲合、食被辛合 → 名义上有喜神，实际已经不归我用。
4. **年干月干先合，未必关日主** → 如年己月甲，年财被月合去，与日主甲乙无分 → 断十神归属要分“谁先合谁”。
5. **六亲以向背论亲疏** → 男命财为妻，财被他合去，妻不亲夫；女命官为夫，官被他合去，夫不亲妻 → 合象可直接映射六亲关系。
6. **日主与用神要相向** → 用神不向日主，或日主不向用神，格局虽成也不美 → “有情”观念从本章开始立起来。

### 格局成败

- **成**：合而向我，或我能取而用之，用神与日主情意相投，格局才真能发用。
- **败**：喜神虽透，却先被别干合去，看似有官印财食，实则全是虚神。
- **救**：后文第5章会进一步说明“本身之合不作合去”“隔位争合不必尽凶”，本章先立总原则。

### 案例拆解

- 本章原文无完整四柱命例，以干支条件示例为主。
- 条件例1：年己月甲。年上之财先被月甲合去，日主即使是甲或乙，也不算得此财。
- 条件例2：女命丁日逢壬。若壬是日主自己可合之夫星，则情密；若先被别位丁合去，则夫星虽透如不透。

### 例外和易错点

- “有合”不等于“能化”，本章重心先在“情向何方”，不是先在化神成败。
- 喜神被合去，不能只看五合口诀说“有情”，必须看究竟归谁所有。
- 日主自己去合官财，与别位先合，是两回事。

### 章际关联

- 第5章专讲“合而不合”，等于对本章作实战修正。
- 第12章“有情无情”直接承本章而来，本章是“情”的源头。
- 与《十神关系速查》“十神被合”部分可互参，但本章更强调归属与向背。

## 第5章 论十干合而不合

### 原文金句

> “此制于势然也，合而不敢合也，有若无也。”
>
> “惟是本身十干合之，不为合去。”

### 核心规则

1. **中隔克神则合不敢成** → 甲己欲合，中间隔庚 → 甲不能越庚去合己，此为“合而不敢合”。
2. **距离太远则合力甚轻** → 甲在年、己在时 → 心虽相契而地远，只作半合看，祸福不过二三成。
3. **合一留一不破格** → 官杀并透时，合去其一而留其一 → 反能去杂存清。
4. **本身之合不作合去** → 乙日见庚官、丁日见壬官，只要是日主自合 → 官夫仍归我，不算被夺。
5. **他位先合则日主失权** → 月上乙先合年庚，日干乙反不能再合 → 这是“被别人合去”，不是“我得官”。
6. **争合妒合主情不专** → 两辛合丙、两丁合壬 → 有合意但不专一，关系杂而不纯。
7. **隔位两合一不作争妒** → 两合一但中间有日主等隔位 → 不必硬断争妒。
8. **不可滥用支合论官** → 以辰酉合、卯戌合之类泛称“合官” → 作者认为是大谬。

### 格局成败

- **成**：利用“合一留一”“合煞留官”“本身自合”去清局、留用、护夫官，都是成格之法。
- **败**：用神被他位先合、争合妒合而情不专、误把远合当近合、误把支合当干合，都会败局。
- **救**：看合先分三层，能不能合、是谁去合、合后是清局还是夺用。

### 案例拆解

- **高太尉命**：庚午、乙酉、甲子、乙亥。
- **判法**：甲日生酉月，原局见庚与两乙。作者定为“两乙合庚，甲日隔之”，不作争妒重凶，而作“合煞留官”看。
- **结论**：此命关键不在“两个乙都来争庚”，而在隔位之后，合象反而帮助清煞存官，所以“无减福也”。

### 例外和易错点

- “本身之合”最易误判。日主自合官财，不是合去，而是我得其情。
- “合而不合”不是完全没事，而是力量很轻，常只应二三成。
- 很多俗说把任何相邻支合都当“合官”“合财”，与本章原意相反。

### 章际关联

- 本章补足第4章“配合性情”，把“向背”落实到“距离、间隔、先后”。
- 第9章“救应”里常用合来清煞、去伤，本章是其技术基础。
- 与《断命流程指南》附录C“合化条件与争合妒合”可以互参，但本章更重实局归属。

## 第6章 论十干得时不旺失时不弱

### 原文金句

> “得时俱为旺论，失时便作衰看，虽是至理，亦死法也。”
>
> “得时为旺，失时为衰；党众为强，助寡为弱。”

### 核心规则

1. **旺衰与强弱分开看** → 得时为旺、失时为衰；党众为强、助寡为弱 → 旺不等于强，衰不等于弱。
2. **得令未必能任事** → 甲乙生寅卯月虽旺，若金火党众、木助寡 → 可旺而不强。
3. **失令未必不任事** → 甲乙生申酉月虽衰，若比印重叠、支中通根 → 可衰而不弱。
4. **喜忌神同受此法** → 不只日主，喜神、用神、忌神也要分旺衰强弱 → 断格不能只评日主。
5. **四柱有根便能任财官** → 不论月令休囚，只要有根，便能受财官食神、当伤官七杀 → 根比表面旺衰更实。
6. **根有轻重层级** → 长生禄旺最重，余气次之，墓库又次 → 看根必须分等级。
7. **干多不如根重** → 一墓库胜一比肩，一余气胜二比肩，一长生禄刃胜三比肩 → 通根胜过干帮。
8. **阴干根法不能照搬阳干** → 乙逢戌、丁逢丑不作墓库根；乙逢午、丁逢酉虽为明根，只比一余气 → 阴根较轻。

### 格局成败

- **成**：把“得令、党众、通根”三层合看，才能判断日主能否承财、任官、驾杀。
- **败**：只看月令，不看通根；只数比肩，不看支根；把阴干根气照搬阳干，都会误判全局。
- **救**：强弱判断摇摆时，以“支根优先于干帮”的原则重排力量。

### 案例拆解

- 本章原文无完整四柱命例，以条件例为主。
- 条件例1：甲乙木生寅卯月，但干见庚辛、支见酉丑，木虽旺而不强。
- 条件例2：甲乙木生申酉月，但比印重叠且年日时支通根比印，木虽衰而不弱。

### 例外和易错点

- “旺”是时令概念，“强”是综合力量概念，不能混用。
- 阴长生、阴墓库常被初学者高估，本章明确要求折算处理。
- “有根便能任财官”不等于“有根就一定贵”，只是先解决能否承受的问题。

### 章际关联

- 本章承第3章“阴阳生死”而来，是十二运与通根判断的实战化。
- 第8章扶抑取用、第12章有力无力，都以本章为支撑。
- 与《断命流程指南》第4步“日主强弱”完全打通。

## 第7章 论刑冲会合解法

### 原文金句

> “是会合可以解冲也。”
>
> “是所谓以刑冲而解刑冲也。”

### 核心规则

1. **三合六合可解冲** → 原局有冲，若冲中一方另得合会 → 冲力可减，甚至不作冲论。
2. **三合六合可解刑** → 原局有刑，若刑中一方另得合会 → 刑力可转移或化去。
3. **解神被合去反生祸** → 原本用来解刑冲的字，被别字先合走 → 反而重新激活刑冲。
4. **会局改向则解法失效** → 合神、解神若转投别局 → 原先能解的刑冲重新成立。
5. **月令用神最怕刑冲** → 四柱之中刑冲本非美事，刑冲用神尤为破格 → 判断轻重先看是否冲到月令用神。
6. **别位刑冲可反救月令** → 用另一处的冲刑去牵制原本伤月令之字 → 虽六亲有伤，但格局未破。
7. **地支关系必须整体看** → 冲、刑、合、会不是单点判断，而是全局动态平衡 → 不能见一个合就断“已解”。

### 格局成败

- **成**：会合得当，能护月令、保用神，即使别宫有刑冲，格仍可存。
- **败**：刑冲直伤月令用神、解神被夺、会局改投他方，都是破格重因。
- **救**：本章最大价值就是“先看有没有解，再看解会不会变质”。

### 案例拆解

- 本章原文无完整四柱命例，以条件例为主。
- 条件例1：甲生酉月，逢卯冲；若支有戌、辰、亥未、巳丑，则可分别用合会解冲。
- 条件例2：丙生子月，卯刑子；若另逢酉冲卯，则卯不再专刑月令，属于“以刑冲解刑冲”。

### 例外和易错点

- 作者对“三刑取义”持保留态度，说明三刑不宜过度玄化。
- 有解不一定真解，解神若转去会局，反会使原冲原刑重新发动。
- 以刑冲解刑冲，只是“保格”之法，不代表六亲全无伤损。

### 章际关联

- 第9章“救应”中“刑冲而会合以解之”正是本章的直接延伸。
- 第15章“相神紧要”也会进一步说明，很多时候真正护格的是解冲之相神。
- 与《断命流程指南》第2步太岁关系、第7步流年刑冲的处理可以互证。

## 第8章 论用神

### 原文金句

> “八字用神，专求月令，以日干配月令地支，而生克不同，格局分焉。”
>
> “当顺而顺，当逆而逆，配合得宜，皆为贵格。”

### 核心规则

1. **用神专求月令** → 先以日干配月令地支，看其生克成何十神 → 格局先立在月令，不立在别处。
2. **用神五法并存** → 扶抑、病药、调候、专旺、通关皆可取用 → 取用不能只有“强弱扶抑”一条路。
3. **四吉神宜顺用** → 财、官、印、食为善神，重在生护成全 → 顺其性而成格。
4. **四凶神宜逆用** → 杀、伤、劫、刃为不善神，重在制伏、化泄、驾驭 → 逆其势而得贵。
5. **见象不能跳过月令** → 官印双透、财食同见、煞食同现，都要先问“月令主的是什么” → 同样四字，主格可能完全不同。
6. **建禄月劫不能把月令本身当用神** → 木生寅卯，日与月同，本气只是身旺，不可直接当用 → 必须另看四柱财官杀食透干会支。
7. **月令无显用神仍以月令为主** → 即使需另取财官杀食，仍是围绕月令之气寻找出路 → “月令无用神”不等于“月令不重要”。
8. **顺逆配合决定贵贱** → 不是有某十神就贵，而是是否顺逆得宜 → 用法比名目更重要。

### 格局成败

- **成**：月令定格，四柱配合得宜，善神得生护，凶神得制化，即成贵格。
- **败**：脱离月令见象直断，把官印双透、食神生财、煞印相生等套成固定模板，最易错格。
- **救**：遇到判断冲突时，回到“提纲专主”重排，先定月令主神，再定别位辅神。

### 案例拆解

- 本章原文无完整四柱命例，以条件例和错判例为主。
- 条件例1：见正官佩印，不可立刻当“官印双全”，还要问究竟是“官格佩印”还是“印格用官”。
- 条件例2：见财透食神，不可立刻当“食神生财”，还要问月令究竟是财格还是食格。

### 例外和易错点

- “用神”在《子平真诠》首先是格局用神，不是现代泛称的“喜用神”。
- 建禄月劫最容易误把月令本身当用神，作者明确否定。
- 调候虽重要，但在本章体系中仍要围绕月令提纲，不可抛开格局空谈寒暖。

### 章际关联

- 第9章开始进入“用神成败救应”，是本章的直接展开。
- 第14章“用神配气候得失”会对本章的“调候法”作专门补充。
- 与《断命流程指南》第5步“格局 > 调候 > 强弱”的排序高度一致。

## 第9章 论用神成败救应

### 原文金句

> “成中有败，必是带忌；败中有成，全凭救应。”
>
> “八字妙用，全在成败救应，其中权轻权重，甚是活泼。”

### 核心规则

1. **官格成法** → 官逢财印，又无刑冲破害 → 官格可成。
2. **财格成法** → 财生官旺，或财逢食生而身强带比，或财格佩印而位置不碍 → 财格可成。
3. **印格成法** → 印轻逢杀、官印双全、身印两旺而用食伤、印多逢财而财轻透 → 印格可成。
4. **食格成法** → 食神生财，或食带杀而无财，或弃食就杀而透印 → 食格可成。
5. **杀格成法** → 身强七杀逢制 → 杀格可成。
6. **伤官格成法** → 伤官生财、伤官佩印而伤旺印有根、伤官旺身弱而透杀印、伤官带杀无财 → 伤官格可成。
7. **刃禄格成法** → 阳刃透官杀而露财印不见伤官，建禄月劫透官见财印、透财见食伤、透杀遇制 → 可成格。
8. **败格总因** → 官逢伤克刑冲，财轻比重或财透杀，印轻逢财，食逢枭，杀逢财无制，伤官见官，阳刃无官杀，建禄月劫无财官 → 皆败。
9. **成中带忌为隐败** → 美格中若再见伤、合、财破印、印护杀却又透财等 → 成局里已埋败因。
10. **救应是翻盘关键** → 官逢伤用印解，财逢劫用食化或生官制劫，印逢财用劫财护印，食逢枭可就杀或生财护食，杀逢食制可用印护杀或财去印存食 → 败局可转。

### 格局成败

- **成格条件**：先看月令主神是否得生、得护、得清，再看四柱有没有刑冲破害。
- **破格条件**：不是“见忌即败”，而是“忌神真正打到了月令主线”才败。
- **救应方法**：合、制、化、护、隔都可作为救法，但必须看救的是主神，还是只是添乱。

### 案例拆解

- 本章原文无完整四柱命例，以格局条件例为主。
- 条件例1：财旺生官，本是美格；若身弱再透官，则官来压身，反成破格。
- 条件例2：伤官见官，本为大忌；但若透财且位置配置得当，能转成“伤官生财、财再生官”，反可解忌。

### 例外和易错点

- “成”不代表终身无破，“败”也不代表绝对无救。
- 同一字既可能救局，也可能带忌，关键在先后和站队。
- 断格不能只背“某格喜某神”，必须追问“这个神现在是在护主，还是在拆主”。

### 章际关联

- 第10章“用神变化”解释为什么同一月令会有不同成败路径。
- 第13章“因成得败、因败得成”则是把本章的成败救应推到更灵活的层面。
- 与《十神关系速查》中多数组合判断直接相通，但本章更强调“以月令统领组合”。

## 第10章 论用神变化

### 原文金句

> “用神既主月令矣，然月令所藏不一，而用神遂有变化。”
>
> “是故八字非用神不立，用神非变化不灵。”

### 核心规则

1. **月令多藏干可换主** → 寅中甲为本主，丙戊亦藏；本气不透而余气透 → 余气可暂代月令主事。
2. **透干可变用神** → 不透甲而透丙，寅月便可能由印转财官路径 → 谁透谁先主事。
3. **会合可变用神** → 寅午戌会火、申子辰会水等 → 月令原神可因会局改性。
4. **变之而善** → 化忌为喜、把原本冲突串成生化链 → 格愈美。
5. **变之而不善** → 把原本清纯之格扭成劫、煞等忌象 → 格遂坏。
6. **变化不必失本格** → 变神虽起，但本气仍透、原神仍有根或有他神制衡 → 可主格不失、兼格并行。
7. **先定主格后定兼格** → 遇多重变化，先看谁秉月令、谁透、谁得局 → 不可被表面新象牵走。
8. **变化判断重先后** → 三合变局若先成，则后面再见官杀伤食，要按变化后之新性质来断，不按旧格硬套。

### 格局成败

- **成**：变化把忌神改成喜神，或把杂局串成顺生链，就是“变之善”。
- **败**：变化若使原来吉格失位、主神被夺、局面更杂，就是“变之不善”。
- **救**：若原神未失根、未失透，或变后仍能归于主线，可作“变而不失本格”。

### 案例拆解

- 本章原文无完整四柱命例，以条件例为主。
- 条件例1：癸生寅月，藏甲透丙，会午会戌，寅午戌三合，原本伤官可化财，不作伤官见官论。
- 条件例2：乙生寅月，月劫秉令，会午会戌，则劫化食伤；若再透戊，则转成食伤生财，不作比劫争财论。

### 例外和易错点

- “变化”不是随便看见会局就变，要看透干、秉令、先后。
- “化忌为喜”最易被误断成“原忌仍在”，本章要求按变化后的主线重判。
- “变而不失本格”说明命局经常一体两面，不可非黑即白。

### 章际关联

- 第11章“纯杂”是对变化后结构品质的评估。
- 第13章“因成得败、因败得成”则是变化进一步引发的高阶结果。
- 与《断命流程指南》中的“格局优先，但需灵活判断”完全同调。

## 第11章 论用神纯杂

### 原文金句

> “用神既有变化，则变化之中，遂分纯、杂。纯者吉，杂者凶。”
>
> “互用而两相得者是也。”

### 核心规则

1. **纯以同心为准** → 两神互用、方向一致、彼此成全 → 为纯。
2. **杂以相悖为准** → 两神并立而互不相谋、彼此拆台 → 为杂。
3. **财官相生为纯** → 如辛生寅月，甲丙并透 → 财与官相生，两相得。
4. **财食相生为纯** → 如戊生申月，庚壬并透 → 财与食相生，可作纯局。
5. **相克得当亦可纯** → 如癸生未月，乙己并透，杀与食虽相克，但制得其宜 → 仍属纯。
6. **官伤相战为杂** → 如壬生未月，乙己并透，官伤同见且互战 → 为杂。
7. **财印互妨为杂** → 如甲生辰月，戊壬并透，印与财相克 → 为杂。
8. **纯杂从作用看，不从数量看** → 不是神少就纯，神多就杂；关键看是否同归一线。

### 格局成败

- **成**：纯局更容易稳定发用，格成之后不易横生内耗。
- **败**：杂局往往表面不错，实际内部互拆，常见有富无贵、有名无实、起伏反复。
- **救**：若能通过合、制、隔，让原本相战的两神重新各安其位，杂局可转纯。

### 案例拆解

- 本章原文无完整四柱命例，以条件例为主。
- 条件例1：辛生寅月，甲丙并透。财官相生，为“纯”。
- 条件例2：壬生未月，乙己并透。官伤相克，两不相谋，为“杂”。

### 例外和易错点

- “相克”不一定就是杂，若克得其所，反可能是纯。
- “同是喜神”不一定就是纯，若彼此争权、抢位，也照样成杂。
- 纯杂不脱离第10章变化而谈，必须建立在“主神已变定”之后。

### 章际关联

- 本章承第10章“变化”而来，是对变化结果的质量评估。
- 第12章“格局高低”则进一步看纯局有没有力、杂局有没有情。
- 与《十神关系速查》中“组合矛盾”思想可互证。

## 第12章 论用神格局高低

### 原文金句

> “然其理之大纲，亦在有情无情、有力无力之间而已。”
>
> “或一字而有千钧之力，或半字而败全局之美。”

### 核心规则

1. **格局高低先看有情** → 用神、辅神能相顾相成、能护主清局 → 为有情。
2. **格局高低再看有力** → 用神有根、有源、有生扶，不浮不虚 → 为有力。
3. **有情兼有力为上格** → 既会配合又站得住 → 层次最高。
4. **有情而力不足为次** → 用神方向对，但根浅势薄 → 可贵而不极贵。
5. **有力而情不全为次** → 力量虽足，但搭配别扭 → 也难登上格。
6. **无情则名成实不成** → 如印用七杀，本是贵格；若身强印旺而再透杀，反成偏枯 → 格低。
7. **无力则好神不能发用** → 杀强食旺而身无根，或身强比重而财无气 → 或贫或夭。
8. **一字轻重可改层次** → 有时只多一个合、一个根、一个护字，全局层次立变 → 不能只看大框架。

### 格局成败

- **成**：用神得情得力，辅神不争，忌神不乱，便是高格。
- **败**：纵有格名，若无情无力，则只是空壳之格。
- **救**：层次不高者，多从“补力”或“补情”下手，前者补根源，后者清配合。

### 案例拆解

- 本章原文无完整四柱命例，以条件例为主。
- 条件例1：甲用酉官，壬合丁而清官，且壬水根深，是“有情兼有力”。
- 条件例2：乙逢酉杀，透丁以制，但若辛丁虽旺而乙根不深，则只是“有力而非力之全”。

### 例外和易错点

- “有情”不是感性词，而是指用神有没有真正服务主线。
- “有力”不是越旺越好，旺而失衡、旺而无情，仍会降格。
- 身强印旺、伤轻印重之类，最容易被误判为“神多即贵”，本章明确否定。

### 章际关联

- 第4章讲“向背”，本章把“向我即有情”推成层次标准。
- 第15章“相神紧要”会继续说明，很多高低其实取决于相神是否得力。
- 与《断命流程指南》“神煞只是辅证，格局层次看主线”一致。

## 第13章 论用神因成得败因败得成

### 原文金句

> “八字之中，变化不一，遂分成败；而成败之中，又变化不测，遂有因成得败，因败得成之奇。”
>
> “观命者毋眩而无主、执而不化也。”

### 核心规则

1. **成局过满反成败** → 原局本成，后添一字去党忌神、坏印、杂官、牵动隐患 → 可因成得败。
2. **化伤为财也会翻车** → 辛生亥月，透丁为用，卯未会财，本可成；若反去党杀 → 因成得败。
3. **印用七杀也怕财搅局** → 癸生申月，秋金重重，略带财可损太过；再逢杀时若成杀印忌财 → 亦因成得败。
4. **败局若得隔解可反成** → 官印逢伤本败，若伤不能越位伤主，反可泄身为秀 → 因败得成。
5. **杂局若得清神可反成** → 杀刃逢食本败，若食神合官留杀，使官杀不杂 → 因败得成。
6. **判断关键在站队** → 新进一字到底帮谁、害谁、清谁、混谁 → 决定是转败还是转成。
7. **不可执死法** → 同一结构在不同先后、远近、轻重下，结果能完全相反 → 断命必须活看。

### 格局成败

- **成而败**：大多因“带忌被引爆”，原来有用的结构反被新神扯偏。
- **败而成**：大多因“关键一字入局救主”，把原本冲突改成清局、泄秀、去杂。
- **救应方法**：本章不是另立新法，而是把前面“变化、纯杂、高低、救应”全部综合动态使用。

### 案例拆解

- 本章原文无完整四柱命例，以条件例为主。
- 条件例1：官印逢伤，本属败格；但辛生戌月、年丙时壬之类，壬不能越戌克丙，反能泄身为秀，故因败得成。
- 条件例2：杀刃逢食，本属败格；但庚生酉月、年丙月丁、时上逢壬，食神合官留杀，官杀不杂，故因败得成。

### 例外和易错点

- “成格”不是结案，只要再添一字仍可能翻盘。
- “破格”也不是死刑，关键看有无转机、有无关键护神。
- 最忌见奇局就眩惑，或执着口诀不肯转弯。

### 章际关联

- 本章是第9章“成败救应”、第10章“变化”、第11章“纯杂”、第12章“高低”的综合应用章。
- 第14章“配气候得失”继续提醒，除了结构变化，还要把寒暖燥湿带入判断。
- 与《断命流程指南》“每个结论必须给出推理链”高度契合，因为本章本质就是追踪推理链的转折点。

## 第14章 论用神配气候得失

### 原文金句
- “論命惟以月令用神為主，然亦須配氣候而互參之。”
- “氣有衰旺，取用不同也。”

### 核心规则
- **月令定格，气候定成败** → 月令用神已经成立 → 仍须复核寒暖燥湿是否允许其发用。
- **冬木官印未必贵** → 冬木再逢寒水、虽透官星印星 → 水寒木冻，官印名美而难成真贵。
- **冬木喜火泄身兼调候** → 身印两旺之冬木透食伤火 → 既可泄秀，又能暖局，多主秀贵。
- **金水伤官见官可通** → 金水成象而气候偏寒 → 见官不以常法论凶，先看火能否调候。
- **冬金伤官带杀更显秀** → 冬金寒凝 → 伤官带杀若能借火发越，往往比寻常格局更清。
- **夏木伤官佩印更佳** → 夏木火炎土燥 → 印可济燥，伤官佩印反成上法。
- **冬水伤官用财难贵** → 冬水寒甚 → 即便伤官生财，多主小富，难言文秀显达。
- **夏木伤官用财贵而不秀** → 夏木燥烈、财星多归燥土 → 虽可见贵，灵秀不足。
- **木火通明与金水相涵有季节条件** → 春木见火可论通明，秋金见水可论相涵 → 夏木、冬金不可硬套。
- **秋金遇官须避壬癸透干** → 庚申月支成子辰水局而透丁官 → 只要壬癸不再透干混局，反可成贵。
- **食神逢印不必一概夺食** → 夏木火盛而轻见正印 → 可视作调候济燥，未必破格。

### 格局成败
- 成格条件：月令用神本身成立，且气候得到校正；寒局得暖，燥局得润，湿局得燥，才华与格局才有真实着力点。
- 破格条件：只见官印、食财、伤官佩印之名，不问季节气候；尤其冬木、冬水、夏木、冬金四类最容易“格美而气败”。
- 救应方法：以调候先行。冬木用火，冬金借火，夏木得水印，秋金成水局但忌壬癸再浊；先把气候修正，再谈十神高低。

### 案例拆解
- 局式例1：**庚生申月，支中或子或辰，天干透丁**
  - 判法：申金得令，又会子辰成水局，金水相涵。
  - 成败关键：丁官可用，但壬癸不可再透，否则官被水混。
  - 结论：这是“调候优先于常法”的典型，属秋金见水而见官无碍。
- 局式例2：**冬木官印双透**
  - 判法：表面是官印双全。
  - 破格原因：寒水冻木，印不能生，官也难贵。
  - 救应：须有火暖局，方能把官印变成真用。

### 例外和易错点
- 易错一：把“伤官见官为祸百端”当成不变铁律，忽略金水见官、秋金用官的调候例外。
- 易错二：只看格名，不看季节，最容易把“能成富”误判成“必成贵”。
- 易错三：把“木火通明”泛化到所有木命见火，实际上只主春木，不主夏木。
- 易错四：把“食神逢印必夺食”一概而论，忽略夏木火炎时印可救燥。

### 章际关联
- 与第15章相连：气候调好之后，还要看谁来辅佐用神，进入“相神”问题。
- 与《穷通宝鉴》互证：都是强调调候不是枝节，而是格局可否兑现的先决条件。
- 与第18、19章互证：吉凶神能否成事，往往不是神本身，而是是否顺应时令。

## 第15章 论相神紧要

### 原文金句
- “月令既得用神，則別位亦必有相，若君之有相，輔者是也。”
- “傷用神甚於傷身，傷相甚於傷用。”

### 核心规则
- **用神之外必有相神** → 月令既已立格 → 四柱中必再找帮助其成立、护持、流通的一字。
- **官格之相多取财** → 官逢财生 → 官为用，财为相。
- **财格之相多取官** → 财旺生官 → 财为用，官为相。
- **杀格之相多取食** → 七杀逢食制 → 杀为用，食为相。
- **凡成局所赖者皆为相** → 不限于财官食印名目 → 只要全局赖此一字才清、才通、才成，便是相神。
- **伤相比伤用更重** → 用神是主线，相神是实现主线的桥梁 → 桥断则格局立败。
- **相神可在天干** → 如壬合丁护官、己合甲存财、戊合癸清杀 → 皆为“以合成相”。
- **相神可在会局** → 卯未会、子辰会等 → 能转局、通关、清气者皆可作相。
- **相神受伤会使有情化无情** → 本来能护官生财之字被合去、被制去 → 格局从清转浊。
- **看命必须同时问主帅与辅臣** → 只找用神不找相神 → 常会误断“格局好却发不大”的命。

### 格局成败
- 成格条件：用神立得住，相神又无破；二者方向一致，格局自然高。
- 破格条件：相神被合、被冲、被克、被夺；尤其“制忌之神”若被伤，格局常立刻从有用变无用。
- 救应方法：优先保护相神。见相神被伤，要看有无第二层辅神或会局补位；若无，则格局层次大降。

### 案例拆解
- 局式例1：**甲用酉官，透丁逢壬**
  - 判法：丁伤官能犯官，壬来合丁，变成“合伤存官”。
  - 相神：壬水。
  - 结论：官格能成，全赖壬之相。
- 局式例2：**戊用子财，透甲并己**
  - 判法：甲煞能扰财，己土合甲，存财成格。
  - 相神：己土。
  - 结论：财格之成立，不在财本身，而在己能清煞。
- 局式例3：**庚生申月，透癸泄气，子辰会局**
  - 判法：癸泄庚本嫌不灵，子辰会水则成金水相涵。
  - 相神：子辰会局。
  - 结论：会局也可作相，不限单一透干。

### 例外和易错点
- 易错一：把“相神”当成次要配角，实际上它常常决定格局能否真正落地。
- 易错二：只把财官食印固定地当相神，忽略“凡成局所赖者皆相”。
- 易错三：只看相神出现，不看相神是否受伤；有相而破，等于无相。
- 易错四：把“伤相”理解为小损，原文明确强调伤相往往比伤用更重。

### 章际关联
- 承接第14章：气候解决后，格局还需辅神才能兑现。
- 引向第16章：四墓杂气月里，常要靠透干、会支中的“相”把主气拎清。
- 与第20章互证：相神先出现还是后出现，也会影响吉凶先后。

## 第16章 论杂气如何取用

### 原文金句
- “四墓者，雜氣也。”
- “透幹會取其清者用之，雜而不雜也。”

### 核心规则
- **四墓皆属杂气** → 辰戌丑未藏干多、主气不单 → 不可像子午卯酉那样一眼定格。
- **杂气先看透干** → 月令所藏何神透出天干 → 先以透神为主取用。
- **杂气再看会支** → 若支成三合三会 → 会局之气可转月令主用。
- **一透一用** → 只透一神 → 就按这一神立格。
- **兼透兼用** → 两神并透 → 两者同参，但须分有情无情。
- **透会并用** → 既透干又会支 → 透干与会支都要纳入主线。
- **顺相成者为有情** → 透官得印、透印得会局、透官再得财生 → 皆属有情。
- **逆相背者为无情** → 透官而会伤、透财又透印相战、透官又透伤破官 → 皆属无情。
- **有情可转无情** → 本来结构顺，但后续火土搅局、辰戌相冲动库 → 可把清局破掉。
- **无情亦可转有情** → 表面相克，若所克者恰是忌神，如月劫、偏官 → 反成佳格。

### 格局成败
- 成格条件：透干、会支、辅神方向一致，能从杂气中提出一条清主线。
- 破格条件：两神并透而互坏、会局反向、财印交战、官伤并见；杂气月最怕“多头并进”。
- 救应方法：宁守一清，不贪双美；见有情将败者，要看会局能否清之，见无情者，要追究它克掉的究竟是不是忌神。

### 案例拆解
- 局式例1：**甲生辰月，透戊则偏财，透癸则正印，透乙则月劫**
  - 判法：杂气月先看谁透。
  - 结论：透干定主用，是杂气取用第一法。
- 局式例2：**甲生辰月，会申子成水局**
  - 判法：不透印也可因会局取水印。
  - 结论：会支可代透干主事。
- 局式例3：**甲生辰月，透癸又会申子**
  - 判法：印透而印局又成。
  - 结论：透会同向，属有情清格。
- 局式例4：**壬生未月，透己为官，而会亥卯成伤**
  - 判法：透官与会支反向。
  - 结论：属无情，官难成。
- 局式例5：**丙生辰月，透戊为食，又透壬为杀**
  - 判法：表面食杀相克。
  - 结论：若成食神制杀，反是美格，属“无情终有情”。

### 例外和易错点
- 易错一：见辰戌丑未就说“开库”，而不先问透了什么、会了什么。
- 易错二：把“两神并透”直接当美，不分是相成还是相战。
- 易错三：见相克就判坏，忽略“所克者若为忌神，反成好局”。
- 易错四：杂气月最怕贪心，想财印官杀并取，结果一局皆浊。

### 章际关联
- 与第15章相通：杂气月尤其依赖相神、会局把主线扶清。
- 与第17章相连：四墓杂气的重点是透干会支，不是刑冲开库。
- 与第20章相通：杂气月里，多神并见时，先后顺序尤为重要。

## 第17章 论墓库刑冲之说

### 原文金句
- “辰戌丑未，最喜刑衝，財官入庫不衝不發--此說雖俗書盛稱之，然子平先生造命，無是說也。”
- “是故四墓不忌刑衝，刑衝未必成格。”

### 核心规则
- **财官入库逢冲必发并非正法** → 民间俗说常主“开库” → 原文明确否定其为普遍法则。
- **四墓先论透干会支** → 杂气月已有正法可取用 → 不必额外依赖刑冲求发。
- **四库以土为主** → 虽藏五行 → 真正被冲而动者，多先是土气。
- **土财土官逢冲有时可发** → 如甲用戊财逢辰戌冲、壬用己官逢丑未冲 → 因所用正是土。
- **干已透则不待冲** → 戊己既在天干清透 → 即便不冲库，格局也可成立。
- **水财水官遇冲反受累** → 己生辰月透壬为财，戌冲则劫动；丁生辰月透壬为官，戌冲则伤官起。
- **不可误把干透之神当库中之物** → 壬官已透干 → 不能再说“入库待戌冲开”。
- **月令非四墓不可乱求冲** → 命局别处带辰戌丑未 → 不能据此强作“开库”论。
- **日坐四库是根，不是入库待解** → 日临辰戌丑未多作库根看 → 不可动辄求冲以“救身出库”。
- **四墓之冲多属冲动，不尽是冲克** → 与子午卯酉相仇之冲不同 → 需分清性质。

### 格局成败
- 成格条件：用神本身清透、会局得力；若所用恰为土气，四墓之冲可能添活气，但仍属辅助。
- 破格条件：把水财水官之类误作“库中神”，用冲去动，反把劫财、伤官引出破局。
- 救应方法：先看所用五行是不是土，再看冲动后出来的是吉神还是忌神；“先辨所出，后论冲效”。

### 案例拆解
- 局式例1：**甲生辰月，戊土透**
  - 判法：偏财已在天干明透。
  - 结论：无需再靠辰戌冲来“开财库”。
- 局式例2：**丁生辰月，壬官透干**
  - 判法：若再求戌冲辰，只会动出辰中伤官。
  - 结论：所谓“冲开官库”实是伤官犯官。
- 局式例3：**癸生辰月，透戊为官，与戌相冲**
  - 判法：官最忌冲，但这里所用为土官，四墓之冲属冲动。
  - 结论：未必破格，可视作“四墓不忌冲”的特例。

### 例外和易错点
- 易错一：把所有“库”都当成等待开启的宝箱。
- 易错二：把“出库”“投库”混为一谈，尤其错把透干之神认成又入库。
- 易错三：看见命局有四墓就想刑冲，不分月令是不是四墓。
- 易错四：忽略四墓冲与子午卯酉冲的性质差别。

### 章际关联
- 承接第16章：四墓核心在杂气取用，不在机械冲库。
- 与第28章互证：藏在支中的气，要看何时、何运被真正引出，而不是一概靠冲。
- 对后学外格、神煞派有纠偏作用：先格局，后库气。

## 第18章 论四吉神能破格

### 原文金句
- “財官印食，四吉神也，然用之不當，亦能破格。”

### 核心规则
- **吉神不等于永远吉** → 财官印食虽名四吉 → 一旦站错队，照样拆局。
- **食神带杀忌再透财** → 食神原可制杀 → 财来生杀，反助忌神破格。
- **春木火旺忌见官** → 木火通明之局重在秀气 → 官来反损其清。
- **杀逢食制忌透印** → 食神已经制杀 → 印来化杀护杀，使食制失功。
- **财旺生官忌露食** → 财官一路本清 → 食神一出，则气杂而官不纯。
- **判断吉神须看其所生所护的是谁** → 吉神若去帮忌神 → 本质上就是破格之神。
- **吉神破格多发生在“二线逆转”** → 主线已成时，又来一吉神改了生克方向 → 比单纯凶神直破更隐蔽。

### 格局成败
- 成格条件：吉神必须顺着主线流通，用财则财生官，用印则印护官，用食则食制杀或生财。
- 破格条件：吉神去生忌、护忌、杂局；尤其“财助杀”“印护杀”“食杂官”最为典型。
- 救应方法：看有无隔字、合神、会局能重新改线；若不能，则宁可去掉这颗吉神，也不宜强留。

### 案例拆解
- 局式例1：**食神带杀而透财**
  - 判法：食本制杀，财来生杀。
  - 结论：财由吉转凶，属“吉神破格”。
- 局式例2：**春木火旺而见官**
  - 判法：木火通明重在秀气，不喜官来拘束。
  - 结论：官虽吉神，在此局反成病。
- 局式例3：**财旺生官而露食**
  - 判法：食神加入后，官路不纯。
  - 结论：食本吉，因杂官而破格。

### 例外和易错点
- 易错一：见财官印食就加分，不先问它到底帮谁。
- 易错二：把“吉神”当作道德标签，不把它放回具体生克链条。
- 易错三：看见食神就说“温和福厚”，忽略它在官格、杀格里可能成为杂神。

### 章际关联
- 与第19章对照：本章说“吉神会坏事”，下一章说“凶神能成事”，合起来才是子平法。
- 与第14章互证：所谓吉神成败，也受季节气候校正。
- 与第20章相关：吉神若先后次序不同，破格轻重也不同。

## 第19章 论四凶神能成格

### 原文金句
- “煞傷梟刃，四凶神也，然施之得宜，亦能成格。”
- “財不忌傷，官不忌梟，煞不忌刃。”

### 核心规则
- **凶神不等于必凶** → 七杀、伤官、枭印、阳刃只要用得其所 → 反能成为解局之药。
- **印轻可借杀生印** → 印根轻薄而无力 → 透杀反能资印成格。
- **财逢比劫可借伤官解围** → 比劫夺财时 → 伤官泄劫生财，可作救神。
- **食神带杀可借枭印发用** → 枭印若能辅助制化、通其关节 → 不必一概视为夺食。
- **财逢七杀可借刃挡灾** → 阳刃若能抗杀、扶身、存财 → 反成成格之力。
- **看凶神先问病点** → 原局病在何处 → 凶神若恰好攻病，就是药。
- **财不忌伤** → 伤官若能生财、解劫，不以常法论坏。
- **官不忌枭** → 枭若能护官、化局，不可以“偏印必怪”轻断。
- **杀不忌刃** → 刃若足以敌杀、成权，反主威名。

### 格局成败
- 成格条件：凶神出现后能闭合制化链条，或扶起原本无力的主线。
- 破格条件：凶神若无节制、无去处，仍旧按凶论；关键不在名字，而在有没有“施之得宜”。
- 救应方法：把凶神放进全局看其职责，是泄、制、化、扶哪一环；只要确在救局，即可转为成格之神。

### 案例拆解
- 局式例1：**印绶根轻，透杀为助**
  - 判法：杀生印，印得源头。
  - 结论：七杀由凶转助，成印格之权。
- 局式例2：**财逢比劫，伤官可解**
  - 判法：比劫夺财，伤官泄比并生财。
  - 结论：伤官非但不凶，反是救财之神。
- 局式例3：**财逢七杀，刃可解厄**
  - 判法：杀重攻身，刃来帮身抗杀。
  - 结论：刃成为保命成局之力。

### 例外和易错点
- 易错一：见七杀、伤官、阳刃就先下凶断，根本不问其是否救局。
- 易错二：把“枭印得用”与“枭神夺食”混为一谈，忽略局面不同。
- 易错三：只记口诀，不追问原局病机，就无法理解“凶神成格”。

### 章际关联
- 与第18章组成对子：吉凶神皆非定性，关键在“得位得用”。
- 与《十神关系速查》中的“食神制杀、羊刃驾杀、伤官生财”互证。
- 与第25章相关：凶神成格之后，行运若再扶凶神，未必凶，仍要看其职责。

## 第20章 论生克先后分吉凶

### 原文金句
- “有同此生克，而先後之間，遂分吉凶者，尤談命之奧也。”

### 核心规则
- **同样组合，先后不同，结局不同** → 十神不是只看有没有 → 更要看谁先到位、谁后破坏。
- **官格财伤并透先财后伤较可救** → 先财生官，再有伤官 → 虽不大贵，后运尚有结局。
- **官格财伤并透先伤后财终局差** → 先有伤官破官，后财再来已难挽回 → 多主终局无成。
- **印格财印并见先财后印尚可** → 己先癸后 → 晚景稍顺。
- **印格先印后财晚景差** → 癸先己后 → 末路衰悴，财坏印力更甚。
- **食神财枭并透先枭后财可望晚亨** → 庚先丙后 → 晚运较好。
- **食神财枭并透先财后枭富贵两空** → 丙先庚后 → 枭来夺食，后运转淡。
- **杀格财食并透先财后食成贵** → 癸先辛后 → 财助杀而后以食制杀，层次高。
- **杀格先食后财反助杀为病** → 辛先癸后 → 财转去生杀、破食之功。
- **隔字能缓冲** → 中间有一字隔开 → 即使两神相克，也可能减力或变线。
- **位置次序可推晚运早运** → 年先时后，不仅关系格局，也对应人生阶段先顺后逆或先逆后顺。

### 格局成败
- 成格条件：先出现的字为后出现的字铺路、解病、护主线；次序顺者，气脉顺。
- 破格条件：先出现的字先破主线，后来的吉神无法追回；尤其官格先伤、印格后财、杀格后财皆凶。
- 救应方法：看有无“间字”隔开冲突，或后字能不能真正改线，而非表面补救。

### 案例拆解
- 局式例1：**甲用酉官，丁先戊后 / 戊先丁后**
  - 判法：前者财能解伤，后者官先得财后又遭伤。
  - 结论：同是财伤并透，前者尚有收局，后者终局难好。
- 局式例2：**甲用子印，己先癸后 / 癸先己后**
  - 判法：前者后印仍可收晚境，后者晚景为财所坏。
  - 结论：印格极重先后。
- 局式例3：**己生卯月，癸先辛后 / 辛先癸后**
  - 判法：前者财先助杀，后食制杀；后者先制后又被财党杀。
  - 结论：杀格中“先成后破”比“先助后制”差得多。
- 局式例4：**丙生辛酉，年癸时己**
  - 判法：伤官因财隔断而力轻。
  - 结论：隔字可减破格之力。
- 局式例5：**辛生申月，年壬月戊，时上丙官**
  - 判法：戊隔壬，不使壬越戊克丙。
  - 结论：有间隔则官可保，格局仍可许贵。

### 例外和易错点
- 易错一：把年月日时当静态陈列，不把它们当“出场顺序”。
- 易错二：只说“财伤并透”“财印并见”，不再细问谁先谁后。
- 易错三：忽略中间隔字的缓冲作用。
- 易错四：把“同时出现”误当“力量同步发生”。

### 章际关联
- 与第15章相通：相神如果来得太晚，也未必救得回格局。
- 与第25章相通：大运流年进入原局，实际上也是“新的先后次序”。
- 与第26章相关：成格变格，本质就是运把顺序和主线改写了。

## 第21章 论星辰无关格局

### 原文金句
- “八字格局，專以月令配四柱，至於星辰好歹，既不能為生克之用，又何以操成敗之權？”
- “格局既成，即使滿盤孤辰入煞，何損其貴？”

### 核心规则
- **格局主权在月令四柱，不在神煞星辰** → 星辰既不能参加生克 → 就不能主导格局成败。
- **局有碍则吉星无功** → 若财官本身都救不了破局 → 更不必指望天德月德之类翻盘。
- **局有用则凶神不凶** → 七杀伤官若正好成格 → 不可因其名凶而轻断。
- **格成不怕孤辰入煞** → 神煞恶名不能推翻已成的格局主线。
- **格破不靠贵人翻案** → 满盘贵人德秀，也不能替代被破的月令用神。
- **古书“禄贵”多指正官** → 不一定是后世神煞体系中的禄堂贵人。
- **“得禄避位”应作得官避位** → 若格局本忌官运、重官运 → 遇官乡反宜退避，不是神煞意义上的“躲贵人”。
- **女命“贵众”多指官众** → 官多则夫多，故主不专一 → 不是说贵人神煞太多就为娼。
- **贵人二德只可作辅助** → 如衣服之于容貌 → 财官成格才是根本，德贵只是增色或险中稍救。

### 格局成败
- 成格条件：月令配四柱的生克链条清楚成立，神煞最多锦上添花。
- 破格条件：舍用神而逐吉星，把神煞当主断法；这是“抛却用神”的大病。
- 救应方法：先重建格局主线，再谈神煞能否助其显现；辅助只能辅助，不能顶替。

### 案例拆解
- 义例1：**“正财得伤贵为奇”**
  - 解释：此处“伤贵”是伤官生财之贵，不是“得了某颗贵人星”。
  - 结论：古书词义必须回到格局语境。
- 义例2：**“贵众则舞裙歌扇”**
  - 解释：此“贵”指官多，女命官多夫杂。
  - 结论：误解“贵”为贵人神煞，就会全篇走偏。
- 义例3：**“贵人头上带财官”**
  - 解释：财官像容貌，贵人像衣饰；衣好不如貌好。
  - 结论：财官成格为本，贵人为末。

### 例外和易错点
- 易错一：把古书里所有“禄”“贵”“德”都机械翻成神煞术语。
- 易错二：先看吉星再看格局，轻重倒置。
- 易错三：把“神煞有救”误读成“神煞可以改格”。

### 章际关联
- 与第22章外格用舍相连：一个是误入神煞，一个是误入外格，都是舍本逐末。
- 与第30章相连：后世以讹传讹，多由望文生义而起。
- 与《断命流程指南》一致：经典引用优先级中，格局、用神永远先于神煞。

## 第22章 论外格用舍

### 原文金句
- “外格者，蓋因月令無用，權而用之，故曰外格也。”
- “提綱有用提綱重，又曰有官莫尋格局，不易之論也。”

### 核心规则
- **外格只在月令无用时启用** → 月令本无可立之神 → 方可权取外格。
- **月令为本，外格为末** → 外格永远是补救法，不是优先法。
- **春木冬水、土生四季且日月同气时可考虑外格** → 常规格局难立 → 才轮到类象、属象、冲财、会禄等。
- **月令自有用神不可另寻外格** → 提纲能立官财杀印食伤 → 就应守本格。
- **天干已有财官杀不可弃本逐末** → 干头现成可用 → 何必再去追冲财、合禄等奇法。
- **“有官莫寻格局”是总纲** → 官已可用 → 不必再往外格上绕。
- **用神已破不等于月令无用** → 财被劫、官被伤 → 这是“用神破”，不是“月令无取”。
- **外格误用多因偷懒** → 不愿细究月令成败救应 → 便拿奇格名字代替分析。

### 格局成败
- 成格条件：月令确实无神可取，且外格所依的类象、会拱、刑合等能完整成立。
- 破格条件：月令已可立格却硬转外格；或用神被破就误称“月令无用”，都是大误。
- 救应方法：先回到月令重审。确认真无用，再取外格；若只是破格，应按救应法修补，不该跳格。

### 案例拆解
- 义例1：**干头有财，何用冲财**
  - 判法：财已明透，可直接立财格。
  - 结论：再去求“冲财格”，属舍近求远。
- 义例2：**干头有官，何用合禄**
  - 判法：官既现前，已可成主线。
  - 结论：再套合禄之名，只会模糊本格。
- 义例3：**月令财被劫、官被伤**
  - 判法：这是用神破损。
  - 结论：应论救应，不应宣布“月令无用”而改投外格。

### 例外和易错点
- 易错一：见到奇格名称就兴奋，忽略月令还在。
- 易错二：把“月令无用”和“月令用神被破”混为一谈。
- 易错三：外格只要像一点就认，不检查是否完整成立。

### 章际关联
- 与第21章同病：都是放弃月令主线。
- 与第29、30章直接呼应：时格、奇格、俗格之滥用，多从这一章能得到纠偏。
- 与第9章成败救应互证：本格破了先求救，不是先跳格。

## 第23章 论宫分用神配六亲

### 原文金句
- “人有六親，配之八字，亦存於命。”
- “其間有無得力，或吉或凶，則以四柱所存……然後以六親配之用神。”

### 核心规则
- **六亲既看宫分，也看十神** → 宫分定位置，应事落点；十神定关系性质。
- **年月日時自上而下配六亲** → 年月偏祖上父母，日主自身与配偶，时柱偏子女晚景。
- **正印为母** → 生我者为母，故以印论。
- **偏财为父** → 偏财为母之夫，故以偏财论父。
- **正财为妻** → 我所克而妻从夫纲，故以财论妻。
- **官杀为子女** → 子女为财所生，故从妻财一路推出官杀。
- **比肩为兄弟** → 同类同气，理最显然。
- **六亲吉凶须并看其是否为用** → 同样是财星，为妻也可能为忌；同样是印星，为母也可能无力。
- **离身越近，应验越灵** → 本身、配偶、子女为切身之事，常最应。
- **提纲得力、年干有用，多主父母得力** → 因年、月与父母源流最近。
- **祖宗兄弟相较不甚验** → 距身较远，信息较弱。

### 格局成败
- 成格条件：六亲之星恰为用神，且所居宫位得力、不受重伤，则该亲多吉而能助命。
- 破格条件：六亲之星虽现，却为忌神、被冲克、或落空位，则有名无实，甚至反成拖累。
- 救应方法：六亲判断不能离开全局喜忌；先定主格，再看六亲星是在扶格还是拆格。

### 案例拆解
- 义例1：**年干有用、提纲得力**
  - 判法：年、月对应父母源流，且用神落在其位。
  - 结论：多主父母双全得力。
- 义例2：**财官伤刃落在不同柱位**
  - 判法：同一十神落年、月、日、时，应验亲属不同。
  - 结论：六亲必须“宫位+十神”联判。

### 例外和易错点
- 易错一：只按十神背“印母财父官子”，不看宫位。
- 易错二：只看宫位，不管这颗星在全局中是喜是忌。
- 易错三：把祖上、兄弟、父母、配偶、子女全部等量齐观，忽略“愈近愈验”。

### 章际关联
- 承接第22章：回到正法之后，开始把格局推入六亲应事。
- 与第24章相连：本章立总法，下一章专论妻子与子息。
- 与《断命流程指南》相通：十神不能只贴标签，必须分析在命局中的真实作用。

## 第24章 论妻子

### 原文金句
- “既看妻宮，又看妻星。”
- “其理不可執一。”

### 核心规则
- **论妻先看妻宫** → 日支所坐为何物 → 是婚姻状态第一观察点。
- **妻宫坐财官多主贤贵** → 日支见财官 → 常主妻能成家助夫。
- **妻宫虽吉仍要配格局喜忌** → 妻宫坐财，若逢印格，反未必美；妻宫坐官，若入伤官格，反多不顺。
- **妻宫坐伤官未必尽凶** → 在财格可生财，在杀格可制杀 → 反主内助。
- **妻宫坐阳刃未必尽凶** → 若全局已成格而日主无气，全靠刃扶身 → 妻反能相关。
- **论妻再看妻星** → 妻星为天干所透之财 → 看其是否成局、破局。
- **妻星透而成局，主内助** → 如官格透财、印多逢财、食伤透财为用 → 即便妻宫平平，仍主妻能助夫。
- **妻星透而破格，防刑克** → 印轻财露、食神伤官、透杀逢财等 → 即便妻宫好，也防婚姻受损。
- **妻宫有用而坐下刑冲，难偕老** → 得美妻不等于能长久。
- **偏正财两透，多主情缘杂或刑克** → “一夫多妻”之象，现代可作情感复杂、婚缘不稳论。
- **论子先看时支** → 时支为子息宫，配长生死绝先判多寡。
- **子息多寡先用阳长生法** → 如甲乙日同以庚为男、辛为女，不可拘泥阴阳逆顺成男女两套。
- **时干子星得用，虽死绝亦可得贵子** → 但数量不繁。
- **时干子星破局，纵使时支生旺也难为子息** → 若再逢死绝，更少希望。

### 格局成败
- 成格条件：妻宫、妻星至少有一处真正扶格；子息宫、子星同样要与全局喜忌一致。
- 破格条件：妻星成破格之神、妻宫遭刑冲；时干子星破局、时支死绝又无救，应在婚姻与子嗣上多见不足。
- 救应方法：婚姻看“宫星并看”，子息看“时支定数+时干定贵贱”；一处有病，要看另一处能否补。

### 案例拆解
- 义例1：**妻宫坐财而命成印格**
  - 判法：财在妻宫本吉，但对印格属坏印之象。
  - 结论：婚姻未必顺美。
- 义例2：**妻宫坐伤官而命成财格**
  - 判法：伤官可生财。
  - 结论：反主妻能内助。
- 义例3：**财格而时干透食**
  - 判法：时干子星有用，即使时支死绝。
  - 结论：仍主子贵，只是不繁。
- 义例4：**时干子星破局且时支死绝**
  - 判法：宫与星双败。
  - 结论：子息艰难。

### 例外和易错点
- 易错一：见日支财官就断“必得贤妻”，不看全局格局喜忌。
- 易错二：见日支伤官、阳刃就断“克妻”，忽略它可能正好生财、制杀、帮身。
- 易错三：论子息只背长生歌，不看时干子星是否得用。
- 易错四：男女子息法拘泥阴阳逆顺，原文明确主张统一从日主取阳法。

### 章际关联
- 由第23章六亲总论展开到婚姻、子女的具体断法。

## 第25章 论行运

### 原文金句
- “論運與看命無二法也。”
- “故運中每運行一字，即必以此一字，配命中干支而統觀之。”

### 核心规则
- **看运与看命同法** → 原局以月令配四柱 → 行运则以运干支重新配原局喜忌。
- **喜运是助所喜之神** → 官用印而运助印、财生官而身轻运助身、月劫用财而运行伤食，皆属美运。
- **忌运是逆所喜之神** → 正官无印而运伤、食神带杀而运财、伤官佩印而运财，皆属败运。
- **似喜实忌须复核原局** → 官逢印运，若原局另有合绊，印未必真能护官。
- **似忌实喜也常见** → 官逢伤运，若命透印，则伤未必破官，反可去病。
- **有的运只行干不行支** → 如丙生子月亥年，丙丁能帮身，巳午却主冲。
- **有的运只行支不行干** → 甲生酉月，申酉可植官根，庚辛却易混杀。
- **同类之干未必都吉凶一致** → 丁亥月年透壬官，逢丙帮身，逢丁反合官。
- **同类之支也不必同论** → 戊生卯月丑年，逢申为长生，逢酉则会丑成伤。
- **冲年月应快，冲日時应缓** → 早年根基之处被冲，应事较急；晚年、配偶、子女之处被冲，常稍后显。
- **美运逢冲为轻，败运逢冲为重** → 冲只决定加速度，不决定善恶本质。
- **有逢冲而不冲者** → 本命已有会合解救，则冲力被卸。
- **有一冲成两冲者** → 运与原局凑成复数之冲，则应力倍增。

### 格局成败
- 成格条件：运来扶用、通关、补根、去忌，且作用点正落在命局病处。
- 破格条件：运来合去用神、冲动忌神、扶忌党病；表面同类，也未必同吉。
- 救应方法：运必须逐字拆开看，不可只说“走财运”“走官运”；先看它进局后到底站哪边。

### 案例拆解
- 局式例1：**官用印以制伤，而运助印**
  - 判法：运直接扶持护官主线。
  - 结论：美运。
- 局式例2：**七杀食制，而运逢枭**
  - 判法：枭夺食，食不能制杀。
  - 结论：败运。
- 局式例3：**甲用酉官，行卯而本命巳酉相会**
  - 判法：卯冲酉，但巳酉会金，冲力减弱。
  - 结论：属于“逢冲而不冲”。
- 局式例4：**乙用申官，原局两申，运又逢寅**
  - 判法：运与命合成二寅冲二申。
  - 结论：属于“一冲得两冲”。

### 例外和易错点
- 易错一：把运看成五行数量增减，不看它与原局哪一字接上。
- 易错二：见同类干支就一概论，忽略“同中有异”。
- 易错三：只看冲，不看本命是否已有会合化解。
- 易错四：把“喜运逢冲”与“忌运逢冲”混同处理。

### 章际关联
- 与第20章相连：大运就是外来新字，继续重演“先后分吉凶”。
- 与第26章直接相接：除了普通喜忌，运还能改格。
- 与第27、28章配合：行干、行支的分别，正是干支有别与透清法的动态应用。

## 第26章 论行运成格变格

### 原文金句
- “命之格局，成於八字，然配之以運，亦有成格變格之要權。”
- “其成格變格，較之喜忌禍福尤重。”

### 核心规则
- **运能补成原局未完成之格** → 本命用神有其半而未全 → 运来会、透、补根，即可成格。
- **成格重在补最后一口气** → 或会局未成，或用神未透，运若补足，即从半成品转正格。
- **运也能变格** → 运把月令暗藏忌神透出，或把清局会成别象 → 原格可能被改写。
- **变格比普通吉凶更重** → 因为不是某件事好坏，而是命局主线整个转向。
- **成格之运未必全喜** → 若原局另有破点，如甲乙混入官路 → 运虽补格，仍可能成而不真。
- **变格之运未必尽凶** → 若原局早有救神，可把变格之害降下来。
- **运判成变，要看它引出的是月令何物** → 透出暗伤官、暗七杀、暗印星，结果各异。

### 格局成败
- 成格条件：运补的是正用，不是杂神；补后全局方向更清而非更浊。
- 破格条件：运把月令中潜伏的忌神翻出，或把原有主线改成别的忌线。
- 救应方法：成格还要复核本命有无掣肘，变格则要追查原局有无先天救神。

### 案例拆解
- 局式例1：**丁生辰月，透壬为官，而运逢申子**
  - 判法：原局官透而未全，运来会成水局。
  - 结论：属于成格。
- 局式例2：**乙生辰月，会申或子成印局，而运逢壬癸透**
  - 判法：原局支成印意，运再透印。
  - 结论：成格更真。
- 局式例3：**丁生辰月，透壬用官，而运逢戌**
  - 判法：戌冲辰，透出辰中伤官。
  - 结论：由官格转坏，属变格。
- 局式例4：**壬生亥月，透己用官，运逢卯未**
  - 判法：卯未会亥成木，化建禄为伤。
  - 结论：原局主线被改写，属变格。

### 例外和易错点
- 易错一：把成格变格只当应期，不当结构变化看。
- 易错二：看到运来补一神就说吉，不看补的是不是正用。
- 易错三：看到运来会局就说凶，不问原局有无预设救应。

### 章际关联
- 承接第25章：在一般喜忌之上，再上一层看“改格权”。
- 与第16章、20章互通：成格变格多见于杂气月、多神并透和先后更替。
- 为第27、28章铺路：变格常经由支藏透出、合会发动而成。

## 第27章 论喜忌干支有别

### 原文金句
- “干主天，動而有為；支主地，靜以待用。”
- “為福為禍，安不得殊？”

### 核心规则
- **天干主动，地支主静** → 干上之神多即刻发用，支中之气多待发动。
- **天干主一，地支藏多** → 一干性质较单，支中常兼数气，故喜忌不可等量齐观。
- **同属金，庚辛与申酉不同** → 庚辛透干是官杀明现，申酉只是其根气、旺地。
- **甲用酉官，逢庚辛为官杀杂** → 因庚辛直接上天干主事。
- **甲用酉官，逢申酉不作官杀杂** → 申酉虽助辛根，但不是另一重明官。
- **逢二辛为官犯重，逢二酉不作此例** → 两辛是两重官，二酉只是官根加厚。
- **透丁为伤，逢午不必同论** → 丁透干主动犯官，午支静藏丁己，性质不专。
- **地支作祸福须待发动** → 如寅戌会火动局，才可能伤官。
- **干支同类不可视作一件事** → 明神与暗根、现象与潜能，本来不同层级。

### 格局成败
- 成格条件：明干与暗支各司其职，支能为干立根而不乱局，干能借支发用而不重浊。
- 破格条件：把支当干、把根当主神、把藏气当明气，容易误判官杀混杂、伤官犯官等。
- 救应方法：先分“明字”“暗根”，再看是否发动；支未动，只作根气参考，不轻断即应。

### 案例拆解
- 局式例1：**甲用酉官，逢庚辛 / 逢申酉**
  - 判法：前者明见官杀混，后者仅增官根。
  - 结论：同属金而喜忌不同。
- 局式例2：**辛坐申酉 / 二辛并透**
  - 判法：辛坐申酉如一官得实地；二辛并透则变成官犯重。
  - 结论：根旺不等于官重。
- 局式例3：**甲用酉官，透丁 / 逢午**
  - 判法：丁透为明伤，午中虽有丁但未必发。
  - 结论：支藏不能直接等同天干透出。

### 例外和易错点
- 易错一：见某五行在支多，就直接按“透干之神”断。
- 易错二：把“官根厚”误断成“重官为病”。
- 易错三：不分明气暗气，导致吉凶过断。

### 章际关联
- 与第28章紧密衔接：既然支静待用，就要看何时被“透清”。
- 与第25章互证：为什么有的运只行干、有的只行支。
- 与第21章同理：判断不能停在名目，要回到生克层级。

## 第28章 论支中喜忌逢运透清

### 原文金句
- “而運逢透清，則靜而待用者，正得其用，而喜忌之驗，於此乃見。”
- “支為干之生地，干為支之發用。”

### 核心规则
- **支中喜忌常处潜伏状态** → 平时只作根气、背景、潜能 → 未必立即应事。
- **逢运透清则潜能变现实** → 大运流年把支中所藏透出天干 → 吉凶立现。
- **辰未可作财，午未可作伤，须看运透何干** → 支本身不必然定性 → 透出戊则财，透出丁则伤。
- **看一干，必倒查四支有无其根** → 干无根，多浮；干有根，力实。
- **看一支，必反查四干谁能发用** → 同一亥支，用壬则为禄，用甲则为长生。
- **一支可兼作两根** → 亥逢壬甲俱全时，可同时为壬之禄、甲之长生。
- **逐干逐支、上下统看是总法** → 不可只看明面四干，亦不可只背藏干表。
- **取运也用同法** → 运来之字加入后，同样要逐干逐支找根与发用关系。

### 格局成败
- 成格条件：支中所藏在适当运岁透出，恰好扶主线、补用神、清格局。
- 破格条件：支中暗藏之忌在运岁被透，原本潜伏的病一下发作。
- 救应方法：平时断命先预判“哪些支将来一透就会出事/出喜”，再结合运岁推应期。

### 案例拆解
- 局式例1：**甲用酉官，逢辰未，运透戊**
  - 判法：辰未中土被运透戊，财星由暗转明。
  - 结论：属“支中财逢运透清”。
- 局式例2：**甲用酉官，逢午未，运透丁**
  - 判法：午未所藏之火因丁透而明。
  - 结论：伤官由静转动，可能伤官犯官。
- 义例3：**命中有甲字，查支中寅亥卯未**
  - 判法：凡见其一，皆为甲根。
  - 结论：干是否有根，决定其可不可任事。
- 义例4：**命中有亥字，干有壬或甲**
  - 判法：壬用亥作禄，甲用亥作长生。
  - 结论：同一支的发用对象不同，功能不同。

### 例外和易错点
- 易错一：把支里藏干一出现就当已经发事。
- 易错二：看见某支就固定说它是某十神，不问运来透清的是谁。
- 易错三：只在原局找根，不把运也纳入“逐干逐支”的检查。

### 章际关联
- 承接第27章：把“支静待用”具体化为“逢运透清”。
- 与第25、26章相通：很多运的成败与变格，都靠这一透。
- 与第17章互证：库气不是只靠冲，也常靠透而显。

## 第29章 论时说拘泥格局

### 原文金句
- “八字用神專憑月令，月無用神，徐尋格局。月令，本也；外格，末也。”

### 核心规则
- **时格不得推翻月令本格** → 月令能立格 → 时柱再奇，也只能辅佐，不可篡位。
- **明杀有制不可误作专食格** → 戊生甲寅月，时上庚甲 → 本是杀有制，不当改判“专食”。
- **正官格不可误作归禄格** → 丙生子月，时逢巳禄 → 本应论正官配归禄帮身，不应说“日禄归时逢官破局”。
- **官印格不可误作朝阳格** → 辛日透丙，时遇戊子 → 本是得官逢印，不必硬扯朝阳。
- **财逢时杀应看生杀攻身** → 不可见“时上偏官”口诀就改主线。
- **暗官受破不可硬作刑合成格** → 癸生巳月，时遇甲寅，应先辨暗官受损。
- **月劫建禄用官通根不可误作拱格** → 癸冬月酉日亥时透戊坐戌，不该硬说拱戌填实。
- **已有正财格不可再扯填实拱贵** → 辛日坐丑，寅年亥月卯时，本格已清。
- **木火通明不可误作鼠贵** → 乙逢寅月，时遇丙子，应守木火通明之正论。
- **凡“某时成某格”之说，先回头查月令** → 有本格时，时格只能降为枝节。

### 格局成败
- 成格条件：月令主线先立住，再看时柱是帮成、帮破，还是仅增色。
- 破格条件：见时柱有奇象就舍本逐末，造成正格被抹掉、全局被误判。
- 救应方法：凡遇时格口诀，先做“回归月令”检查；月令若已有财官印食杀可用，时格一律降级处理。

### 案例拆解
- 局式例1：**戊生甲寅之月，时上庚甲**
  - 正判：明杀有制。
  - 误判：专食之格。
  - 结论：典型“因时格误正格”。
- 局式例2：**丙生子月，时逢巳禄**
  - 正判：正官格，归禄帮身。
  - 误判：日禄归时，逢官破局。
  - 结论：时格不可压月令。
- 局式例3：**辛日透丙，时遇戊子**
  - 正判：得官逢印。
  - 误判：朝阳格。
  - 结论：奇名好听，不如本格可靠。

### 例外和易错点
- 易错一：把时柱口诀当高级断法，反而把月令本法丢掉。
- 易错二：见“归禄”“朝阳”“鼠贵”等名目就强行套格。
- 易错三：把时柱视为能改命局中心，而不是局部条件。

### 章际关联
- 承接第22章：这是“外格用舍”在时柱层面的具体纠偏。
- 与第30章衔接：时格之误一旦流传，就会变成以讹传讹。
- 与第21章同理：名字再漂亮，也不能代替格局分析。

## 第30章 论时说以讹传讹

### 原文金句
- “八字本有定理，理之不明，遂生異端，妄言妄聽，牢不可破。”
- “若看命者不究其本，而徒以彼既富貴遷就其說以相從，無惑乎終身無解日矣！”

### 核心规则
- **异端多起于理不明** → 不懂阴阳、生克、月令本法 → 最容易生出奇谈怪论。
- **俗书歌诀不可代替原理** → 只背体象、口号，不明阴阳根本 → 易误入歧途。
- **拘泥外格是假活变** → 不知专寻月令，却以奇格为高明。
- **扶强抑弱不可当死法** → 不察具体喜忌，单拿旺衰口号断命，必多误。
- **行运不能把同类干支一概而论** → 不问“同中有异”，就会把运判粗糙化。
- **误读经典词义会生偏见** → 书中字重轻重若不懂，容易从一句话走偏整套法。
- **伪书、俗书害人甚深** → 以无知妄作之说冒充经典，后学最易误信。
- **偶然应验不可升格为通法** → 只撞对一次，就把偏法当铁律，是命理大忌。
- **古命例常有时辰失真** → 后人又喜按贵命硬改时辰，最易造出假贵格。
- **见富贵不可反推理论** → 不能因为某人已贵，就迁就出一套支持他的格名。
- **误收格局与硬入外格是两大祸根** → 一是原式收错，一是明明本格自美，偏去外求。

### 格局成败
- 成格条件：守住月令、阴阳、生克、喜忌、运法五大正脉，再谈案例验证。
- 破格条件：望文生义、听信俗书、以贵命迁就奇格、拿偶验当通法。
- 救应方法：凡见奇论，先反查四点：月令是否可立、喜忌是否一致、时辰是否可靠、命例是否误收。

### 案例拆解
- 命例1：**壬申、癸丑、己丑、甲戌**
  - 原判：杂气财旺生官。
  - 误判：改作乙亥时，论时上偏官。
  - 问题：旺财生杀，已自顾不暇，何来取贵。
  - 结论：属于误收格局。
- 命例2：**己未、壬申、戊子、庚申**
  - 原判：食神生财。
  - 误判：弃月令，硬作戊日庚申合禄格。
  - 问题：本局自有财食之美，无须局外求官。
  - 结论：属于硬入外格。
- 通案义例：**贵命时辰未确**
  - 现象：术者探得生日，再按自己理解添配时辰。
  - 后果：格名愈奇，离事实愈远。
  - 结论：命例可信度先于案例名气。

### 例外和易错点
- 易错一：因为某种说法“很像”“很巧”就信，不回头查原理。
- 易错二：把富贵结果当理论起点，倒推时辰、倒推格局。
- 易错三：把偶然撞中的偏法当成放之四海而皆准的秘诀。

### 章际关联
- 是第21、22、29章的总收束：神煞、外格、时格三类错误，最后都会发展成“以讹传讹”。
- 也是全段第14-30章的方法论总结：调候、相神、杂气、行运、干支、六亲，全部都必须回到“月令主线+生克原理”。
- 对后续阅读正官、财、印、食、杀诸章有总纲意义：不守本法，后面所有格局章都会被读歪。

## 第31章 论正官

### 原文金句
- “正官者分所當尊，如在國有君，在家有親，刑衝破害，以下犯上，烏乎可乎？”
- “然而遇傷在於佩印，混煞貴乎取清。”

### 核心规则
1. **官星重尊** → 正官为秩序、名分、法度之神 → 最忌刑冲破害，见之多主地位受损、名节受伤。
2. **官喜生护** → 既然官忌伤害，则凡财生官、印护官皆为喜 → 官星得生得护，格局易贵。
3. **财印并透最上** → 官格中财印同时透出，且互不妨碍 → 既有财生官，又有印护身护官，为大贵之象。
4. **财印相碍降格** → 官格中财印虽同见，但财印相合相战 → 官失左右辅弼，只成“孤官无辅”，贵而不高。
5. **官格单用财优于单用印** → 单用印虽能护官，但亦能泄官；单用财则专事生官 → 官格一般以财为第一助神。
6. **化官为印而透财更秀** → 官能化印，印又得财流通 → 官、印、财三者成连续生化链，格局层次极高。
7. **伤官犯官须佩印** → 官格见伤，不可硬扛 → 必用印星制伤、护官、安身，方能转祸为福。
8. **官杀混杂必须取清** → 官格混入七杀，不可官杀并存并用 → 要么合杀留官，要么制杀存官。
9. **合杀留官属正法** → 杀混官而有天干合去其杀 → 不以“有杀”为忌，反以“去杂还清”为贵。
10. **财来去印未必一定破格** → 若局中印多而能分工，一印制伤、一印合财，各司其职 → 财未必坏印，反能成清官格。
11. **官格判断先看清纯后看旺衰** → 官星是否受伤、受混、受冲，比单纯官旺官弱更重要 → 先保“清”，再论“力”。
12. **官格贵在有情有辅** → 官孤无辅、官清而寒，虽正不显 → 必有财印配合，方能发为实贵。

### 格局成败
- 成格条件
  - 月令官星得位，且不遭刑冲破害。
  - 官星清纯，不与七杀并透混杂。
  - 财印至少得其一助官，最好财印并透而不相碍。
  - 日主不至太弱，能任官气；若身轻，则须印扶。
  - 见伤官时，有印星护官制伤。
- 破格条件
  - 伤官明透直犯官星。
  - 七杀混杂而无合无制。
  - 财印相战，致官失财生印护。
  - 官星遭地支刑冲破害。
  - 官多成重官，反主压身、招灾、失衡。
- 救应方法
  - **伤官犯官**：用印制伤护官。
  - **官杀混杂**：合杀留官，或制杀存官。
  - **官轻身轻**：扶身兼护官。
  - **印多碍财**：以有情之财疏印，但必须不致坏印。

### 案例拆解
- **薛相公命**：`甲申 壬申 乙巳 戊寅`
  - 格局：正官格配财印。
  - 关键：乙日见申中官，壬印、戊财并透，中隔乙木，水土不相战。
  - 结论：财生官、印护身，且财印两不相碍，所以“大贵”。
- **壬戌 丁未 戊甲 乙卯**（原文如此，第三柱疑有讹字）
  - 格局：杂气正官格。
  - 关键：虽“透干会支，最为贵格”，但壬财与丁印相合，财印不能两用。
  - 结论：官成孤立之象，所以“不上七品”，为典型的有官而辅神失位。
- **金状元命**：`乙卯 丁亥 丁未 庚戌`
  - 格局：官格并用财印。
  - 关键：无伤官、无杂杀，去忌存喜。
  - 结论：官、印、财路纯而不杂，成大贵。
- **宣参国命**：`己卯 辛未 壬寅 辛亥`
  - 格局：官格遇伤佩印。
  - 关键：未中己官透出，支会木局伤官旺，两辛印星解伤护官。
  - 结论：遇伤不败，反因佩印成贵。
- **李参政命**：`庚寅 乙酉 甲子 戊辰`
  - 格局：官格混杀取清。
  - 关键：甲用酉官，庚金七杀混杂，乙木合庚。
  - 结论：合杀留官，属于“杂煞而取清”。
- **范太傅命**：`丁丑 壬寅 己巳 丙寅`
  - 格局：官格带伤，用印兼化财。
  - 关键：巳丑会金伤官，丙丁两透可分工；丁合壬去财，丙制伤留官。
  - 结论：看似透财破印，实则去财清官，为“无情而愈有情”的妙局。

### 例外和易错点
- 不可把“官格喜印”机械理解为“见印必贵”。印能护官，也能泄官；若印重而无财流通，未必高贵。
- 不可见财就判“财生官吉”。若财去合印、坏印，反会使官失护。
- 官杀混杂不是一概凶，关键看能否“取清”。
- 官格见伤也不是立即破尽，若有印隔伤制伤，反能成“伤官佩印”式的贵局。
- 对官格最容易误判之处，是只数十神，不看十神之间有没有分工、有没有隔位相安。

### 章际关联
- 与第32章直接相连：本章讲“官格怎样成”，下一章讲“成了之后运该怎么走”。
- 与第35章《论印绶》互证：官格靠印护，印格也可借官成贵，二者是一条双向通道。
- 与《十神关系速查》中的“官印相生”“伤官见官”“官杀混杂”完全对应，是断仕途、单位、编制类命局的核心母章。
- 与《滴天髓》重“官星有理会”的思路一致：官不在多，在清，在有理，在有护。

## 第32章 论正官取运

### 原文金句
- “正官而用財印，身稍輕則取助身，官稍輕則助官。”
- “正官佩印，運喜財鄉，傷食反吉。”

### 核心规则
1. **官格取运先分支** → 先看原局是用财、用印、带伤、带杀 → 不能把一切官格都按同一运法处理。
2. **官格第一看官与身谁轻** → 身轻则先扶身，官轻则先扶官 → 运法围绕“补短板”展开。
3. **官露忌合** → 官星明露最怕大运天干来合去 → 官被合，权柄名位被牵制。
4. **官露忌重官** → 原局已有正官，再逢运透重官 → 官多反为压身、拘束、是非。
5. **官露忌再混杀** → 正官局最怕行运透七杀 → 清格被搅浑，层次直降。
6. **官格也忌地支冲官** → 运支刑冲命局官星或月令官根 → 不论合成何局，先论不利。
7. **正官用财** → 喜印绶、身旺之地 → 忌食伤，因为食伤直接克官。
8. **正官用财但身旺财轻** → 若日主太旺、财轻官弱 → 仍宜再行财官运，补其主线。
9. **正官佩印** → 喜财运 → 财能制印之太过，使官印流通。
10. **官重身轻而佩印** → 此时重点不在求财，而在扶身 → 宜身旺运，不必执着财运。
11. **官带伤食而用印** → 喜官旺印旺 → 忌财运，因为财会坏印而护伤。
12. **印多时财运不忌** → 若印绶叠出，财来未必坏印，反能疏通。
13. **官带杀而用劫合杀** → 财运、伤食运、身旺运、印运皆可 → 但唯忌再露七杀。
14. **官带杀而用伤官合杀** → 财运、食伤运可行 → 不宜印运，因为印会扶身并牵制合杀之功。

### 格局成败
- 成格取运
  - 顺原局成格路线行运，即“原局怎么成，运就怎么扶”。
  - 官用财者，运助财官或助身。
  - 官佩印者，运以流通官印为主。
  - 官带伤而用印者，运务必保印清官。
- 破格行运
  - 运来合官、冲官、混杀、重官，皆易破局。
  - 财印官三者的原有分工，一旦被运打乱，贵局即变。
- 救应关键
  - 原局身轻者，扶身比“再加官”更重要。
  - 原局官轻者，助官比“盲目补印”更有效。

### 案例拆解
- **本章无新增四柱命例**，但可用前章局型推运：
  - **薛相公命型**：财印并用之官格，若行身旺印旺之地，更能扶官护身；若逢食伤强运，则有损官名。
  - **宣参国命型**：官带伤而用印，最怕财运坏印；若行印旺官旺之乡，则贵气持续。
  - **李参政命型**：合杀留官局，最忌运中再透七杀，破其“清”。

### 例外和易错点
- “官佩印喜财”不等于所有财运都吉，前提是财能疏印而不坏印。
- “官用财忌食伤”也要看原局有无印护；若官带杀、又以伤食合杀，则食伤反成用。
- 最常见误判是只看“喜神”，不看“喜神在这个分支里扮演什么角色”。
- 官格行运的核心不是五行数量，而是“清局是否还能保持”。

### 章际关联
- 承接第31章：本章把“官格成法”落实到运法。
- 与第34章、第36章、第38章结构相同：都是“先分局，再分运”。
- 与《断命流程指南》“格局 > 调候 > 强弱”的顺序吻合：官格取运，先守格局主线，再谈扶抑。

## 第33章 论财

### 原文金句
- “財喜根深，不宜太露，然透一位以清用，格所最喜，不為之露。”
- “旺而露傷，何苦用彼？徒使財遇傷而死生官之具，安望富貴乎？”

### 核心规则
1. **财以有根为贵** → 财星根深则真财可用 → 浮财无根，多为虚名虚利。
2. **财不宜太露** → 透一位以清用最佳 → 透多则露白，易受劫夺。
3. **财旺生官则不忌露** → 因财既生官，官可护财 → 好比府库有守，露而不失。
4. **财旺生官成贵局** → 条件是身强、不透伤官、不混七杀 → 财能顺流到官，为富贵双全。
5. **财用食生** → 身强、不露官、略带一位比劫最佳 → 比劫不是争财，而是帮身任财。
6. **财用食生最忌透官身弱** → 若官一透而身转弱 → 财不能专用，格局即坏。
7. **财格佩印** → 孤财不贵，须印帮身 → 财印并行则富而能贵。
8. **财印必须两不相碍** → 财印相战则难成大贵 → 纵有富处，也止于小富。
9. **财用食兼用印** → 食印不相碍，或为保暗官而去食 → 皆属贵格。
10. **财用伤官只在特殊情形可取** → 财不旺而比劫强，需透伤官化劫生财 → 属权用，不属常用。
11. **财旺无劫不可再透伤** → 伤官本非美物，若财本已足，再透伤只会损财、坏其生官之源。
12. **财带七杀可贵** → 或合杀存财，或制杀生财 → 财与杀的冲突被处理好，反成贵局。
13. **财用杀印** → 杀党为忌，印来化杀 → 财、杀、印形成特殊富贵结构。
14. **财用杀印最忌财杀并透而印孤** → 财去生杀，印又独弱 → 非但不贵，亦未必富。
15. **暗官可抬高财格** → 壬午、癸巳等月令，虽单透财，若月令藏有暗官，亦能成贵。
16. **有时须弃财就杀** → 劫刃太重，财难保 → 与其守财受夺，不如转从杀路。

### 格局成败
- 成格条件
  - 财有根、有源、有路可走。
  - 最佳路线是财旺生官，次则食生财、印扶身、杀印成局。
  - 日主须能任财，至少不能“财多身弱”。
  - 比劫、七杀、伤官若见，必须被合理安置。
- 破格条件
  - 财多身弱，富屋贫人。
  - 财露太多无官护，易遭劫夺。
  - 财印交战，印被坏则富而不贵。
  - 财旺再透伤，损财伤官，败其生官之具。
  - 财杀并透而无印化，财反党杀攻身。
- 救应方法
  - **财轻比重**：用食伤化劫生财，或用官制劫护财。
  - **财多身弱**：先扶身，再用财。
  - **财印相碍**：求隔位、求分工，不可混战。
  - **财带杀**：合杀存财或制杀生财。
  - **劫刃太重**：必要时弃财就杀。

### 案例拆解
- **葛参政命**：`壬申 壬子 戊午 乙卯`
  - 格局：财露而生官。
  - 关键：乙木财透似露，但因财能生官，官可护财。
  - 结论：此为“财露不忌”的代表局。
- **杨待郎命**：`壬寅 壬寅 庚辰 辛巳`
  - 格局：财用食生。
  - 关键：壬水食神透，庚金得身强，辛劫微露，反增有情。
  - 结论：财食相生而身能任之，为美格。
- **曾参政命**：`乙未 甲申 丙申 庚寅`
  - 格局：财格佩印。
  - 关键：孤财本不贵，赖印助身以承财。
  - 结论：财印并用，故能取贵。
- **乙未 己卯 庚寅 辛巳**
  - 格局：财印相碍之局。
  - 关键：乙财与己印“两不相能”。
  - 结论：有好处而不大，只主小富。
- **吴榜眼命**：`庚戌 戊子 戊子 丙辰`
  - 格局：财用食兼用印。
  - 关键：庚食与丙印，中隔两戊，不相妨碍。
  - 结论：食印两全，故贵。
- **平江伯命**：`壬辰 乙巳 癸巳 辛酉`
  - 格局：去食护官。
  - 关键：表面食印相克，实则为存巳中戊官而去食。
  - 结论：保住官根，比保食更重要。
- **汪学士命**：`甲子 辛未 辛酉 壬辰`
  - 格局：财用伤官。
  - 关键：甲财透于未库，辛金为劫，壬伤官化劫生财。
  - 结论：此为“不得已而用伤”的典型。
- **毛状元命**：`乙酉 庚辰 甲午 戊辰`
  - 格局：财带七杀，合杀存财。
  - 结论：杀不直攻，财得保全，故贵。
- **李御史命**：`庚辰 戊子 戊寅 甲寅`
  - 格局：财带七杀，制杀生财。
  - 结论：先处理杀，再保财，故能成格。
- **赵侍郎命**：`乙丑 丁亥 乙亥`（原文仅见三柱，疑脱一柱）
  - 格局：财用杀印。
  - 关键：印化杀兼有解冻之功，且“不露财以杂其印”。
  - 结论：杀印清纯，故贵。
- **林尚书命**：`丙寅 癸巳 癸未 壬戌`
  - 格局：单透财而贵。
  - 关键：月令有暗官，不可只看明干。
  - 结论：财格有暗官可抬层次。
- **王太仆命**：`丙辰 癸巳 壬戌 壬寅`
  - 格局：弃杀就财。
  - 关键：透丙藏戊，“美者存而恶者弃”。
  - 结论：不守死法，顺全局取清。
- **一尚书命**：`丙辰 丙申 丙午 壬辰`
  - 格局：劫刃太重，弃财就杀。
  - 结论：财路已不可行，转杀路方可成。

### 例外和易错点
- 财露并非必凶，关键看有没有官护。
- 比劫并非必坏财格，若身弱任财，略带比劫反有情。
- 伤官不是财格常用之神，只在“财轻、劫重、须化劫生财”时权用。
- 财格不能只盯“富”，必须同时看能否生官、能否取贵。
- 月令暗官、暗杀不可忽视，很多“单透财而贵”都靠这个抬格。

### 章际关联
- 与第34章直接相连：本章讲财格结构，下一章讲财格行运分路。
- 与第31章形成互证：官格喜财生，财格以生官为最高路线，财官是双向相成。
- 与《十神关系速查》的“财官相生”“食伤生财”“贪财坏印”“劫财夺财”四条高度一致。
- 与《断命流程指南》“身能任财”原则完全一致，说明格局法并不脱离强弱法。

## 第34章 论财取运

### 原文金句
- “財格取運，即以財格所就之局，分而配之。”
- “財用煞印，印旺最宜，逢財必忌。”

### 核心规则
1. **财格取运看最终落点** → 是生官、食生、佩印、伤官、带杀、杀印 → 各有不同运法。
2. **财旺生官** → 喜身旺、印绶 → 不利七杀、伤官。
3. **生官后再透印** → 若财已生官而后透印 → 伤官运不一定大害，因为印可护官。
4. **生官带食破局** → 原局若因食伤伤官 → 运喜印绶制食，逢杀反吉，可压食护官。
5. **财用食生，财食重身轻** → 宜助身 → 否则泄耗太过，不能任财。
6. **财用食生，财食轻身重** → 仍行财食 → 以成其食生财之路。
7. **财用食生不忌杀运** → 杀可制比劫护财 → 反而官印有时会晦其财食之功。
8. **财格佩印** → 喜官乡 → 官能通财印之气；身弱者尤喜印旺。
9. **财用食印** → 财轻补财食，身轻补比印 → 官运多碍，杀运不忌。
10. **财带伤官** → 财运最亨 → 杀运不利，官印运亦难见其美。
11. **财带七杀** → 不论合杀存财、制杀生财 → 皆喜食伤、身旺之方。
12. **财用杀印** → 以印旺为第一喜神 → 再逢财运，多半破印生杀，立为大忌。
13. **财用杀印不忌伤食** → 若伤食不去党杀，反可泄身生财、辅助全局。

### 格局成败
- 成格运路
  - 财生官局，行身旺印旺之运最稳。
  - 财食局，按身轻身重分扶抑。
  - 财佩印局，行官印相承之乡为佳。
  - 财带杀局，行能制杀、能任财之运为佳。
- 破格运路
  - 财格最怕财运去党杀、坏印、损官。
  - 财格生官局若再逢伤官，多主官路受阻。
  - 财用杀印局再逢财，为典型破格运。
- 救应关键
  - 财食重则先扶身。
  - 财印局若印弱，以印为先，不可盲目追财。

### 案例拆解
- **本章无新增四柱命例**，可按前章局型推运：
  - **葛参政命型**：财旺生官，行印绶、身旺之地最稳；若遇七杀伤官，易损官路。
  - **杨待郎命型**：财用食生，若财食过重则先扶身；若身重则继续走财食。
  - **曾参政命型**：财格佩印，行官运、印运可抬层次。
  - **赵侍郎命型**：财用杀印，印旺最好，财运最忌。

### 例外和易错点
- “杀运不忌”只适用于特定财食、财杀结构，不可泛化到所有财格。
- 官运与印运并不一定都好，财用食印时官运反易来碍事。
- 财格行运，不能只看“见财发财”，因为很多财格败运恰恰就是再逢财。
- 财格最容易错在不分局型，见财格就统一断“喜财官”。

### 章际关联
- 紧承第33章，属于财格的运法分论。
- 与第32章的官格取运互相映照：官格怕伤，财格怕杀、怕坏印，都是“顺主线、避拆线”。
- 与《断命流程指南》的“用神喜忌要结合格局来源”一致。

## 第35章 论印绶

### 原文金句
- “印綬喜其生身，正偏同為美格，故財與印不分偏正，同為一格而論之。”
- “偏官本非美物，藉其生印，不得已而用之。”

### 核心规则
1. **印绶以生身为本** → 正印偏印在印格中皆可用 → 重点不在偏正，而在能否生身成局。
2. **印透官为正路** → 印格见官，官不只是生印，也可直接取用 → 属最正的贵格之一。
3. **身旺印强不怕太过于一时** → 只要官星清纯 → 官可裁印、印可护身，仍可成贵。
4. **印格带伤食而贵，须伤食被制服** → 伤食若被印、比、官制住，不伤官根 → 仍可贵。
5. **印格用伤食** → 适用于身强印旺，恐其太过 → 以伤食泄秀。
6. **印浅身轻不可层层用伤食** → 否则生身之源被泄尽 → 寒贫之局。
7. **印格用七杀是权法** → 七杀本凶，因其能生印，不得已借之 → 必须是身印有一轻一重，需其调平。
8. **身轻印重可借杀生印兼制身** → 杀使印不滞、身不浮。
9. **身重印轻可借杀补印之不足** → 杀生印，再由印生身，构成回路。
10. **身印并重而再用杀** → 结构太偏 → 多主孤贫，不为佳格。
11. **用杀兼带伤食** → 杀有制、身有泄 → 不论身旺印重，皆可成贵。
12. **印多而用财** → 印重身强，透财抑印，属于权用 → 财须根深，方不致坏印。
13. **印轻财重无劫救** → 典型贪财破印 → 贫贱之局。
14. **印重财轻而露食伤** → 财得食生，轻而不轻 → 可以就富，但未必贵。
15. **印格带食而贵，须合财存食或合官存食** → 即不是简单“食伤见印必凶”，而是看谁被保留。
16. **印格官杀竞透仍可贵** → 条件是能合杀留官，或对官杀有制。
17. **极端时可化印为劫，弃印就财官** → 说明印格也能变局，不是死守印。
18. **印透七杀而劫财存煞印** → 属难看之局 → 可贵，但必须细审气势。

### 格局成败
- 成格条件
  - 印为月令主神，有官、杀、食、财等辅助其流通。
  - 身与印的轻重关系明确，有可调之机。
  - 官清、杀有制、财有根、食有度。
  - 印不被重财直破，或破而有比劫救。
- 破格条件
  - 印浅身轻却层层泄于伤食。
  - 印轻财重无救，应“贪财破印”。
  - 身印并重再用杀，偏枯太甚。
  - 官杀并透而无合无制，局面混杂。
- 救应方法
  - **印重身强**：用财抑印，或用伤食泄秀。
  - **身轻印重**：借杀生印兼制身。
  - **身重印轻**：借杀补印之力。
  - **印遭财破**：求比劫救印，或求财轻有根不致尽坏。

### 案例拆解
- **张参政命**：`丙寅 戊戌 辛酉 戊子`
  - 格局：印格透官。
  - 关键：身旺印强，而官星清纯。
  - 结论：印格最正之贵局。
- **朱尚书命**：`丙戌 戊戌 辛未 壬辰`
  - 格局：印格带伤食而贵。
  - 关键：壬被戊制，不伤官。
  - 结论：有伤而不为患，仍可贵。
- **临淮侯命**：`乙亥 己卯 丁酉 壬寅`
  - 格局：印格兼带官伤。
  - 关键：己为乙制，己不碍官。
  - 结论：制得其所，故贵。
- **李状元命**：`戊戌 乙卯 丙午 乙亥`
  - 格局：印格用伤食泄秀。
  - 关键：身强印旺，需泄其秀气。
  - 结论：食伤在这里不是坏神，而是成名之具。
- **茅状元命**：`己巳 癸酉 癸未 庚申`
  - 格局：身轻印重，用杀。
  - 结论：借杀调平身印，故贵。
- **马参政命**：`壬寅 戊申 壬辰 壬寅`
  - 格局：身重印轻，用杀。
  - 结论：借杀补印之不足，成可用之局。
- **汪侍郎命**：`辛酉 丙申 壬申 辛亥`
  - 格局：印多而用财。
  - 关键：财透有根，可以抑印而不致破尽。
  - 结论：属“权而用之”的高阶印局。
- **牛监薄命**：`庚寅 乙酉 癸亥 丙辰`
  - 格局：印重财轻兼露食。
  - 关键：乙合庚而不生癸，所以为贵。
  - 结论：不是简单财食同来，而是“有保有弃”。
- **己未 甲戌 辛未 癸巳**
  - 格局：合财存食之贵。
  - 结论：印局带食，不怕食现，怕的是食无去处。
- **辛亥 庚子 甲辰 乙亥**
  - 格局：印格官杀竞透，合杀留官。
  - 结论：混中取清，仍成贵局。
- **壬子 癸卯 丙子 己亥**
  - 格局：官杀有制之印格。
  - 结论：有制则混不为害。
- **赵知府命**：`丙午 庚寅 丙午 癸巳`
  - 格局：化印为劫，弃印就财官。
  - 结论：印格也可变局，不可固执守印。
- **庚戌 戊子 甲戌 乙亥**
  - 格局：印透七杀，劫财存煞印。
  - 结论：属难局，需细审，不可草断。

### 例外和易错点
- 印格最常见误区是“印多即贵”。实际上印若不流通，常主闭塞、清高、难发。
- 用杀于印格是“不得已而用之”，不是看见七杀就当贵。
- 印多用财不等于贪财坏印，前提是印重身强、财有根、抑而不破。
- 食伤出现在印格，不必先入为主判凶，要看它是在泄秀，还是在坏印。
- 印格官杀竞透，不是立刻判“混杂”；先看能否合、能否制。

### 章际关联
- 与第36章是成局与取运的正反两面。
- 与第31章互证：官格喜印护，印格喜官成，官印本是一组高阶搭配。
- 与《十神关系速查》的“杀印相生”“贪财坏印”“枭神夺食”三条关系最密。
- 与《滴天髓》“印多反不灵”的经验可互参，本章比《滴天髓》更强调“印要流通”。

## 第36章 论绶印取运

### 原文金句
- “其印綬用官者，官露印重，財運反吉，傷食之方，亦為最利。”
- “印用七煞，運喜傷食，身旺之方，亦為美地，一見財鄉，其凶立至。”

### 核心规则
1. **印格取运仍先分用法** → 用官、用伤食、用杀、兼带伤食、遇财、官杀竞透，各论各的。
2. **印用官** → 官露印重时，财运反吉 → 财可疏印并通官。
3. **印用官亦利伤食** → 因伤食可泄印秀身 → 只要不伤官根，反是美运。
4. **用官而带伤食** → 喜官旺印旺 → 伤食运反成祸，因会再伤官。
5. **此类印局逢杀不忌** → 杀可制伤、通关，有时比官更有力。
6. **印绶用伤食** → 财运反吉 → 财能承食伤之泄，使秀气有落点。
7. **印用伤食忌官运** → 官来克伤食，反伤其用，故易见灾。
8. **印用伤食逢杀反福** → 杀能激活食神制杀或伤官驾杀之路。
9. **印用七杀** → 喜伤食、喜身旺 → 最忌财乡，财会坏印并党杀。
10. **用杀兼带伤食** → 喜身旺印绶，伤食运亦美 → 官财两运多不吉。
11. **印绶遇财** → 第一喜劫地 → 劫可救印抗财。
12. **官印运对印遭财之局亦可行** → 因能扶印助身。
13. **印格官杀竞透** → 喜食神伤官取清 → 印旺身旺之地亦利。
14. **官杀竞透再逢财运最凶** → 财生官杀，混局加重。
15. **印用食伤而印轻** → 仍不利见财 → 因财会连根拔印。

### 格局成败
- 成格运路
  - 印用官，宜走财、伤食通关之运。
  - 印用伤食，宜走财、伤食、杀运，不宜官。
  - 印用杀，宜走伤食、身旺、印旺运。
  - 印遭财者，宜比劫救印。
- 破格运路
  - 财运对多数印格是第一风险，尤其印轻、用杀、官杀竞透时最凶。
  - 官运对“印用伤食”之局多属破格。
  - 官杀再透会加重混杂，不利清局。
- 救应关键
  - 印轻时，先保印根。
  - 印重时，财、食可疏，但必须不至失衡。

### 案例拆解
- **本章无新增四柱命例**，可据前章典型局型推运：
  - **张参政命型**：印用官，行财运未必坏，反能疏印通官。
  - **李状元命型**：印用伤食，行财运更能承泄成名；官运反易压住秀气。
  - **茅状元、马参政命型**：印用杀，最忌财运坏印党杀。
  - **汪侍郎命型**：印重用财，财运可美，但若财过重则转成破印。

### 例外和易错点
- “印格喜印运”是最常见的误判之一。很多印格忌再增印，反喜财食来疏。
- “印格见财必凶”也不绝对，印重用财者，财反是成格之神。
- 官、杀在印格中的吉凶完全取决于它们扮演“成印”还是“混局”的角色。
- 印格取运必须同时看原局“身与印谁重谁轻”，否则容易把疏印误判成坏印。

### 章际关联
- 与第35章前后呼应，是印格运法总纲。
- 与第32章的官格取运互参：官格怕财坏印，印格有时反喜财疏印，取运必须按结构而非按字面喜忌。
- 与《断命流程指南》“同一十神，因格局不同而喜忌相反”的原则一致。

## 第37章 论食神

### 原文金句
- “食神本屬洩氣，以其能生正財，所以喜之。故食神生財，美格也。”
- “更有印來奪食，透財以解，亦有富貴，須就其全局之勢而斷之。”

### 核心规则
1. **食神本质是泄气** → 食神本身不是终点 → 它之所以为美，在于能生财。
2. **食神生财为第一美格** → 财要有根，不必偏正并透 → 身强、食旺、财透，则多主大贵。
3. **食神不宜偏正财叠出** → 财太杂则不专 → 富贵虽有，终不巨大。
4. **藏食露伤** → 主人性刚 → 因食柔而伤刚，伤露则气性偏峻。
5. **夏木用财** → 因火炎土燥，食神生财往往转为武贵 → 不一定走文路。
6. **食神不用财，也可走杀印路** → 就杀印者，多主威权显赫。
7. **无印单露偏官亦可贵** → 条件是无财 → 财若见则会党杀破局。
8. **金水食神而用杀** → 尤为贵秀 → 金水气清，得杀则有权而不浊。
9. **食神忌印，但有气候例外** → 夏火炎燥，木焦需印滋养 → 此时透印不碍。
10. **食神忌官，但金水不忌** → 即“金水伤官可见官”同理，因气候与清气特殊。
11. **单用食神** → 有财运则富，无财运则贫 → 说明食神必须有去处。
12. **印来夺食，可透财解印** → 若全局许可，仍能富贵。
13. **食神而官杀竞出，也可成局** → 但层次不高，贵而不甚。
14. **食神合杀存财** → 最为贵格 → 兼得权与富。
15. **食神透杀本忌财** → 因财会党杀 → 但若财先杀后，中间有食隔，则财不能直接党杀，仍可贵。
16. **食神格判断一定看全局之势** → 不可背口诀一刀切，尤其印、官、财、杀同现时更如此。

### 格局成败
- 成格条件
  - 身强能泄，食神有力。
  - 最佳是食神生财，财有根而不杂。
  - 次佳是食神就杀印，成权贵之路。
  - 食神遇印、官、杀时，要么有调候理由，要么有隔断、分工、化解。
- 破格条件
  - 食神无财可归，无杀可就，只成空泄。
  - 食神透杀又见财，财去党杀攻身。
  - 印来夺食而无财解。
  - 官杀竞出过杂，层次大减。
  - 财多杂出，损其专一，不成大格。
- 救应方法
  - **印来夺食**：透财以解。
  - **食神透杀**：最怕财党杀，若有食隔财杀，可减其害。
  - **食神太重**：可就杀印，转泄为权。
  - **夏木食神见印**：以调候为先，不可机械以“食神忌印”断坏。

### 案例拆解
- **梁丞相命**：`丁未 癸卯 癸亥 癸丑`
  - 格局：食神生财。
  - 关键：身强、食旺、财透，且财有根。
  - 结论：属食神生财的大贵局。
- **谢阁老命**：`己未 壬申 戊子 庚申`
  - 格局：食神生财。
  - 结论：与梁丞相命同为“身强食旺财透”的代表。
- **沈路分命**：`丁亥 癸卯 癸卯 甲寅`
  - 格局：藏食露伤。
  - 关键：伤官外露，性情偏刚。
  - 结论：点明食伤并见时对性情的影响。
- **龚知县命**：`甲午 丁卯 癸丑 丙辰`
  - 格局：偏正叠出之食神财局。
  - 关键：财星多见而不专。
  - 结论：富贵不巨。
- **黄都督命**：`己未 己巳 甲寅 丙寅`
  - 格局：夏木用财。
  - 关键：火炎土燥，食神生财多转武职。
  - 结论：贵多就武。
- **常国公命**：`辛卯 辛卯 癸酉 己未`
  - 格局：食神不用财而就杀印。
  - 结论：威权显赫。
- **胡会元命**：`戊戌 壬戌 丙子 戊戌`
  - 格局：食神单露偏官而无财。
  - 关键：无财党杀。
  - 结论：亦可成贵。
- **舒尚书命**：`丁亥 壬子 辛巳 丁酉`
  - 格局：金水食神而用杀。
  - 结论：贵而且秀。
- **钱参政命**：`丙午 癸巳 甲子 丙寅`
  - 格局：食神忌印之例外。
  - 关键：夏火太炎，木焦，透印不碍，先解燥。
  - 结论：调候优先于常法。
- **刘提台命**：`癸酉 辛酉 己卯 乙亥`
  - 格局：食神透杀，财先杀后，食居其中。
  - 关键：食神隔开财杀，使财不能直接党杀。
  - 结论：本忌见财，因有间隔而仍可贵。

### 例外和易错点
- 食神并不是“天然吉神”，没有去处就只是泄气。
- “食神忌印”“食神忌官”都不是绝对，夏木、金水两类是最重要例外。
- 食神透杀见财，本来最凶，但要检查财杀之间有没有食神隔断。
- 食神格若财太杂、偏正并见太多，常是“有富贵而不巨”。
- 断食神格，必须同时看气候、强弱、去处，不可只背一句“食神生财”。

- 与第35章印格相对：印可夺食，也可在特殊气候下救食，显示十神喜忌必须动态判断。
- 与《十神关系速查》的“食神生财”“食神制杀”“枭神夺食”是同一组核心知识。

## 第38章 论食神取运

### 原文金句
- “食神生財，財食輕，則行財食，財食重則喜幫身。”
- “食神太旺而帶印，運最利財，食傷亦吉，印則最忌，官煞皆不吉也。”

### 核心规则
1. **食神取运先看成局路线** → 是生财、用杀印、带杀、太旺带印、透财解印 → 运法各不相同。
2. **食神生财，财食轻** → 宜行财食运 → 继续完成泄秀生财之路。
3. **食神生财，财食重** → 宜帮身 → 防止泄过、财重压身。
4. **食神生财一般忌官杀** → 官杀克身、扰乱食财主线 → 多不美。
5. **食用杀印** → 喜印旺 → 最忌财乡，因为财会坏印党杀。
6. **食用杀印而身旺** → 食伤运亦为福运 → 泄身并助制杀。
7. **行官行杀也可吉** → 仅限原局已成杀印路，且印足可化 → 不可泛化到普通食神局。
8. **食伤带杀** → 喜行印绶 → 身旺及食伤运也多美。
9. **食伤带杀最忌财** → 财去生杀，破坏制杀平衡。
10. **食太重而杀轻** → 印运最利 → 财运反而可吉，因为可承食泄，但须原局不致党杀。
11. **食神太旺而带印** → 最喜财 → 财可制印、承食，使泄秀有出口。
12. **食神太旺带印，再见印最忌** → 印愈多则愈夺食、闭秀。
13. **此类局官杀亦不吉** → 因官杀会打断食财、食印之间既有平衡。
14. **食神带印而透财解印** → 喜财旺与食伤运 → 印与官杀皆忌。

### 格局成败
- 成格运路
  - 食神生财局，按财食轻重决定扶身或助财食。
  - 食神就杀印局，以印为第一关键。
  - 食伤带杀局，以印为制衡核心。
  - 食神太旺带印局，以财疏印为第一美运。
- 破格运路
  - 普通食神局逢官杀，多数不利。
  - 食用杀印局最怕财运破印。
  - 食神带印局最怕再行印运，形成“夺食闭秀”。
- 救应关键
  - 财食重者先扶身。
  - 杀轻者防制过。
  - 印重者求财疏印。

### 案例拆解
- **本章无新增四柱命例**，可据前章局型推运：
  - **梁丞相、谢阁老命型**：食神生财，若财食太重则先扶身；若原局财轻则继续财食运。
  - **常国公、胡会元命型**：食用杀印，印运最关键，财运多忌。
  - **舒尚书命型**：金水食神用杀，官杀运不一定坏，关键在印与食能否维持平衡。
  - **钱参政命型**：食神带印有调候意义，仍要防印过重夺食。

### 例外和易错点
- “官杀之方俱为不美”主要针对食神生财主线，不适用于已经转入杀印路线的食神局。
- “食太重而杀轻，逢财反吉”是有条件的，不可忘记前提是财不致党杀太过。
- 食神局见印，不要只问“有无枭夺食”，还要问印是不是在调候、是不是已被财解。
- 取运时最容易错在不分“原局主线”，把所有食神局都当成同一种命。

### 本批次总提炼

#### 一、四大核心十神的成格条件
- **正官**：以“清、护、尊”为三字诀。官星要清，不可混杀；要护，不可伤官直犯；要尊，不可刑冲破害。财印并透而不相碍，是官格最美状态。
- **财**：以“根、专、流”为三字诀。财要有根，不可浮；要专，不可过露杂透；要流，最好流向官，次则得食生、印扶、杀印成局。
- **印绶**：以“生、通、平”为三字诀。印能生身，但不能闭塞；要靠官、杀、食、财把印用活；身印轻重失衡时，再考虑财抑印、食泄秀、杀生印。
- **食神**：以“泄、归、变”为三字诀。食神必须有去处，归财最富，归杀印最贵；若被印夺、被官阻、被财党杀，则需看能否隔断、解印、转路。

#### 二、四大核心十神的破格条件
- **正官破格**：伤官犯官、官杀混杂、财印交战、官被刑冲破害、重官压身。
- **财格破格**：财多身弱、财露无护、财旺再透伤、财杀并透无印化、印被财坏。
- **印格破格**：印轻财重无救、印浅身轻而层层泄、身印并重再用杀、官杀混透无制。
- **食神破格**：食无归宿、食透杀又见财、印来夺食无财解、官杀混杂太重、财太杂不专。

#### 三、断命时的实战顺序
1. 先定月令主神，是官、财、印、食哪一路。
2. 再看这一路有没有被破。
3. 若被破，看有没有救应。
4. 救应成立，再分格局高低。
5. 最后才看运，且运必须跟着原局成局路线走。

#### 四、与他书交叉验证
- 与《滴天髓》互证处：官要有理会，杀要制化，印要流通，食要有去处。
- 与《三命通会》互证处：官印相生、财官双美、杀印相生、食神制杀等经典组合，本批次均有根源。
- 与《穷通宝鉴》互证处：夏木、金水等气候例外，证明“调候优先于死口诀”。
- 与 `knowledge/断命流程指南.md` 互证处：格局判断先于流年运断；与 `knowledge/十神关系速查.md` 互证处：本批次可作为十神组合的原典展开版。

## 第39章 论偏官

### 原文金句
> 七煞之格局亦不一：煞用食制者，上也，煞旺食強而身健，極為貴格。

> 有煞而雜官者，或去官，或去煞，取清則貴。

### 核心规则
- **杀用食制为上** → 七杀当令或有力、食神亦强、日主能任 → 七杀格最高层次在“食神制杀”。
- **杀用食制忌露财** → 食神在前而财又透 → 财转食生杀，制神反成助杀，容易破格。
- **杀用食制忌透印** → 印透去食护杀 → 食神失去制杀功能，七杀失控。
- **财先食后可成贵** → 先有财生杀，后有食制杀 → 杀得其源而又受控，反成上格。
- **印先食后可成贵** → 食神太旺，印来节制食神之太过 → 不是护杀，而是防食太重，仍可成格。
- **杀重身轻不宜硬用食** → 七杀太重、日主太弱 → 用食神虽可制杀，但身受不住，宜转印化杀。
- **杀用印贵在“有情”** → 印与杀同根同气、相承有序 → 印虽护杀，却能化杀生身，此时反贵。
- **杀用财只在特定结构可用** → 财不是单纯党杀，而是“去印存食”“清格” → 财方可转凶为吉。
- **身重杀轻可借财清格** → 日主偏重、杀轻且化印 → 以财拆开浊局，使主用归清，仍可取贵。
- **杂气七杀不必执着透财** → 七杀藏于杂气，若整体用神已清 → 即使干头不透财，也可取贵。
- **官杀混杂必须取清** → 月令主神为杀则留杀去官；月令主神为官则留官去杀 → 不能反取。
- **无食制杀可借印或阳刃** → 七杀无食神时，不是必败 → 若有印可化，有刃可敌，仍可成局。
- **制杀不可太过不可执一** → 书言“制杀不可太过”有理，但行运财印亦可发福 → 不能死守一句。

### 格局成败
- **成格条件**
  - 七杀有根有力，不是虚杀。
  - 食神、印绶、财星、阳刃之中，至少有一路能真实驾驭七杀。
  - 日主不能太弱，至少要能承受杀气。
  - 官杀混杂时必须取清，保住月令主神。
- **破格条件**
  - 食神制杀而财透转食生杀。
  - 食神制杀而印透夺食护杀。
  - 七杀太重而身弱，却仍勉强用食。
  - 官杀并透却不取清，反成混乱。
  - 财杀并见却只是单纯党杀，无制无化。
- **救应方法**
  - 杀重身轻 → 转印化杀。
  - 食重杀轻 → 以财生杀，或去印存食，使杀可用。
  - 官杀混杂 → 合去其一，先清后论贵贱。
  - 七杀无食 → 借印当杀、借刃当杀。

### 案例拆解
- `乙亥 乙酉 乙卯 丁丑`
  - 乙木日主，酉月七杀当令，时透丁火食神，亥卯扶身。
  - 结构：杀旺、食强、身健。
  - 结论：典型“食神制杀”上格，所以原文直断“极等之贵”。
- `壬辰 甲辰 丙戌 戊戌`
  - 丙火日主，辰中暗藏七杀，壬水透出，戊土食神重，甲木印透。
  - 结构：先有杀，再有重食制杀，甲印不是护杀，而是节食太过。
  - 结论：印先食后而成贵，是“食神太重须印调停”的范例。
- `丙寅 戊戌 壬戌 辛丑`
  - 壬水日主，戊土七杀、辛金印星同通月令。
  - 结构：杀印同根，有情相生。
  - 结论：七杀用印，不以“印护杀”为忌，反成贵格。
- `戊戌 甲子 丁未 庚戌`
  - 丁火日主，戊土食神受制不足，庚金财星透出。
  - 结构：财来去印存食，又生不足之杀，再由食制之。
  - 结论：财不是坏事，而是“清食、生杀、再制杀”，层次更高。
- `甲申 乙亥 丙戌 庚寅`
  - 丙火日主，身重杀轻，杀又化印，庚财透出清格。
  - 结构：借财使混局归清。
  - 结论：不是标准杀格，却仍可因“清格”取贵。
- `癸卯 丁巳 庚寅 庚辰`
  - 庚金日主，月令偏官为主，同时官杀混见。
  - 结构：去官留杀。
  - 结论：因为月令用杀，所以要保杀，不可误以“官贵于杀”而留官。
- `丙子 甲午 辛亥 辛卯`
  - 辛金日主，子冲午，去杀留官。
  - 结构：月令若以官为重，则应存官去杀。
  - 结论：与上例相反，说明“取清”必须服从月令主神。
- `戊辰 甲寅 戊寅 戊午`
  - 戊土日主，杀无食制，以印当杀。
  - 结论：七杀格并非只能靠食神，印绶亦可成局。

### 例外和易错点
- “印能护杀”不是一概凶。只要印杀有情，护杀其实就是化杀。
- “财来见杀必凶”也不成立。若财的作用是清格、去印、存食，反而是贵征。
- 官杀混杂时，不能机械地认为“官比杀贵，所以永远留官”。月令主神是谁，就保谁。
- 七杀格的核心不是“看见杀就压死”，而是“让杀有源、有制、有归宿”。
- “制杀不可太过”只是常理，不是铁律。若运来财印能调平，仍可发福。

## 第40章 论偏官取运

### 原文金句
> 煞用食制，煞重食輕則助食，煞輕食重則助煞，煞食均而日主根輕則助身。

> 煞無食制而用刃當煞，煞輕刃重則喜助煞，刃輕煞重，則宜制伏。

### 核心规则
- **杀格取运先看成局方式** → 食制、印化、财清、刃当，各有不同，不可一概论。
- **杀用食制，杀重补食** → 七杀过旺、食神不足 → 取运宜扶食神。
- **杀用食制，食重补杀** → 食神过重、七杀不足 → 取运宜扶杀，使制化有对象。
- **杀食均衡而身轻补身** → 杀与食已经平衡，但日主根气不足 → 先保身。
- **杀用食制忌官混** → 行运再见正官 → 官杀混杂，清局被坏。
- **杀用食制畏印夺食** → 行运印旺 → 食神被制，杀失控。
- **杀用印不利财乡** → 财来坏印 → 印化杀的链条被断。
- **杀用印见伤官反美** → 伤官泄身、制官、通关 → 常有助局面流通。
- **印旺身旺皆福地** → 杀用印之局，印运、比印帮身之运，多为顺运。
- **杀用财而去印存食者喜财** → 财能完成“清格”任务 → 财运、伤食运、透杀运皆顺。
- **杀用财而去印存食者忌劫** → 劫财夺财 → 财之清格作用失效。
- **杀用财而助杀不足者要分财足不足** → 财足则喜食印帮身，财不足则喜再增财并露杀。
- **官杀并见取运守清** → 去官留杀或去杀留官之后，行运不可伤害已取清的那一路。
- **用刃当杀时先衡刃杀轻重** → 杀轻刃重宜助杀，刃轻杀重宜制杀或扶刃。
- **刃当杀不怕印运** → 因无食神可夺，印不会造成“夺食护杀”的老问题。
- **七杀既纯忌杂官** → 行运再来官星，常把清局搅浊。

### 格局成败
- **成格关键**
  - 行运要顺原局成局路线补偏救弊。
  - 取清后的格局最怕再混。
  - 杀格最重“控制链条”不断裂。
- **败局征象**
  - 杀用食制再行印运。
  - 杀用印再入财地。
  - 杀用财却遇劫财夺财。
  - 官杀原已取清，运中再把另一方扶起。
- **运中救应**
  - 杀轻食重 → 杀运可起。
  - 食轻杀重 → 食运可起。
  - 身轻 → 比印运可起。
  - 财不足 → 财运可起。

### 案例拆解
- 本章无新命例，重在承接上一章的几种杀格。
- 可回看 `乙亥 乙酉 乙卯 丁丑`
  - 若原局为食神制杀，则行运重点就是“补食、补杀、补身”三分法。
- 可回看 `丙寅 戊戌 壬戌 辛丑`
  - 若原局为杀印相生，则财运最危险，印旺身旺之运反佳。
- 可回看 `戊戌 甲子 丁未 庚戌`
  - 原局财的责任是清格，则劫财运最忌，财运、食伤运反吉。

### 例外和易错点
- 七杀格取运不是“逢印就吉、逢财就凶”。要先问原局到底是用印还是用财。
- 杀格不怕帮身，而怕“控制链断裂”。这比单纯看身旺身弱更关键。
- “伤官为美”只限杀用印之局，不能反推所有杀格都喜伤官。
- 刃当杀时，不能套用食神制杀那套“怕印”逻辑，因为根本不是同一路成局。

### 章际关联
- 与第39章《论偏官》构成一组：前章讲成格，后章讲行运补偏。
- 与第44章《论阳刃取运》互通：都强调“先辨轻重，再决定补哪一边”。
- 与《断命流程指南》的“格局 > 调候 > 强弱”完全一致，杀格行运尤其如此。

## 第41章 论伤官

### 原文金句
> 傷官雖非吉神，實為秀氣，故文人學士，多於傷官格內得之。

> 有傷官佩印者，印能制傷，所以為貴，反要傷官旺，身稍弱，始為秀氣。

### 核心规则
- **伤官本凶而可化秀** → 伤官天然抗官，不是吉神，但它也是才气、文气、机锋之源。
- **夏木见水最秀** → 夏季木燥，水来润木，伤官得调候，秀气更高。
- **冬金见火最秀** → 冬金寒冷，火来温煅，伤官不再寒涩，文章才名更显。
- **伤官用财最利** → 伤官生财，把“逆气”转成“生官之具”，故能转凶为吉。
- **伤官生财须身强有根** → 日主太弱则泄过头，财虽见而不能任。
- **化伤为财高于普通生财** → 伤官本体直接转为财路，比单纯“伤官生财”更有情更秀。
- **财伤有情亦贵** → 财与伤各有根气、配合得宜 → 秀气不下于化伤为财。
- **伤官佩印要伤旺身弱** → 伤官旺、印根深、日主略弱 → 印制伤而护身，最为高格。
- **佩印忌印多伤轻** → 伤官轻而身重，再加印多 → 变成闭塞不秀，反主贫。
- **伤官兼用财印可行，但须两清不碍** → 财印本相克，必须彼此隔开，不直接交战。
- **财太旺可带印，印太重可带财** → 两边都允许存在，但目的在“调停中和”，不是让它们互打。
- **伤官用杀印，须伤多身弱且无财** → 赖七杀生印来帮身制伤；若再来财，则坏印助杀，反败。
- **伤官用官只限特殊格** → 常格不用，金水独宜，且要财印为辅。
- **伤官用官忌官伤并透** → 孤官无辅、官伤对立 → 福力不大。
- **非金水见官亦有例外** → 若已化伤为财，则不作“伤官见官”，而作“财旺生官”。
- **官杀并透亦须取清** → 即使金水伤官，也不能容官杀混杂。

### 格局成败
- **成格条件**
  - 伤官之气要有明确出路：生财、佩印、用杀印、特殊用官。
  - 调候得宜，尤其夏木、冬金最见高格。
  - 日主强弱与伤官轻重必须平衡。
  - 财印同见时，要隔而不战。
- **破格条件**
  - 伤官见官而无化、无辅。
  - 伤官佩印却印过重、伤太轻。
  - 伤官兼财印但财印直接交战。
  - 伤官用杀印却见财来坏印。
  - 官杀并透而不取清。
- **救应方法**
  - 伤官太放 → 以印节之。
  - 伤官太寒太燥 → 先以调候救之。
  - 伤官见官 → 若能化伤为财，可改论财生官。
  - 财印互伤 → 以天干隔位、地支分路求“两清”。

### 案例拆解
- `壬午 己酉 戊午 庚申`
  - 戊土日主，酉申金旺，伤官生财。
  - 结构：身强有根，伤官有力而归财。
  - 结论：标准“伤官生财”贵格。
- `甲子 乙亥 辛未 戊子`
  - 辛金日主，亥子水旺为伤，甲乙木财透，且未亥拱木意重。
  - 结构：化伤为财。
  - 结论：比普通伤官生财更秀，所以原文断为状元。
- `己卯 丁丑 丙寅 庚寅`
  - 丙火日主，伤官与财星同根月令。
  - 结构：财伤有情。
  - 结论：秀气与化伤为财相近。
- `壬申 丙午 甲午 壬申`
  - 甲木日主，午火伤官旺，壬水印深，夏木得润。
  - 结构：伤官旺、印根深、身稍弱。
  - 结论：这是“伤官佩印”教科书命例。
- `丁酉 己酉 戊子 壬子`
  - 戊土日主，财势偏重，带印调停，丁壬隔戊己不相碍。
  - 结构：伤官兼用财印，财多带印。
  - 结论：财印两清而成贵。
- `壬戌 己酉 戊午 丁巳`
  - 戊土日主，印太重而带财，丁壬亦隔而不碍。
  - 结构：伤官兼用财印，印多带财。
  - 结论：与上例相反方向，但同样以中和取贵。
- `己未 丙子 庚子 丙子`
  - 庚金日主，子水伤官太多，丙火七杀透，借杀生印以帮身制伤。
  - 结构：伤官用杀印。
  - 结论：伤重身弱之局，不靠财，不靠比，而靠杀印链条成格。
- `戊申 甲子 庚午 丁丑`
  - 庚金日主，子月金水伤官，藏癸露丁，有官星得禄，戊甲为辅。
  - 结构：金水伤官用官。
  - 结论：官不孤、财印辅之，所以可成丞相格。
- `丙申 己亥 辛未 己亥`
  - 辛金日主，冬金用官，又化伤为财。
  - 结构：官、财、伤协调，且有调候。
  - 结论：属于原文所谓“极秀极贵”。
- `甲子 壬申 己亥 辛未`
  - 己土日主，非金水常格，但已化伤为财。
  - 结构：不作伤官见官，而作财旺生官。
  - 结论：这是“例外成立”的关键示范。

### 例外和易错点
- “伤官见官，为祸百端”是大原则，不是无条件绝对律。金水、化伤为财，都可能例外。
- 伤官佩印不是“见印就贵”，而是要“伤旺、身稍弱、印有根”。若身旺伤轻印多，反闭塞。
- 伤官带财印，不是随便三样同见就好，必须财印两清不打架。
- 伤官用杀印最怕财。很多人一见财就说“伤官生财”，这里恰好相反。
- 伤官最大的判断难点不是“凶不凶”，而是“最后落到财、印、官、杀哪一路”。

### 章际关联
- 与第42章《论伤官取运》直接衔接：原局先分成财、印、官、杀四路，行运再分别处理。
- 与第39章《论偏官》互证：伤官用杀印，本质仍是借杀生印、以印制伤。
- 与第47章《论杂格》末段批评“伤官伤尽”相连：说明伤官不可只凭口诀断生死贵贱。
- 与《十神关系速查》中的“伤官见官”“伤官生财”“伤官佩印”相互印证。

## 第42章 论伤官取运

### 原文金句
> 傷官用財，財旺身輕，則利印比；身強財淺，則喜財運，傷官亦宜。

> 傷官而用煞印，印運最利，傷食亦亨，雜官非吉，逢財即危。

### 核心规则
- **伤官取运先辨归宿** → 原局伤官落在财、印、杀、官哪一路，运法完全不同。
- **伤官用财，财旺身轻喜印比** → 财太旺而身受不住 → 比印运救身。
- **伤官用财，身强财浅喜财运** → 身能任泄而财不足 → 财运、伤官运皆可发。
- **伤官佩印宜官杀运** → 官杀可生印、成链条 → 常属顺运。
- **伤官佩印亦喜印运** → 印可节伤护身，维持原局主线。
- **伤官佩印不怕再逢伤食** → 只要印根能制，伤食不一定成祸。
- **伤官佩印忌财地** → 财坏印，伤官失控。
- **伤官兼财印，财多带印者喜助印** → 因原局财偏重，行运要帮印调平。
- **伤官兼财印，印多带财者喜助财** → 因原局印偏重，行运要帮财泄印。
- **伤官用杀印，印运最利** → 杀印链条以印为核心，先扶印。
- **伤官用杀印，伤食运亦可** → 在印根稳的前提下，伤食有时能发用。
- **伤官用杀印忌杂官** → 官来混杀，局面不清。
- **伤官用杀印逢财即危** → 财坏印，且可能转助杀。
- **伤官带杀，伤重杀轻时印财皆可** → 印稳格，财亦可辅助流通。
- **伤官带杀，杀根重则喜伤食** → 用伤食裁杀；印绶、身旺亦可，但财为凶。
- **伤官用官，运喜财印** → 官要有辅、有源、有护。
- **伤官用官，不利再逢食伤** → 食伤再来冲官，最易败局。
- **官露而财印两旺，比劫伤官运仍非吉** → 因比劫、伤官都可能冲坏成局之官。

### 格局成败
- **行运成功的原则**
  - 补原局不足的一端。
  - 保原局最清、最贵的链条不断。
  - 避免运来改换用神路线。
- **败运信号**
  - 佩印局走财地。
  - 用杀印局走财运、官混之运。
  - 用官局走食伤、比劫重运。
  - 财印兼用局，运中只扶强的一边，不顾调停。
- **运中救应**
  - 财旺身轻 → 印比救。
  - 印多财少 → 财运救。
  - 杀根太重 → 伤食裁之。
  - 官弱无辅 → 财印扶之。

### 案例拆解
- 仍以前章命例分流理解最准确。
- `壬午 己酉 戊午 庚申`
  - 伤官用财。
  - 若后运财多身轻，则喜比印；若身强财浅，则喜再行财伤。
- `壬申 丙午 甲午 壬申`
  - 伤官佩印。
  - 官杀运、印运多顺；若重入财地，则财坏印而失秀。
- `己未 丙子 庚子 丙子`
  - 伤官用杀印。
  - 印运最稳，财运最险，杂官运易乱格。
- `戊申 甲子 庚午 丁丑`
  - 金水伤官用官。
  - 财印运可扶官，重逢食伤则容易回到“伤官见官”。

### 例外和易错点
- 伤官运法不能只按“身旺喜泄、身弱喜扶”来算，必须先服从格局路线。
- “伤食不碍”只适用于佩印局，不等于所有伤官格都不怕食伤再来。
- 用官局的危险在于，原局好不容易把伤官驯服，一走食伤运就可能前功尽弃。
- 财、印都可能是吉神，也都可能是祸根，关键只看它在当前格局中扮演什么角色。

### 章际关联
- 与第41章《论伤官》形成“原局分类 + 行运补偏”的完整系统。
- 与第40章《论偏官取运》方法一致：都是先分成格路线，再按轻重补救。
- 与《断命流程指南》的“大运流年必须沿原局主线推演”相一致。

## 第43章 论阳刃

### 原文金句
> 陽刃者，劫我正財之神，乃正財之七煞也。

> 刃宜伏制，官煞皆宜，財印相隨，尤為貴顯。

### 核心规则
- **阳刃是劫财之极** → 比普通劫财更猛，专夺正财，所以称“刃”。
- **阳刃只属五阳干** → 甲丙戊庚壬有之，阴干无真正阳刃。
- **刃格首重制服** → 官、杀皆可制刃，是成格第一条件。
- **刃格喜财印相随** → 与他格杀忌财印不同，阳刃借官杀制刃，不怕杀伤身，故反喜财生官杀、印护官杀。
- **阳刃用官，透刃不害** → 官能直接制刃，即使刃透，也不至为病。
- **阳刃露杀，透刃无成** → 刃会合杀，七杀贪合忘克，失去制刃之功。
- **官杀根深者贵大** → 制刃之神透而有根，贵重层次高。
- **官杀根浅者贵小** → 虽可成局，但权轻位浅。
- **制刃带伤食不一定坏** → 若有印护官、伤食裁杀、或合官去杂留杀，仍可贵。
- **阳刃格利于留杀** → 官杀并见时，刃格常偏向留杀取清，因为杀的制刃力更猛。
- **丙午刃格尤宜财印** → 午中己土可克水，若再得财印扶助，制刃力量更稳。
- **戊午刃格可化刃为印** → 若丙透、支会火局，可视为印局渐成，去刃存印更清。
- **财杀并透于戊午刃格反忌** → 这不是普通“生杀制刃”，而是“去印存杀”，会把格局拉坏。
- **阳刃用财本非正路** → 但若财根深、又有伤食把刃气转去生财，则可富可贵。
- **刃财相搏则败** → 若无伤食转化，刃与财正面冲突，难以成局。

### 格局成败
- **成格条件**
  - 必有官杀来制刃，且制神最好透干有根。
  - 财印能围绕官杀服务，而不是破坏制刃链条。
  - 官杀混见时要取清。
  - 若改走财路，必须先有伤食导刃生财。
- **破格条件**
  - 露杀又透刃，刃合杀，杀失其功。
  - 官杀无根、制刃无力。
  - 戊午刃格财杀并透，犯去印存杀。
  - 用财而无伤食转化，形成刃财相搏。
- **救应方法**
  - 官杀太重 → 以伤食裁损，或以印护官杀。
  - 官杀并见 → 合去其一，留清者。
  - 刃过旺而财弱 → 先上官杀，不可直接逐财。
  - 刃财相战 → 以伤食通关。

### 案例拆解
- `己酉 丙子 壬寅 丙午`
  - 壬水日主，子月阳刃，丙火财生官，官透有力。
  - 结构：官透根深，旺财生之。
  - 结论：标准阳刃用官大贵格。
- `辛酉 甲午 丙申 壬辰`
  - 丙火日主，午月阳刃，壬杀透而根浅。
  - 结构：透杀制刃，兼得财印相扶。
  - 结论：虽杀根浅，仍可成贵，但层次略次于官根深者。
- `甲午 癸酉 庚寅 戊寅`
  - 庚金日主，官被伤，戊土合癸以护官。
  - 结构：带伤食而有印护。
  - 结论：说明“带伤食不破刃格”，关键看是否护住制刃之神。
- `甲寅 庚午 戊申 甲寅`
  - 戊土日主，阳刃配双杀，且杀根太重。
  - 结构：以食神裁损过重之杀。
  - 结论：制刃之神不是越重越好，太重也要修剪。
- `丙戌 丁酉 庚申 壬午`
  - 庚金日主，官杀竞出，壬合丁官。
  - 结构：去官留杀，取清。
  - 结论：刃格往往宜留杀，不宜两头并用。

### 例外和易错点
- 刃格与杀格对财印的态度相反。很多人容易混淆：杀格常怕财印坏制神，刃格反常常要财印扶制神。
- “透刃必凶”不成立。阳刃用官时，刃透并不怕，怕的是用杀而刃合杀。
- 带伤食不等于破格。只要伤食承担的是护官、裁杀、去杂的任务，仍属成局。
- 阳刃用财不是常法，只能算旁门活路，必须先有伤食转化。

### 章际关联
- 与第44章《论阳刃取运》相连：原局贵在制刃，运法仍以扶制神、避乱局为核心。
- 与第39章《论偏官》互证：留杀取清、借杀制刃，是七杀权威功能的另一种落地。
- 与第45章《论建禄月劫》互证：三者都属比劫旺格，但阳刃比建禄月劫更烈、更险。

## 第44章 论阳刃取运

### 原文金句
> 陽刃用官，則運喜助官，然命中官星根深，則印綬比劫之方，反為美運。

> 陽刃而官煞並出，不論去官去煞，運喜制伏，身旺亦利，財地官鄉反為不吉也。

### 核心规则
- **阳刃用官喜助官** → 官弱则宜官运、财运扶官。
- **官根深者可喜印比** → 因官已稳，印比不再助刃为祸，反能扶身承贵。
- **阳刃用官忌伤食合官** → 伤食来犯、来合，官失制刃之权。
- **阳刃用杀，杀轻喜助杀** → 七杀不足以制刃 → 先补杀。
- **阳刃用杀，杀重喜身旺印绶** → 杀太重时，先保身并护制杀链条。
- **阳刃用杀，伤食不尽为忌** → 在杀过重情形下，伤食可裁损，不一定破格。
- **官杀并出之局喜制伏运** → 原局既已混，运中宜帮助取清、裁杂。
- **官杀并出之局身旺亦利** → 身能承受制刃结构，则不怕再扶身。
- **阳刃格不宜财地官乡** → 尤其原局已混局时，财官再来常引发争战和再混。
- **刃格行运不纯看身旺身弱** → 重点在制刃之神能否持续有效。
- **比劫运不一定坏** → 用官且官根深时，比劫运未必为凶。
- **印运不一定保守** → 用杀而杀重时，印运反可安局。

### 格局成败
- **顺运标准**
  - 官杀得到扶助而不过分。
  - 官杀混局得到制伏、取清。
  - 身能承局，不被过重杀气压垮。
- **败运标准**
  - 官被伤食合去。
  - 财再生官杀，引发混局。
  - 官乡再来，杂神俱动，斗争加剧。
- **运中救应**
  - 官弱 → 助官。
  - 杀弱 → 助杀。
  - 杀重 → 印、身、伤食裁损。
  - 混局 → 取清、制伏。

### 案例拆解
- `己酉 丙子 壬寅 丙午`
  - 若后运官弱，则宜再助官；若官根已稳，印比运亦可承权。
- `辛酉 甲午 丙申 壬辰`
  - 原局用杀，若壬杀太轻，则喜再助杀；若后运杀太盛，则转喜印运护身。
- `丙戌 丁酉 庚申 壬午`
  - 官杀并出而取清之局，最怕运中又把被去的一路扶起。

### 例外和易错点
- 很多人把比劫运视为刃格绝忌，其实若官根深稳，比劫反可帮身任贵。
- “财地官乡反凶”是针对刃格结构说的，不是说所有财运官运都坏，而是说再把争战之源点燃就坏。
- 刃格行运必须紧盯“官是否还能制刃、杀是否还能制刃”，这是第一问题。

### 章际关联
- 与第43章《论阳刃》前后呼应：原局讲制刃条件，行运讲怎样保住制刃链条。
- 与第40章《论偏官取运》方法一致：都按“轻重平衡”而非“吉神凶神标签”取运。

## 第45章 论建禄月劫

### 原文金句
> 建祿者，月建逢祿堂也，祿即是劫。

> 故建祿與月劫，可同一格，不必加分，皆以透干支，別取財官煞食為用。

### 核心规则
- **建禄本质就是月令比劫当权** → “禄”不是天然贵气，而是劫气当令。
- **建禄与月劫同格而论** → 不必人为拆成两套体系。
- **不能见禄就拿禄为用** → 禄只是底盘，不是成格用神。
- **建禄用官，以官透为奇** → 官星明透，才有制比劫、立秩序之力。
- **建禄用官要财印相随** → 财可生官，印可护官，不可孤官无辅。
- **官而兼财印为三奇** → 财官印齐备，且由官隔开财印不相伤 → 格局更大。
- **建禄用财必须带食伤** → 比劫与财正冲，必须靠食伤转化，才能“转劫生财”。
- **化劫为财是上法** → 劫气直接化成财气，层次很高。
- **化劫为生亦秀** → 劫气不直接化财，但转成生财之源，也属漂亮格局。
- **建禄用杀必须制伏** → 七杀虽可立威，但若无制则比劫与杀互战。
- **用杀再逢财本不美** → 财党杀，容易加压日主。
- **去杀存财可成贵** → 若能把党杀之财转为主用财，则反成佳局。
- **无财官杀可用伤食泄秀** → 比劫太旺时，泄其太过，仍能成文秀之格。
- **春木秋金用伤食尤贵** → 木逢火则明，金生水则灵，是泄秀上乘。
- **官杀竞出必须取清** → 合杀留官或制杀留官，仍是老原则。
- **两官竞出须制伏** → “争正官不可无伤”，官多不如官清。
- **孤官无辅格局小** → 即便不破，也难大贵。
- **用官透伤食不一定破格** → 若伤能被合去、去伤存官，反可留清。
- **用财不透伤食难发端** → 没有通关之神，财与比劫直接相战，难以启动富局。
- **官杀重而无制，待运制伏亦可发** → 但前提是不能重到危身。

### 格局成败
- **成格条件**
  - 月令比劫旺是事实，但必须另找财官杀食立局。
  - 用官要官透且有辅。
  - 用财要食伤通关。
  - 用杀要有制伏。
  - 无财官时，用伤食泄秀也可。
- **破格条件**
  - 把禄本身误当用神。
  - 孤官无辅。
  - 用财而无食伤。
  - 官杀并见不清。
  - 官杀太重而无制。
- **救应方法**
  - 官弱 → 财印辅官。
  - 财被劫 → 伤食化劫生财。
  - 杀太重 → 合杀、制杀。
  - 比劫太旺无出路 → 用伤食泄秀。

### 案例拆解
- `庚戌 戊子 癸酉 癸亥`
  - 癸水日主，子月建禄，用官而印护。
  - 结构：官有印护，不成孤官。
  - 结论：建禄用官可贵的正例。
- `丁酉 丙午 丁巳 壬寅`
  - 丁火日主，午月建禄，用官而财助。
  - 结构：财生官，官制劫。
  - 结论：建禄不怕旺，怕无治理之神。
- `庚午 戊子 癸卯 丁巳`
  - 癸水日主，财官印全。
  - 结构：官隔财印，成“三奇”。
  - 结论：这是建禄格中层次最高的路数之一。
- `甲子 丙子 癸丑 壬辰`
  - 癸水日主，子月劫旺，以丙火财为用，兼有食伤通关。
  - 结构：转劫生财。
  - 结论：说明建禄用财必须借食伤，不可赤手夺财。
- `己未 己巳 丁未 辛丑`
  - 丁火日主，劫财之火化为金财。
  - 结构：化劫为财。
  - 结论：比普通“伤食转财”更高一层。
- `庚子 甲申 庚子 甲申`
  - 庚金日主，劫财之金化为生财之水。
  - 结构：化劫为生。
  - 结论：说明月劫不一定坏，关键看是否转成财源。
- `丁巳 壬子 癸卯 己未`
  - 癸水日主，建禄用杀而会局制伏。
  - 结构：杀要用，但必须制伏。
  - 结论：建禄用杀不是单靠凶神压服，而是“杀可用但须驯服”。
- `戊辰 癸亥 壬午 丙午`
  - 壬水日主，用杀而又逢财，最后合杀存财。
  - 结构：先看去留，后论吉凶。
  - 结论：说明“逢财不美”也有例外。
- `甲子 丙寅 甲子 丙寅`
  - 甲木日主，无财官而用伤食泄秀。
  - 结构：木火通明。
  - 结论：比劫旺局不一定非要见官财，也可走文秀一路。
- `癸卯 庚申 庚子 庚辰`
  - 庚金日主，用伤食泄秀。
  - 结构：金水相涵。
  - 结论：秋金见水，灵秀之格。
- `己酉 乙亥 壬戌 庚子`
  - 壬水日主，用官而伤透，庚合乙去伤存官。
  - 结构：官伤并透而能留清。
  - 结论：不是所有“伤官见官”都坏，要看能否去伤存官。

### 例外和易错点
- “建禄身旺必贫”过于粗暴。若有财官杀食立局，建禄反能成大格。
- 用财时最容易误判。很多人只见财就说富，却忽略建禄最先会劫财，必须有食伤过桥。
- 官伤并透未必立败，关键看最后是“官留”还是“伤留”。
- 两官、官杀同见都不是“多多益善”，比劫旺格尤其忌浊。

### 章际关联
- 与第46章《论建禄月劫取运》是完整前后篇。
- 与第43章《论阳刃》同属比劫旺格，但建禄月劫较缓，阳刃更烈。
- 与第41章《论伤官》互通：建禄无财官时用伤食泄秀，和伤官格的“秀气”思路相接。

## 第46章 论建禄月劫取运

### 原文金句
> 祿劫用官，印護者喜財，怕官星之逢合，畏七煞之相乘。

> 祿劫而用傷食，財運最宜，煞亦不忌，行印非吉，透官不美。

### 核心规则
- **建禄用官而印护者喜财** → 财可生官，使官更有力。
- **建禄用官怕官逢合** → 官被合去，制劫之力失效。
- **建禄用官畏七杀相乘** → 杀来混官，破其清纯。
- **建禄用官时伤食未必立凶** → 若官有印护，伤食不一定伤根本。
- **建禄用官时比劫也未即凶** → 因原局本就比劫当令，关键仍在官能否镇得住。
- **财生官者喜印** → 财官太强时，需要印来护身、润格。
- **财生官者宜官有根** → 官根不固则贵力不足。
- **财生官者畏伤食相侮** → 伤食来克官，最易败局。
- **财生官者逢财更有功** → 财是这一路主用，财运可发。
- **财生官者杂杀有碍** → 杀来混官，仍属败点。
- **建禄用财而带伤食，财食重喜印** → 泄财太过，印可护身节泄。
- **建禄用财而带伤食，财食轻喜助财** → 财不足，则宜再扶财。
- **建禄用财不喜印比乱入** → 尤其财轻时，再见印比易阻发财。
- **建禄用财逢杀无伤** → 七杀不一定坏财路。
- **建禄用财遇官非福** → 官来制伤，可能堵住转劫生财之桥。
- **建禄用杀而食制，食重补杀** → 杀不足则扶杀。
- **建禄用杀而食制，杀重补食** → 杀过旺则扶食。
- **用杀带财，若合杀存财，则财运不忌** → 财已转成主用，不再单纯党杀。
- **用杀带财，透官无虑，身旺亦亨** → 在合杀存财的清局里，官与身旺都不一定坏。
- **合财存杀再用食制，仍按轻重定运** → 杀轻补杀，食轻补食。
- **建禄用伤食，财运最宜** → 财是伤食的归宿。
- **建禄用伤食，煞运不忌** → 伤食可制煞，未必为祸。
- **建禄用伤食，印运非吉** → 印来制伤食，堵塞泄秀之路。
- **建禄用伤食，透官不美** → 官来压伤食，秀气受损。
- **伤食太重时印亦不忌** → 因印可节制过度泄气。
- **官杀并出取清后，喜伤食比肩** → 可助制伏、助泄秀。
- **官杀并出取清后，印财官未必佳** → 容易重新搅浊原局。

### 格局成败
- **顺运关键**
  - 永远围绕原局所取的官、财、杀、伤食主线走。
  - 保持“财官清、财食通、杀有制、伤有归宿”。
- **败运信号**
  - 官被合、杀来混官。
  - 用财局行官运堵住伤食。
  - 用伤食局再走印运压住泄秀。
  - 取清后的官杀局再走印财官把浊神扶起。
- **运中救应**
  - 财食重 → 印。
  - 财食轻 → 财。
  - 食重杀轻 → 杀。
  - 杀重食轻 → 食。

### 案例拆解
- `庚戌 戊子 癸酉 癸亥`
  - 用官而印护。
  - 行运喜财，不喜官被合、杀混官。
- `甲子 丙子 癸丑 壬辰`
  - 用财带伤食。
  - 财轻则喜财运，财食太重则反喜印来收敛。
- `丁巳 壬子 癸卯 己未`
  - 用杀而食制。
  - 食重则补杀，杀重则补食。
- `甲子 丙寅 甲子 丙寅`
  - 用伤食泄秀。
  - 财运最宜，印运反易压住木火通明之象。

### 例外和易错点
- 建禄用官时“伤食不能为害”不是绝对免疫，只是官有印护时不至立败。
- 用财局“逢煞无伤”也不是所有七杀都吉，关键看七杀有没有把财局搅成杀局。
- 用伤食局平时忌印，但若伤食太重，印又可救偏，这就是“不可执一”。

### 章际关联
- 与第45章《论建禄月劫》互为表里：前章讲怎么立局，本章讲怎么守局。
- 与第40章《论偏官取运》、第42章《论伤官取运》方法同构，都是“看原局线路，再定补偏”。

## 第47章 论杂格

### 原文金句
> 雜格者，月令無用，以外格而用之，其格甚多，故謂之雜。

> 大約要幹頭無官無煞，方成格，如有官煞，則自有官煞為用，列外格矣。

### 核心规则
- **杂格只在月令无用时启用** → 正格能立，就不必借外格。
- **杂格首重“干头无官无杀”** → 官杀若明透，自有官杀为用，应回归正格。
- **透财尚可，但财太重不取外格** → 财根深、财透两位，则应以财为主，不再立杂格。
- **一方秀气格重“五行成全”** → 如甲乙全亥卯未、寅卯辰，又生春月，是木一气成象。
- **一方秀气喜印露体纯** → 印可护其纯粹，不喜官杀来破其一方之气。
- **从化格须化神得令** → 化出之物得时乘令，是成格第一要件。
- **从化格须四支局全** → 仅有天干五合不够，地支必须成全化神根气。
- **从化不全仍可贵但降格** → 即使不满最高条件，只要处于化神旺地，仍可次贵。
- **倒冲格须“众冲能动”** → 支中字多，力量足以把对宫财官冲出；孤冲不成。
- **倒冲格忌填实** → 一填实，对冲借势即失。
- **朝阳格须干头无木火** → 有火则无待于朝，有木则财来触怒戊土，格不成。
- **合禄格须命无官星** → 既已明见官星，就不必再借合禄求官。
- **合禄格忌填实与官杀透出** → 一旦官杀现成，外格即废。
- **从财格必须真从** → 四柱皆财、身无气，方能弃命保财。
- **从财见印不从** → 印能生身，日主不肯从财。
- **从财见官杀亦不从** → 不存在“既从财又从杀”的混从。
- **从杀格必须真从** → 四柱皆杀、日主无根，方能弃命从杀。
- **从杀见食伤不从** → 食伤制杀，破从势。
- **从杀见印不从** → 印化杀生身，日主又不肯从。
- **井栏格有严苛时令** → 庚金生三七月方可取此格。
- **井栏格忌已透丙丁巳午** → 财官既已现成，就不必借冲起。
- **刑合格靠“刑出财官”** → 癸日甲寅时，借寅刑巳得财官。
- **刑合格忌庚申、戊己** → 庚申冲木、戊己现官杀，都会破格。
- **遥合格要“遥而未实”** → 若已有子字合住丑，或官杀已透，就不算遥合。
- **甲子遥巳格可废** → 作者自己都说牵强，仅因旧例姑存。
- **大量神怪格可弃** → 拱禄、拱贵、鼠贵、魁罡、金神等，作者多斥为牵强附会。
- **杂格也必须服从五行正理** → 即使成立，也只是增重基础，不可凌驾用神常理。
- **伤官伤尽之说不可执死** → 末段专门质疑“见官必祸、行伤运必贵”的绝对论。

### 格局成败
- **成格条件**
  - 月令正格确实无用。
  - 干头不再明透官杀，或财未重到足以单独立格。
  - 该杂格的形势、气势、时令、支局条件全部满足。
  - 仍符合五行生克常理。
- **破格条件**
  - 明明正格可立，却硬套杂格。
  - 外格条件不全，只取一点象意便强论。
  - 干头官杀已透，仍说杂格。
  - 从格不真从，化格不真化，冲格冲不动。
  - 迷信拱禄、鼠贵、魁罡、金神等牵强名目。
- **救应方法**
  - 杂格不成时，应退回正格或普通五行法处理。
  - 从化不足时，按次贵、偏势格看，不必强拔。
  - 任何杂格只要见到明确用神失衡，都以五行常理优先校正。

### 案例拆解
- `癸亥 乙卯 乙未 壬午`
  - 木一气成象，生于春月。
  - 结构：一方秀气格，喜印露体纯。
  - 结论：不是普通比劫局，而是木势全体成格。
- `甲戌 丁卯 壬寅 甲辰`
  - 丁壬化木，支见寅卯辰，卯月当令。
  - 结构：从化格。
  - 结论：虽未必最完满，但已属大贵。
- `戊午 戊午 戊午 戊午`
  - 四午倒冲子财。
  - 结构：倒冲格，靠众午冲动子位之财。
  - 结论：说明倒冲必须“字多力众”。
- `甲寅 庚午 丙午 甲午`
  - 午多倒冲子官。
  - 结论：倒冲不只是冲财，也可冲官。
- `戊辰 辛酉 戊子`
  - 辛日朝阳格，干头无木火。
  - 结构：借戊朝丙，引官而来。
  - 结论：朝阳格最重“无木火杂破”。
- `己未 戊辰 戊辰 庚申`
  - 戊日合禄格。
  - 结论：命无官星，借庚申合乙成官路。
- `己酉 癸未 癸未 庚申`
  - 癸日合禄格。
  - 结论：借申合巳之意成官。
- `庚申 乙酉 丙申 乙丑`
  - 四柱皆财，丙火身无气。
  - 结构：弃命从财。
  - 结论：从财格必须真从，见印即坏。
- `乙酉 乙酉 乙酉 甲申`
  - 乙木日主，四柱皆杀而无根。
  - 结构：弃命从杀。
  - 结论：见食伤、见印都破从。
- `戊子 庚申 庚申 庚申`
  - 庚金三申，取井栏格。
  - 结论：若再透丙丁巳午，则不再是井栏。
- `乙未 癸卯 癸卯 甲寅`
  - 癸日甲寅时，刑合得财官。
  - 结构：刑合格。
  - 结论：若见庚申、戊己则破。
- `辛丑 辛丑 辛丑 庚寅`
  - 丑多遥会巳，取官。
  - 结构：遥合格。
  - 结论：若有子字或官杀已透，则不成遥合。
- `甲申 甲戌 甲子 甲子`
  - 甲子遥巳旧例。
  - 结论：作者认为此格“似觉无情”，基本可废。

### 例外和易错点
- 杂格不是“高阶秘籍”，恰恰是正格无用时的补充方案。
- 看见三合、会局、成象，不等于一定是杂格；要先检查月令、官杀、财星是否已足以立正格。
- 从财、从杀最易误判。只要日主有根、有印、有制，就谈不上真从。

### 总结：十神下半部的总纲

#### 一、偏官（七杀）总纲
- 七杀不是单纯凶神，而是最需要“控制链”的权星。
- 成格主路只有几条：食神制杀、印化杀、财清格、刃当杀。
- 偏官最大禁忌不是“见杀”，而是“杀失控、官杀混、制神被夺”。

#### 二、伤官总纲
- 伤官最怕无归宿，最喜归财、归印、归官、归杀中的某一条清路。
- 伤官的高低，往往由调候决定一半：夏木要润，冬金要暖。
- “伤官见官”为大忌，但化伤为财、金水伤官见官属于明确例外。

#### 三、阳刃总纲
- 阳刃是比劫中最烈者，必须制服或导流。
- 阳刃与七杀的关系很特殊：杀格怕财印乱扶，刃格反常常喜财印扶制神。
- 阳刃成败，不在刃旺不旺，而在“官杀能否真制刃”。

#### 四、建禄月劫总纲
- 建禄月劫不是贵格名目，而是“比劫当令”的原始局势。
- 之后必须另借官、财、杀、伤食立局。
- 用财必须带食伤，这是本章最不可忽略的一条。

#### 五、杂格取舍总纲
- 月令能立正格，绝不轻用杂格。
- 杂格成立的前提，是正格无用且形势条件严密满足。
- 一切杂格最终都要回到五行生克、气势顺逆，不能被格名绑架。

#### 核心规则

- **杂格总原则** → 四柱已经形成偏旺、从化、从势之局 → 行运必须顺其既成气势，不可逆行硬破。

- **先辨真假格** → 若月令已有正用、天干已有清透之财官印杀 → 先按正格论，不必另立杂格。
- **取运重“顺势”不重“名目”** → 同样叫“倒冲”“合禄”“遥合” → 取运不看名字玄不玄，只看原局实际偏向何气。
- **偏旺格忌逆气** → 一方旺格最怕官杀来克、本气被破、异类硬入。
- **从化格忌还原** → 化气格最怕日主本气被拉回原身，化神被克，或官杀破化。
- **从格忌扶身** → 从财、从杀最怕印比扶身、日主得根，导致“从而不真”。
- **伪杂格须还原正格** → 徐注多处明确指出，某些旧名格其实只是食神格、偏官格、偏财格、从儿格，不应误判。

### 二、曲直仁寿格与五种一方旺格

#### 原文金句

> 「甲乙日主，支全亥卯未或寅卯辰，乃曲直仁壽格也。」  
> 「氣勢偏旺於木，宜行水木火運，官煞運最忌，財運亦不宜。」

#### 核心规则

- **曲直格成象** → 甲乙日主 + 地支全亥卯未或寅卯辰 → 木气专旺，成曲直仁寿格。
- **炎上格成象** → 丙丁日主 + 地支全寅午戌或巳午未 → 火气专旺，成炎上格。
- **稼穑格成象** → 戊己日主 + 地支全辰戌丑未 → 土气专旺，成稼穑格。
- **从革格成象** → 庚辛日主 + 地支全巳酉丑或申酉戌 → 金气专旺，成从革格。
- **润下格成象** → 壬癸日主 + 地支全申子辰或亥子丑 → 水气专旺，成润下格。
- **五格同法** → 五种格名虽异 → 取运原则完全一致，都是“顺其本气，顺其流通”。
- **专旺不喜官杀** → 旺木忌金、旺火忌水、旺土忌木、旺金忌火、旺水忌土 → 官杀来克即破势。
- **专旺不喜逆耗** → 虽不如官杀那样立破，但财运若耗泄过头，也会损成象之纯粹。
- **专旺可喜本气、印、泄** → 本气运可助势，印运可扶源，泄秀之运可使偏旺之气有出路。
- **成格关键在“全”与“偏”** → 不是单有三合三会名义就算，必须实见全局偏向一边，其余异气难以抗衡。
- **破格关键在“异类透出得势”** → 若官杀有根有力、财星逆耗太重、或日主反被牵回普通强弱格 → 便不再是纯专旺。

#### 取运方法

- **曲直格** → 喜水木火。
  水能生木，木能助木，火能泄木之秀。
- **炎上格** → 喜木火土。
  木能生火，火可助势，土可承火、泄火之烈。
- **稼穑格** → 喜火土金。
  火能生土，土可助势，金能泄土之秀。
- **从革格** → 喜土金水。
  土能生金，金可助势，水能泄金之秀。
- **润下格** → 喜金水木。
  金能生水，水可助势，木能泄水之秀。
- **五格通忌** → 最忌官杀直接来破本气，其次忌与本气相战的逆行之运。

#### 案例拆解

**原书命例 1：曲直仁寿格**

```text
癸亥 乙卯 乙未 壬午
```

- 日主：乙木。
- 格局判断：地支亥卯未三合木局，日主又是乙木，木势成林，典型曲直格。
- 为什么成格：
  亥卯未全，木气一边倒。
  天干又见乙、壬、癸，都是生扶木势，不见有力金杀破局。
- 取运重点：
  喜水木继续扶助根源。
  喜火泄秀，使木气有文明之用。
  忌金官杀斩木。
  土财耗木，亦不为佳。

**原书命例 2：炎上格（借“倒冲格”条下举例）**

```text
甲寅 庚午 丙午 甲午
```

- 日主：丙火。
- 格局判断：三午并临，寅午半会火势，庚金无根，徐注明确说“置之不论，气偏木火，格成炎上”。
- 为什么成格：
  火有三午得地，甲木两透生火。
  庚金虽见，但无根无助，不足破格。
- 取运重点：
  最宜土运泄火之气。
  木火运仍属顺势。
  水运最忌，金运亦非首选。

**稼穑格、从革格、润下格**

- 本章未另附独立四柱案例，只给出成格框架。
- 取法上完全依曲直、炎上的母式推演: `本气、印、泄为顺；官杀逆克为忌。`

#### 例外和易错点

- **易错点 1** → 见三合三会就断专旺。  
  错在只认地支名称，不看天干异气是否足以破局。
- **易错点 2** → 认为专旺只能走比劫印。  
  错在忽略“泄秀”也是顺势，木旺喜火、火旺喜土、金旺喜水，都是本章明示思路。
- **易错点 3** → 把“财运亦不宜”机械推广到所有五格。  
  曲直格是木旺耗于土财，故财不宜；其他四格也应按五行顺逆推，不可生搬字面。
- **易错点 4** → 把炎上格只看成“火越旺越好”。  
  徐注在后文强调“最宜土运泄火之气”，说明专旺也怕偏枯，仍需有出路。

### 三、化气格

#### 原文金句

> 「丁壬化木，生於春月，時逢甲辰，木之元神透出，乃化木格。」  
> 「化神喜行旺地，最宜東方寅卯辰比劫鄉，而忌官煞，日主還原之地亦忌。」

#### 核心规则

- **化气格成立** → 天干成五合 + 生于化神旺月 + 化神元神透出或得根 → 才能真化。
- **本例成木化** → 丁壬化木 + 生春月 + 时逢甲辰透木根 → 化木成立。
- **化神为第一判断中心** → 一旦真化 → 不再按原日主强弱取运，而按化神之旺衰取运。
- **喜化神旺地** → 化木喜寅卯辰，化火喜巳午未，化土喜辰戌丑未，化金喜申酉戌或巳酉丑，化水喜亥子丑或申子辰。
- **喜生化神之运** → 若所行运能生扶化神，也可为吉，不必拘泥只走本气。
- **忌官杀破化** → 官杀若直接克制化神，破掉新形成的气势，即为大忌。
- **忌日主还原** → 所行大运若强拉原日主复起，使“化而复返”，也属破格。
- **五种化气同法** → 甲己化土、乙庚化金、丙辛化水、丁壬化木、戊癸化火，原理一致。
- **不同化局忌神不同** → 是否为忌，不看抽象名字，只看该运是生化神还是克化神。

#### 取运方法

- 真化之后，以**化神**为用神中心。
- **首选**：化神临官、帝旺、长生之地。
- **次选**：能生化神之印运。
- **大忌**：官杀来克化神。
- **大忌**：原日主得根得势，导致化局还原。

#### 案例拆解

```text
甲戌 丁卯 壬寅 甲辰
```

- 日主：壬水。
- 合化关系：丁壬合木。
- 成格关键：
  生于卯月，木当令。
  时上甲辰，甲木透出，辰中乙癸戊，仍以木气、湿土培木为助。
  全局气势偏木，不再以壬水原身为重。
- 取运结论：
  最宜东运寅卯辰，顺化木之势。
  亥子丑运虽属壬水印地，但能生木，也可吉。
  官杀金运破木，为忌。
  若走使壬水原神复旺、化木不真的路，也不吉。

#### 例外和易错点

- **易错点 1** → 见五合就当化气格。  
  徐注明示必须“生于化神旺月”且“元神透出”，否则只是合而不化。
- **易错点 2** → 化气格只喜化神本气，不喜印。  
  本章明确指出，丁壬化木走亥子丑印地，因能生木，仍吉。
- **易错点 3** → 所有化气格都怕印。  
  不对。是否忌印，关键是印会不会把局势拉回原日主。
- **易错点 4** → 甲己化土也走寅卯辰。  
  徐注专门举反例：甲己化土若行寅卯辰，木克土化神，反成大忌。

- `knowledge/断命流程指南.md` 的合化条件“化神当令、无冲破”与本章完全一致。
- `knowledge/三命通会/卷十.txt` 有“丙辛化水到水乡”“丁壬化木临木位”之语，与徐注同调。

### 四、倒冲格

#### 原文金句

> 「兩神成象，而氣勢偏於火土，為從旺格。」  
> 「所謂倒衝最忌填實，即此意也。」

#### 核心规则

- **倒冲名不重要，成象最重要** → 虽俗称倒冲格 → 徐注实际先看“两神成象”“从旺”。
- **第一例是火土从旺** → 四午并列，火土一气 → 取运核心不是“冲出什么”，而是顺火土之势。
- **第二例是炎上格** → 徐注直言“庚金无根，置之不论，气偏木火，格成炎上”。
- **最怕填实** → 所谓“填实”，本质就是外来异气把原先可用的偏旺结构堵死、冲散、拉偏。
- **偏枯须防燥烈** → 火土太燥时，虽喜顺势，也要取“带水之土”这种润燥而不逆势的路。
- **见逆运则灾** → 水运、木运若直接逆转火土格局，往往应灾很快。

#### 取运方法

- 火土从旺例 → 最宜金运泄土，尤喜带水之湿土运，如庚辰、辛丑。
- 忌纯水运。
- 忌木运。
- 木火炎上例 → 最宜土运泄火；木火运亦可顺势；水运最忌。

#### 案例拆解

**原书命例 1**

```text
戊午 戊午 戊午 戊午
```

- 日主：戊土。
- 格局判断：四柱火土一片，徐注不取“冲禄”神煞义，直接判为“两神成象”“从旺格”。
- 取运重点：
  喜金运泄土。
  以庚辰、辛丑为最佳，因为既有金，又有湿土护局，不至于燥裂。
  见纯水则逆旺，不是滋润，而是骤灾。
  木运克土、助火，也不宜。

**原书命例 2**

```text
甲寅 庚午 丙午 甲午
```

- 已在前节说明，本质是炎上格。
- 徐注把它归在“俗所谓倒冲格”之下，目的就是纠偏：不要执着名称，要看真实气势。

#### 例外和易错点

- **易错点 1** → 一见倒冲就死记“怕填实”。  
  徐注这里已经把“填实”翻译成五行语言：异气逆势、格局被堵，即为填实。
- **易错点 2** → 火土太旺就一定喜水。  
  本例恰恰相反，纯水是逆势破局，不是调候解救。
- **易错点 3** → 只要名称叫倒冲，取运都一样。  
  第一例是火土从旺，第二例是炎上格，实际喜忌并不完全相同。

### 五、朝阳格

#### 原文金句

> 「此金水傷官，原無官星，氣勢偏於金水，以順其性。」  
> 「行土金水運為美，火運為忌。」

#### 核心规则

- **朝阳格本质** → 不是神秘贵格，而是金水伤官格。
- **成格条件** → 金水成势 + 原局无官星来犯伤官。
- **取运核心** → 顺金水之性，不要火来克金、耗水。
- **土亦可喜** → 土能生金，且可统摄金水之气，所以土运为美。
- **木运分清浊** → 带水之木，因金水能转生木，尚可；带火之木则助火逆势，不宜。
- **见官破格** → 若官星出现，则从“金水伤官无官可顺”转为“伤官见官”，格性全变。

#### 取运方法

- 喜土金水运。
- 木运要看是否带水。
- 火运为大忌。

#### 案例拆解

```text
戊辰 辛酉 辛酉 戊子
```

- 日主：辛金。
- 格局判断：酉酉双临，子水泄金，土金水成势，且原局无火官星，故按金水伤官顺气取运。
- 取运重点：
  土运生金，金运帮身，水运泄秀，皆美。
  火来则伤官见官，破其顺势。
  若木中藏水，可视作金水流通到木，尚不为大害。

#### 例外和易错点

- **易错点 1** → 辛日见子时就一定是六阴朝阳。  
  徐注并不按“朝阳格”神名取，而是落回金水伤官。
- **易错点 2** → 金水伤官一定喜木财。  
  本例只说“带水之木尚可”，并未把木运列为主喜。

#### 与他书交叉验证

- `knowledge/穷通宝鉴.md` 有“辛逢卯日，子时，名曰朝阳”，偏重传统格名。

- 徐注的独特之处在于：不先信格名，先判它是不是金水伤官无官之局。

### 六、合禄格（实为土金食神格）

#### 原文金句

> 「此土金食神也。比劫重重，氣勢偏於土金，以金運洩土之秀為最吉，水運亦美，火運為忌，木亦不美。」  
> 「俗以庚合乙為官星，稱為合祿格。」

#### 核心规则

- **本例不必先信合禄名目** → 先看实际气势是土金食神格。
- **成格关键** → 比劫重重，土旺生金，金能泄土，格局之妙在“食神泄秀”。
- **最喜金运** → 因为金能承土气，发其秀。
- **水运次吉** → 金水相生，食神之气继续流通。
- **火运为忌** → 火再生土，使土愈重而闭塞。
- **木运不美** → 土盛木折，木不足以疏土，反受伤。
- **旧说可参考但非中心** → 若从“庚合乙为官”立合禄，取运大略也相近；但判断中心仍应放在土金之势。

#### 取运方法

- 首喜金运。
- 次喜水运。
- 忌火运。
- 木运不美。

#### 案例拆解

```text
己未 戊辰 戊辰 庚申
```

- 日主：戊土。
- 格局判断：辰辰未皆土，庚申透出，土旺生金，金为食神泄秀，故从土金食神论。
- 取运重点：
  金运最能发秀。
  水运承金而流通，亦美。
  火运增土，反致壅滞。
  木运欲疏土而力单，常被反折。

#### 例外和易错点

- **易错点 1** → 见庚合乙就机械按合禄格推。  
  徐注承认旧名存在，但明说真正好处仍在土金食神之气。
- **易错点 2** → 土旺一律喜木疏。  
  本例土势太重，木不但未必成用，反有“土盛木折”之患。

### 七、合禄格辨伪（实为偏官佩印）

#### 原文金句

> 「月令偏官，年上透出，時上庚印化煞為用，格正局清，有何不美？」  
> 「若取巳中戊土官星，豈非官煞混雜耶？」

#### 核心规则

- **此例不是杂格** → 月令偏官透年，时上庚印化杀，已经是标准偏官佩印。
- **正格优先** → 既然正格能成立，就不必再绕去立“合禄格”。
- **印化杀为用** → 此造之清，在于庚印化煞，而不在申合巳中戊土。
- **硬取暗官反成混杂** → 若再把巳中戊土拉出来当官星，反而造成官杀混杂。
- **辨格先看月令透神** → 月令偏官既已透出，取格主线就已经确定。

#### 取运方法

- 不应按杂格取运。
- 应按偏官佩印格取运：以护印、化杀、维持清局为主。
- 忌再引动官杀混杂之气。

#### 案例拆解

```text
己酉 辛未 癸未 庚申
```

- 日主：癸水。
- 格局判断：未中己土偏官，年干己透，时干庚印，形成杀印相生。
- 徐注结论：这已经“格正局清”，无需另立合禄。
- 取运重点：
  不是去等“申合巳中戊土”这种暗官发动。
  而是守住印化杀这条正格主线。

#### 例外和易错点

- **易错点 1** → 见暗合暗拱就比明透月令更重要。  
  徐注态度很清楚：明透月令是主，暗合只是枝节。
- **易错点 2** → 只要能多拉出一个官，就一定更贵。  
  不对。多拉一个官，往往就是把清格变浊格。

#### 与他书交叉验证

- `knowledge/十神关系速查.md` 的“杀印相生”可以直接解释此局。
- 本例是整章里最典型的“伪杂格纠偏”样板。

### 八、从财格

#### 原文金句

> 「丙火臨申，坐於病地，四柱無根，時上己丑又來生金，氣勢偏於金旺，為棄命從財格也。」  
> 「運宜行土金水，南方火鄉最忌，木亦不利。」

#### 核心规则

- **从财格成立** → 日主极弱 + 财星成势 + 四柱无根 + 印比不能扶身。
- **乙从庚化不作印论** → 本例乙木虽然可作印，但因乙庚合化，不再按印星扶身看。
- **日主坐病地** → 丙火坐申，本就弱；再见申酉丑会金，更无立足处。
- **从财必须真弃命** → 只要还有根气、比印能扶，就不是纯从财。
- **土金水皆可顺财势** → 土生金，金为财，水泄金而使财气流通，故皆可行。
- **火运最忌** → 火扶日主、又克金财，直接破从。
- **木运不利** → 木为印，生扶日主，也会破从财之局。

#### 取运方法

- 喜土运、生财源。
- 喜金运、助财势。
- 喜水运、泄财秀。
- 忌火运扶身破从。
- 忌木运生身破从。

#### 案例拆解

```text
庚申 乙酉 丙申 己丑
```

- 日主：丙火。
- 格局判断：
  申酉丑连成金势，财星一片。
  丙火坐申病地，四柱无寅午巳等根。
  乙木又被庚合，不能真作印。
  己丑再生金，故为真从财。
- 取运重点：
  土金水一路皆顺局。
  火来扶身最坏。
  木来生火，也会破局。

#### 例外和易错点

- **易错点 1** → 天干有一个印就不能从。  
  本例告诉你，不但要看“有没有印”，还要看“印能不能成立”。乙庚合后，印性已失。
- **易错点 2** → 从财只喜财运，不喜水运。  
  徐注明说土金水皆可，说明从财也喜财势流通，不是只许更旺不许泄秀。

#### 与他书交叉验证

- `knowledge/断命流程指南.md` 对从财格的总则是“喜食伤生财、财运，忌印比”，与本章完全一致。
- `knowledge/十干精解与补充.md` 也有“从财格喜食伤生财、财运；忌印比运”的提要，本章则提供了真从的判法细节。

### 九、从杀格

#### 原文金句

> 「乙木無根，氣勢偏於金，為棄命從煞格。」  
> 「金運最美，水土亦吉。木運為乙木逢根，火運逆其旺勢，皆忌見。」

#### 核心规则

- **从杀格成立** → 日主无根极弱 + 官杀成势 + 印比不能救。
- **本例是金杀成局** → 三酉一申，金势极重，乙木毫无立锥之地。
- **金运最美** → 直接助杀势。
- **水土亦吉** → 土生金，水泄金而流通，不破从杀之意。
- **木运最忌** → 木来即给乙木生根，日主一有根气，从杀即破。
- **火运亦忌** → 火制金杀，逆其旺势。
- **与从财同法** → 本质都属于“弃命从势”，只是所从之神不同。

#### 取运方法

- 首喜金运。
- 次喜土运、水运。
- 忌木运。
- 忌火运。

#### 案例拆解

```text
乙酉 乙酉 乙酉 甲申
```

- 日主：乙木。
- 格局判断：
  三酉一申，官杀金势压倒一切。
  乙木无寅卯亥未根，无印星解救。
  甲木虽透，但孤立无根，不足挽局。
  故为真从杀。
- 取运重点：
  金最美，顺杀势。
  水土仍可助金、泄金。
  木火都会打破“全盘归杀”的态势。

#### 例外和易错点

- **易错点 1** → 有甲木透出就不能从。  
  关键不在“透不透”，而在“有没有根、有没有势”。无根孤甲仍救不了乙木。
- **易错点 2** → 从杀见火一定是好事，因为可制杀。  
  对正格也许如此，对从杀却恰恰相反；制杀就是破从。

#### 与他书交叉验证

- `knowledge/断命流程指南.md` 写从杀“喜财生杀、杀运，忌食伤制杀”，与本章精神一致。
- `滴天髓阐微.txt` 在杂格总论中将“从官杀”列为变格正理，不与奇诡外格同列，和徐注立场接近。

### 十、井栏叉格

#### 原文金句

> 「庚金乘旺洩秀，支全申子辰水局。氣勢偏於金水，當順其勢以取運。」  
> 「土金水運均美，木運亦可，行火運逆其旺勢不利。」

#### 核心规则

- **井栏叉格本质** → 庚金旺而泄于水，金水成象。
- **成格关键** → 申子辰全，且庚金能乘旺泄秀。
- **顺势比名目更重要** → 不必纠缠“叉冲何方”，要看是否已成金水之局。
- **土金水皆美** → 土生金，金助局，水泄秀，三类都顺。
- **木运亦可** → 金水能流通至木，木不至于立即破局。
- **火运最忌** → 火克金、耗水，直接逆旺势。

#### 取运方法

- 喜土、金、水。
- 木运可行但非上选。
- 火运不利。

#### 案例拆解

```text
戊子 庚申 庚申 庚辰
```

- 日主：庚金。
- 格局判断：
  三庚透干，申申得禄，辰子会水，金水成象。
  徐注核心不在“井栏叉冲贵气”，而在“庚金乘旺泄秀”。
- 取运重点：
  土金水最好。
  木可作为次顺之运。
  火来则破金水之象。

#### 例外和易错点

- **易错点 1** → 井栏叉一定只喜东方财地。

- **易错点 2** → 井栏叉一定贵。

### 十一、遥合格（丑遥巳格）

#### 原文金句

> 「此俗所謂醜遙巳格。土金成局，生於十二月，時上寅木無氣，不能為用。」

- **木火为逆运** → 木火若无根反来逆势，多徒增冲战。
- **遥巳格只是旧名** → 取运仍回归土金之势。

#### 取运方法

- 喜土金水。
- 忌木火。

#### 案例拆解

```text
辛丑 辛丑 辛丑 庚寅
```

- 日主：辛金。

  三辛并列，丑月寒湿土金成局。

  故不从“遥合巳禄”玄义取运，而从土金势象取运。
- 取运重点：
  土金水皆顺。
  木火若来逆旺，反生波折。

#### 例外和易错点

- **易错点 1** → 时上见寅，就强行当财。  
  徐注明说“寅木无气，不能为用”。
- **易错点 2** → 遥合格一定喜官。

### 十二、丑遥巳禄格（又名刑合，实为从儿格）

#### 原文金句

> 「以寅刑出巳中戊土為格，其實乃從兒格也。」  
> 「從格忌見比劫，而從兒有食傷引化，不忌比劫，此為不同之點。」

#### 核心规则

- **旧名刑合格，实质从儿格** → 徐注直接改判，不承认它是独立神格。
- **从儿格成立** → 食伤成势，日主不与之争，气势偏于儿神所代表之五行。
- **本例气偏于木** → 癸水生卯木、寅木，木气外泄，形成从儿。
- **最喜木火** → 木为食伤本气，火为食伤所生之财，顺其流通。
- **官杀大忌** → 官杀来克木、制食伤，破其从儿之路。
- **印运亦忌** → 印来生身，使日主回头，不再纯从儿。
- **比劫是本格特例** → 一般从格忌比劫，但从儿格因有食伤引化，比劫未必立坏。

#### 取运方法

- 首喜木运。
- 次喜火运。
- 忌官杀运。
- 忌印运。
- 比劫不作一般从格那样重忌，需看是否仍能顺木火之流。

#### 案例拆解

```text
乙未 癸卯 癸卯 甲寅
```

- 日主：癸水。
- 格局判断：
  两卯一寅，木势极旺。
  癸水之气尽泄于木，形成从儿而非“寅刑巳中戊土”的刑合贵格。
- 取运重点：
  木火最好，顺木势而生财。
  官杀克木最忌。
  印运回头生身，也会破从儿。
  比劫较一般从格宽，不必一见即凶。

#### 例外和易错点

- **易错点 1** → 把所有从格都按“忌比劫”处理。  
  本章专门指出，从儿格是例外。
- **易错点 2** → 只盯“寅刑巳”神煞关系。  
  徐注态度非常明确：那只是旧口诀，真正成局的是木泄水势。

#### 与他书交叉验证

- `knowledge/断命流程指南.md` 已列从儿格“喜财运、忌印运”，和本章“木火最喜、印运亦忌”正合。
- 本章最重要的补充，是指出从儿格对比劫的容忍度高于一般从格。

### 十三、子遥巳禄格（又名遥合格，实为偏财格）

#### 原文金句

> 「其實月令偏財，用財損印，何必另取格局？」  
> 「戌藏丁火，生起財星，遇運透清為美，庚辛申酉官煞生印為忌。」

#### 核心规则

- **此例不是遥合格主论** → 月令偏财已明，先按偏财格看。
- **兼有财损印之象** → 用财则印受损，故吉凶重点在财印消长。
- **戌中丁火可生财** → 火透则财星得生，财路更清，故“遇运透清为美”。
- **官杀生印为忌** → 金官杀发动，转生印星，反来扶身坏财。
- **午冲子、丑刑戌皆忌** → 一是冲破原局结构，一是刑坏财库与火根。
- **不要为了旧口诀另立遥合** → 正格已足，另加遥合只会模糊主线。

#### 取运方法

- 应按偏财格兼财损印取运。
- 喜火透生财、土财得清之运。
- 忌庚辛申酉之官杀生印运。
- 忌午冲子、丑刑戌之类破局之运。

#### 案例拆解

```text
甲申 甲戌 甲子 甲子
```

- 日主：甲木。
- 格局判断：
  月令戌土为偏财。
  戌中藏丁火，可以生财。
  双子滋印，财印之间已有拉扯，所以关键在“用财损印”，而不在遥合名目。
- 取运重点：
  火透可生财而清。
  土财得生亦美。
  庚辛申酉发动官杀生印，反使印重坏财，为忌。
  午冲子、丑刑戌，都属破局之象。

#### 例外和易错点

- **易错点 1** → 甲子日再逢子时，就先按子遥巳禄格断。  
  徐注明确反对，认为月令偏财已经足够立格。
- **易错点 2** → 官杀一定是贵气。  
  本例官杀一来会生印，反而坏掉偏财之用。

### 十四、本章结论

#### 一句话总括

`杂格取运，贵在辨气，不贵在记名。`

#### 可直接落地的实战结论

- **专旺格** → 喜本气、印、泄，忌官杀逆克。
- **化气格** → 喜化神旺地与生化神之运，忌还原、忌破化。
- **从财格** → 喜土金水，忌火木扶身。
- **从杀格** → 喜金土水，忌木火破从。
- **从儿格** → 喜食伤与财，忌官杀与印；比劫不必一概重忌。
- **朝阳、合禄、井栏、遥合等旧名格** → 先还原成食神格、伤官格、偏财格、偏官格、金水成象格，再谈取运。

#### 本章最重要的纠偏价值

- 不要为了“格名好听”硬立杂格。
- 不要把“填实、遥合、刑合、倒冲”当成脱离五行的神秘按钮。
- 真正高明的判断，是把所有杂格都翻译成普通但精确的五行语言:
  偏旺就顺旺。
  从化就顺化。
  从财就顺财。
  从杀就顺杀。
  正格可立时，就回正格，不必旁生枝节。

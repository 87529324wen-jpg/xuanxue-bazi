# 子平真诠精读 — 核心推理规则提取

## 一、用神总纲（最核心）

**"八字用神，专求月令，以日干配月令地支，而生克不同，格局分焉。"**

### 用神五法

1. **扶抑** — 日元强者抑之，弱者扶之。月令之神太强则抑，太弱则扶
2. **病药** — 以扶为喜，则伤其扶者为病；以抑为喜，则去其抑者为病。除病神即药
3. **调候** — 金水生于冬令、木火生于夏令，气候太寒太燥，以调和气候为急
4. **专旺** — 四柱气势偏于一方，其势不可逆，惟有顺其气势为用（从格/化格/专旺格）
5. **通关** — 两神对峙，强弱均平，各不相下，须调和之

### 顺用与逆用

**四吉神（财/官/印/食）→ 顺用：**
- 财喜食神相生，生官护财
- 官喜透财相生，生印护官
- 印喜官煞相生，劫财护印
- 食喜身旺相生，生财护食

**四凶神（煞/伤/劫/刃）→ 逆用：**
- 七煞喜食神制伏，忌财印资扶
- 伤官喜佩印制伏，生财化伤
- 阳刃喜官煞制伏，忌官煞俱无
- 月劫喜透官制伏，利用财而透食化劫

## 二、成格/败格/救应

### 成格条件
- **正官格成**：官逢财印，无刑冲破害
- **财格成**：财生官旺 / 财逢食生而身强带比 / 财格透印位置妥贴
- **印格成**：印轻逢煞 / 官印双全 / 身印两旺用食伤 / 印多逢财而财透根轻
- **食神格成**：食神生财 / 食带煞而无财 / 弃食就煞而透印
- **七煞格成**：食神制煞 / 印绶化煞 / 阳刃合煞

### 败格条件
- **正官格败**：官逢伤克刑冲
- **财格败**：财轻比重 / 财透七煞
- **印格败**：印轻逢财 / 身强印重而透煞
- **食神格败**：食神逢枭 / 生财露煞
- **七煞格败**：七煞逢财无制
- **伤官格败**：非金水而见官 / 生财带煞 / 佩印而伤轻身旺
- **阳刃格败**：无官煞

### 救应规则
- 官逢伤 → 透印解之 / 杂煞合煞清之 / 刑冲会合解之
- 财逢劫 → 透食化之 / 生官制之 / 逢煞食神制煞生财
- 印逢财 → 劫财解之 / 合财存印
- 食逢枭 → 就煞成格 / 生财护食
- 煞逢食制 → 印来护煞 / 逢财去印存食

**"八字妙用，全在成败救应，其中权轻权重，甚是活泼。"**

## 三、用神变化

**"用神非变化不灵"**

月令所藏不一，用神遂有变化。以寅月为例：甲为本主（知府），丙为长生（同知），戊亦长生（通判）。若寅月不透甲而透丙，则同知作主。

### 变之而善
- 辛生寅月，逢丙化财为官
- 壬生戌月逢辛化煞为印
- 癸生寅月，藏甲透丙，会午会戌 → 伤化为财

### 变之而不善
- 丙生寅月，本为印绶，甲不透而会午戌 → 化为劫
- 丙生申月，本属偏财，藏庚透壬，会子会辰 → 化为煞

## 四、纯杂之分

**"纯者吉，杂者凶"**

- **纯**：互用而两相得（如辛生寅月甲丙并透 → 财与官相生）
- **杂**：互用而两不相谋（如壬生未月乙己并透 → 官与伤相克）

---

## 五、格局高低

**"有情无力、有力无情"是判断格局高低的核心**

- **有情**：用神与日主有情（如甲用酉官，壬合丁以清官）
- **有力**：用神自身有力（如身强煞露而食神又旺）
- **有情兼有力**：最高格局
- **有情而非情之至**：格局高而次
- **有力而非力之全**：格局高而次
- **无情无力**：贱格

---

## 六、旺衰强弱辨析（关键！）

**"得时为旺，失时为衰；党众为强，助寡为弱"**

- 旺≠强，衰≠弱
- **虽旺而弱**：得时但党寡（如甲乙木生寅卯月，但干庚辛支酉丑 → 金党众木助寡）
- **虽衰而强**：失时但党众（如甲乙木生申酉月，但比印重叠年日时通根比印）

### 通根层级（非常重要）
- **天干得一比肩** < **支中得一墓库**（如甲逢未、丙逢戌）
- **得二比肩** < **支中得一余气**（如乙逢辰、丁逢未）
- **得三比肩** < **支中得一长生禄旺**（如甲逢亥寅卯）
- **阴长生不作此论**（如乙逢午、丁逢酉 → 仅比余气）

**"盖比劫如朋友之相扶，通根如家室之可托；干多不如根重，理固然也。"**

---

## 七、刑冲会合解法

### 基本规则
- 三合六合可以解刑冲
- 如甲生酉月逢卯冲 → 支有戌则卯戌合不冲 / 有辰则酉辰合不冲

### 因解反得刑冲
- 如甲生子月，支逢二卯，又逢戌 → 戌卯合去其一 → 一合一刑

### 会合不能解刑冲
- 如子年午月日坐丑 → 丑子合可解冲 → 但时逢巳酉 → 丑巳酉会 → 子复冲午

### 以刑冲解刑冲
- 如丙生子月卯刑子 → 支又逢酉 → 酉冲卯不刑月令
- **"刑冲用神尤为破格，不如以另位之刑冲解月令之刑冲"**

---

## 八、十干得时不旺失时不弱

**"日干不论月令休囚，只要四柱有根，便能受财官食神而当伤官七煞"**

- 长生禄旺 = 根之重者
- 墓库余气 = 根之轻者
- **干多不如根重** — 3个天干比肩不如1个地支长生禄旺

---

## 九、十干合化规则

### 合而不合的情况
1. **隔位相间** — 如甲己中间有庚间隔 → 合而不敢合
2. **隔位太远** — 如甲在年干己在时上 → 合而不能合（半合，祸福得十之二三）
3. **合而无伤** — 如甲生寅卯月时两透辛官，年丙合月辛 → 合一留一
4. **本身之合** — 五阳逢财五阴遇官俱是作合，但本身合不为合去

### 争合妒合
- 两辛合丙 / 两丁合壬 → 一夫不娶二妻 → 情不专
- 两合一而隔位 → 全无争妒

## 十、行运规则

**"论运与看命无二法"**

- 喜神运 → 吉
- 忌神运 → 凶
- 运能成格变格（大运可以改变原局格局）
- 运中透清（地支中藏干在运中透出天干）→ 力量大增

---

## 十一、六亲宫位

- 年柱 = 祖宗/父母（根基）
- 月柱 = 父母/兄弟（提纲）
- 日柱 = 本人/配偶（夫妻宫）
- 时柱 = 子女/晚年（辅佐）

**"月令有用神得父母力，年有用神得祖宗力，时有用神得子孙力"**

## 核心推理模式（直接可用于benchmark）

### 推理模式1：取用神
1. 看月令 → 确定格局类型
2. 配四柱 → 看成格/败格
3. 有败格因素 → 看有无救应
4. 无救应 → 格局低

### 推理模式2：判断旺衰
1. 先看月令（50%权重）
2. 再看通根（地支藏干/长生禄旺）
3. 最后看天干帮身
4. **干多不如根重**

### 推理模式3：流年断事
1. 看流年与原局的天干作用（合/克/生）
2. 看流年地支与原局地支的刑冲会合
3. 看是否动了用神/相神/忌神
4. 动用神 → 吉；动忌神 → 凶；用忌都动 → 看救应

### 推理模式4：格局高低判断
1. 有情（用神与日主配合好）+ 有力（用神自身强）= 最高
2. 有情而无力 / 有力而无情 = 中等
3. 无情无力 = 最低
4. 成中有败（带忌）→ 看救应程度减等

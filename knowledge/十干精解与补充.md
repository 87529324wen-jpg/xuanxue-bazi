## 一、十天干精解（滴天髓原文+解析）

### 甲木（参天大树）
滴天髓："甲木参天，脱胎要火。"
解析：纯阳参天之木，生旺须有丙火引化。春不宜庚金克制（金克未旺之木）。秋不宜土（秋金旺土厚埋木）。火旺辰土护根则贵。水旺亥子浮根则弱。地支有根天干流通则富贵绵长。

### 乙木（花草藤萝）
滴天髓："乙木虽柔，刲羊解牛。"
解析：柔弱如藤草，能克未土解丑。怀丁（暗藏丙丁）可克庚辛。跨酉骑申（金局）需火解救。藤萝依附甲木大树，则四季皆宜。

### 丙火（太阳）
滴天髓："丙火猛烈，欺霜侮雪。"
解析：刚烈，寒冬不畏霜雪。能炼庚金（偏财格）。逢辛则合化水反弱。土多则火气泄散（土众成慈）。水旺则显节操之志。寅午戌火局加甲木生火，反有焚灼之忧。

### 丁火（灯火星光）
滴天髓："丁火柔中，内性昭融。"
解析：阴柔灯烛之火。乙木为母孝顺相生。合壬水为忠义之象。旺时温和不激烈。衰时亦不至穷绝。有甲木嫡母生扶则秋冬皆可存活。

### 戊土（高山厚土）
滴天髓："戊土固重，既中且正。"
解析：高山厚重，中正沉稳。静则收缩，动则开辟，主宰万物。水润则万物生长（须癸水滋润）。火燥则物病枯焦（须防过燥）。寅戌位须避冲。

### 己土（田园湿土）
滴天髓："己土卑湿，中正蓄藏。"
解析：田园卑湿之土，中正而能蓄藏。甲乙木旺无妨（木克松土可疏）。水旺亦可承受。火弱则晦暗（需丙暖）。金多反光（辛金秀气）。须得生扶方旺。

### 庚金（刚刃利剑）
滴天髓："庚金带煞，刚健为最。"
解析：刚健如利刃，自带杀气。得壬水则清秀（金白水清）。得丁火锻炼则锐利（最需丁火）。戊土湿润则生金。土干则金脆无用。须丁甲配合方成贵格。

### 辛金（珠玉首饰）
滴天髓："辛金软弱，温润而清。"
解析：柔弱如珠玉，温润清秀。畏戊土掩埋（浊金）。喜壬水淘洗（金白水清）。秋令成熟时最为清贵。可扶社稷（官印相生）救生灵（辛合丙）。

### 壬水（江河大海）
滴天髓："壬水通河，能泄金气。"
解析：如大江大河，能泄庚辛金气。刚健周流不停息。通根亥子之地，水势浩大。甲木化水（食神泄秀）有情。戊土从杀（水旺从势）相济。

### 癸水（雨露泉水）
滴天髓："癸水至弱，达于天津。"
解析：阴柔至弱，天乙贵人之水。得辰（龙）则运化神奇。不担忧火土（癸能滋润）。金生水本无需论。合戊土见丙火则化（戊癸化火格）。

## 二、神煞在四柱的详细断法

### 天乙贵人在四柱
- 年柱贵人 → 护祖上，出身有靠
- 月柱贵人 → 助父母，少年有贵人
- 日柱贵人 → 助自身，一生逢凶化吉
- 时柱贵人 → 子女贤，晚年有靠

### 文昌贵人在四柱
- 文昌坐命（日柱）→ 利文职考试
- 文昌临学宫（月柱）→ 书香气最旺
- 文昌临印绶 → 更加聪慧

### 华盖在四柱
- 华盖临日支 → 婚姻有阻
- 华盖+空亡 → 僧道之命
- 华盖+将星 → 技术型权威
- 华盖逢冲 → 精神层面大变动

### 驿马在四柱
- 驿马临财星 → 财从外来
- 驿马临官星 → 异地升官
- 驿马逢冲 → 奔波更甚
- 驿马逢合 → 动而止

### 桃花在四柱
- 桃花临日支 → 坐桃花，感情最旺
- 桃花逢吉神 → 感情美满
- 桃花逢凶神（空亡、刑冲）→ 感情多变
- 桃花带七杀 → 因色惹祸

## 三、十神深层含义（补充）

### 比肩旺
个性独立，不服约束，易与上司矛盾。最怕正官制衡。
身旺比肩多 → 穷困、竞争激烈、合伙纠纷
身弱比肩多 → 反而帮身，吉

### 劫财旺
冲动好赌，财来财去，感情多变。男命劫财旺 → 妻子不顺。
劫财夺财 → 破财、合伙被坑
劫财配七杀 → 胆大妄为

### 正财
正当财富、妻（男命）、稳定收入、努力所得。
正财透干 → 有正当职业、收入稳定
正财坐库 → 财能守

### 偏财
意外之财、父（男命）、情人（男命）、不劳而获之财。
偏财旺 → 赚钱门路多但不稳定
偏财合印 → 贪财坏印

### 七杀（偏官）在女命
七杀为情人星（非正缘）。七杀旺 → 感情复杂，多婚或有婚外情。
女命日坐七杀 → 丈夫强势/婚姻有压力
女命七杀有制 → 嫁贵夫

### 正官在女命
正官为丈夫星（正缘）。正官透干 → 婚姻稳定
正官被合 → 丈夫被抢/丈夫有外心
正官逢冲 → 婚姻变动

## 四、格局补充详解

### 正官格取格
月令透出正官，或月支正官藏透，日主与官星生克有情。
典型忌：甲日庚月（庚为七杀非正官）、癸日甲月（甲为伤官，伤官见官大忌）

### 七杀格取格
月令透出七杀，须有制（食神/印）或有化（印）。
典型：甲日庚月须丙丁制庚或戊己合杀

### 偏印格（枭印格）
月令透出偏印（枭神），偏印克食神为枭神夺食。
偏印格成格：须有财星制枭，或有比劫化枭

### 从财格取格
日主极弱，财星极旺，无比劫印星救助，弃命从财。
从财格喜：食伤生财、财运
从财格忌：印比运（帮身破格）

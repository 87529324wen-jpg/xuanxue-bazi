## 二、核心心法（从滴天髓提取）

### 1. 衰旺不是简单数数

**错误做法**：数天干地支有几个帮身、几个克泄耗，算比例。
**正确做法**：

滴天髓原文："得时俱为旺论，失令便作衰看，虽是至理，亦死法也。"

关键原则：
- **月令权重50%以上**，但不是唯一标准
- "日干不论月令休囚，只要四柱有根，便能受财官食神"
- **通根比天干帮身更重要**："天干得一比肩，不如地支得一余气墓库"
- "盖比肩如朋友之相扶，通根如家室之可托"

**物极必反（颠倒之理）**：
- 木太旺者似金，喜火炼（旺到极点，需要克制而非泄）
- 木旺极者似火，喜水克（更极端的旺，需要更强的克制）
- 木太衰者似水，宜金生（衰到极点，需要生扶）
- 木衰极者似土，宜火生（更极端的衰，需要更强的生扶）

**实战意义**：不要看到身旺就一定用财官泄耗，要看旺到什么程度。太旺和旺极的处理方式不同。

### 2. 阴阳干本质区别

滴天髓："五阳从气不从势，五阴从势无情义。"

- **阳干（甲丙戊庚壬）**：刚健，不畏财煞，有恻隐之心，处世不苟且
- **阴干（乙丁己辛癸）**：柔顺，见势忘义，有鄙吝之心，处世多骄谄

**实战意义**：
- 阳干不容易从格（从财/从杀），因为"从气不从势"
- 阴干容易从格，因为"从势无情义"
- 阳干日主即使身弱，只要有一点根就不从
- 阴干日主身弱无根时，更容易从

### 3. 十干性情（不是比喻，是本质）

滴天髓对每个天干都有四句口诀，核心要点：

**甲木参天，脱胎要火**：
- 甲是纯阳之木，体固参天
- 春不容金（旺木欺衰金，木坚金缺）
- 秋不容土（秋土虚薄，被下攻之木倾陷）
- 火炽乘龙（火太多要坐辰，辰为水库能制火）
- 水宕骑虎（水太多要坐寅，寅能纳水气）

**乙木虽柔，刲羊解牛**：
- 乙是甲之质，承甲之生气
- 坐丑未能制柔土（刲羊解牛）
- 怀丁抱丙，跨凤乘猴（有丙丁则不畏申酉金）
- 藤萝系甲，可春可秋（甲寅多见，弟从兄义）

**丙火猛烈，欺霜侮雪**：
- 纯阳之火，不畏秋冬
- 能煅庚金（克强暴）
- 逢辛反怯（合柔顺，丙辛合水）
- 土众成慈（见戊己多则不凌下）
- 水猖显节（遇壬癸旺则不援上）
- 虎马犬乡，甲来成灭（寅午戌全再见甲，火太旺反焚）

**丁火柔中，内性昭融**：
- 阴火，较丙则柔中
- 抱乙而孝（明使辛金不伤乙木）
- 合壬而忠（暗使戊土不伤壬水）
- 旺而不烈，衰而不穷
- 有嫡母（甲木），可秋可冬

**戊土固重，既中且正**：
- 阳土，高燥
- 见丙火而焦坼（丙火太旺会烤焦戊土）
- 需己土（卑湿之体，能收元阳之气）

**己土卑湿，中正蓄藏**：
- 阴土，卑湿
- 能纳水气，能养金藏

**庚金带煞，刚健为最**：
- 阳金，刚健
- 最喜丁火锻炼（丁火柔中，能制庚金而不伤）
- 忌丙火（猛烈，丙庚相战）

**辛金软弱，温润而清**：
- 阴金，柔软
- 畏土之叠（土多金埋）
- 乐水之盈（水多金清）

**壬水通河，能泄金气**：
- 阳水，通河之水
- 刚中之德，周流不滞
- 癸水阴柔，逢丙火而涸乾

**癸水至弱，达于天津**：
- 阴水，至弱
- 随龙而运，升天则为云雨

## 三、格局方法论（从子平真诠提取）

### 用神五法

**1. 扶抑**：日元强者抑之，弱者扶之
- 身旺 → 用官杀/财星/食伤克泄耗
- 身弱 → 用印星/比劫生扶

**2. 病药**：找出命局的"病"，用"药"解
- 以扶为喜 → 伤其扶者为病 → 去病为药
- 以抑为喜 → 伤其抑者为病 → 去病为药
- 例：身弱用印，财星坏印 → 财是病 → 比劫制财是药

**3. 调候**：气候太寒太燥时，调和为急
- 金水生于冬令 → 寒，需丙丁火调候
- 木火生于夏令 → 燥，需壬癸水调候
- **调候优先于扶抑**（穷通宝鉴的核心）

**4. 专旺**：四柱气势偏于一方，不可逆
- 从格（从财/从杀/从儿/从势）
- 化格（天干五合化气）
- 一方专旺（曲直/炎上/稼穑/从革/润下）

**5. 通关**：两神对峙，强弱均平
- 金木对峙 → 用水通关
- 水火对峙 → 用木通关
- 木土对峙 → 用火通关

### 顺用逆用法则

**四吉神（财官印食）→ 顺用**：
- 财喜食神相生，生官护财
- 官喜透财相生，生印护官
- 印喜官煞相生，劫财护印
- 食喜身旺相生，生财护食

**四凶神（煞伤劫刃）→ 逆用**：
- 七煞喜食神制伏，忌财印资扶
- 伤官喜佩印制伏，生财化伤
- 阳刃喜官煞制伏，忌官煞俱无
- 月劫喜透官制伏，利用财透食化劫

### 格局成败救应

- 成格：用神得到辅助（如正官格透财生官）
- 败格：用神被破坏（如正官格见伤官）
- 救应：败格被解救（如伤官见印制伏）

## 四、神煞系统（从三命通会提取）

### 必查神煞（按重要性排序）

**第一梯队（必须查）**：
1. 天乙贵人（以日干查，两种查法都查）
2. 华盖（以年支查）
3. 驿马（以年支查）
4. 将星（以年支查）
5. 桃花（以年支查）
6. 空亡（以日柱旬查）
7. 文昌（以日干查）

**第二梯队（重要参考）**：
8. 羊刃（以日干查，阳干才有）
9. 禄神（以日干查）
10. 天德/月德（以月支查）
11. 魁罡（庚辰/庚戌/壬辰/戊戌日）

**第三梯队（辅助判断）**：
12. 劫煞/亡神/天罗地网/十恶大败等

### 神煞交互关系

- 将星×华盖 = 技术型权威（有领导力但爱独处）
- 天乙贵人×华盖 = 贵人来自同类（聪明有深度的人）
- 驿马×桃花 = 感情变动大（容易异地恋/异国恋）
- 空亡×华盖 = 精神世界极深，但现实感弱
- 羊刃×七杀 = 权力型命格（有制则贵，无制则凶）
- 文昌×华盖 = 学术型命格

## 五、实战断命流程（整合四书）

### 第1步：排盘
- 四柱八字（渊海子平的藏干表）
- 十神标注（渊海子平的十神定义）
- 纳音五行

### 第2步：日主强弱（滴天髓心法）
1. 先看月令（权重50%+）
2. 看通根（地支比天干重要）
3. 看天干帮身/克泄耗
4. 综合判断：身旺/身弱/中和/太旺/旺极/太衰/衰极

### 第3步：取用神（子平真诠五法）
1. 先看是否需要调候（穷通宝鉴）
2. 再看格局（月令透出什么十神）
3. 判断顺用还是逆用
4. 确定用神/喜神/忌神

### 第4步：查神煞（三命通会）
1. 天乙贵人（日干+年干两种查法）
2. 华盖/驿马/将星/桃花/空亡
3. 神煞交互关系

### 第5步：十神组合分析（子平真诠+三命通会）
- 贪财坏印 / 印化杀 / 官印相生
- 官杀混杂 / 食神制杀 / 伤官见官
- 劫财夺财 / 食伤生财 / 财官相生
- 每个组合引用经典原文

### 第6步：大运流年（三命通会+子平真诠）
1. 大运与原局的关系（成格/变格/败格）
2. 流年太岁检查（渊海子平的太岁理论）
3. 流年与原局天干地支的作用
4. 逐月分析

### 第7步：输出
- 性格（滴天髓的十干性情）
- 事业（格局高低+用神喜忌）
- 感情（夫妻宫+正官/正财位置）
- 健康（三命通会的疾病诊断）
- 外貌（三命通会的性情相貌）
- 时间窗口（大运流年）

## 1. 十干坐支兼得月时及行运吉凶（《论十干坐支兼得月时及行运吉凶》及甲乙、丙丁、戊己、庚辛、壬癸五组）

- 原文金句：「消息盈虚玄妙理，要精休旺说行藏。」「喜行身旺鬼衰运，忌行身衰鬼旺乡。」
- 核心规则详解：
  - **先看坐支，再看月时** → 日干坐下是根脚，月令是气候，时上是归宿，三者不分先后就会把“坐财官”与“财官有气”混为一谈。
  - **甲乙木取金水成材** → 甲木偏阳，最喜秋冬金水；乙木偏阴，喜春令荣发，也喜亥卯未、申子辰成局。木若无金，只成繁枝；木若见金火得宜，方能成器。
  - **丙丁火最怕偏枯** → 丙丁春夏得令可成，秋冬必须借金水调剂。火太燥则孤，水太盛则灭，故原文反复强调“水火既济”“相停为美”。
  - **戊己土喜木疏与水润** → 戊土如城堤，宜火扶、木疏；己土如田园，宜刑冲耕凿、木来成用。土重无木则顽，木重土轻则崩。
  - **庚辛金分强弱与炉冶** → 春金得财，夏金喜官杀配印，秋金过旺须火炼，冬金寒湿须火温。金无火则不成器，火无根则反熔其身。
  - **壬癸水最看火土节制** → 春水喜财，夏水喜官印，秋水要火土，冬水多取从旺、飞禄、比劫成势。水多无堤，反成漂流之命。
  - **官杀有制才可论贵** → 五组干象都反复出现“官杀混杂”“有煞无制”“制太过”“制不及”四种病。不是见官即贵，见杀即凶，而是看能否化煞、制煞、任煞。
  - **时上为终局** → 同一月令，时上能把官引至死绝，也能把伤官引回旺地，所以晚景、子息、结局，多看时柱。
- 格局成败条件：
  - 成格条件：日干得地，所取财官印食在月令或时上有气，且岁运顺着“扶起用神、制服病神”的方向走。
  - 破格条件：官杀混杂无制、伤官见官、比劫夺财、印被财坏、冲提纲而无救。
  - 救应方法：身弱用印比，官杀重用食制或印化，财重身弱靠比劫扶身，火燥水泛则回到调候。
- 经典命例拆解：
  - **甲午冬生遇子时**：原文明说“甲午冬生遇子时，格全印绶”，关键不在午为死地，而在亥子印旺反救其木，是“死中得生”的范式。
  - **丙申戊戌甲午乙亥**：原文举作“甲趋乾又地天交泰”，说明甲木在火土财官与亥印之间取得平衡，贵在气机流通，不在单纯身旺。
  - **辛酉、癸未、丙子、癸巳**：夏后十日用金水有气而贵，说明节令转换点比单看月份更细。
- 例外与易错点：
  - 最易错的是把“坐财官”直接当成“财官有用”。坐下虽有财官，若月令休囚、岁运冲坏，反成虚名虚利。
  - 第二个大错是把“有煞”一概视为凶。原文最重“身煞轻重”和“制化得中”，并不反对用煞。
  - 第三个易错点是阴阳干同五行不同性。甲乙、丙丁、庚辛、壬癸都不能混断。
- 与前书关联：
  - 与《子平真诠》“格局先看月令”完全一致，但《三命通会》把“时上归宿”和“岁运成败”说得更细。
  - 与《滴天髓》“旺者宜泄、弱者宜扶”“偏枯为病”互证。
  - 与 `knowledge/十神关系速查.md` 的官杀混杂、食神制杀、贪财坏印诸条直接对接。

## 2. 十二月支得日干吉凶（《论十二月支得日干吉凶》全章）

- 原文金句：「月令当时为最。」「旺处生来墓库绝，墓库发来生旺脱。」
- 核心规则详解：
  - **子月总法** → 寒水专权，木火欲用，金水太过则漂，土来须分“寒湿之土”还是“堤岸之土”。
  - **丑月总法** → 杂气最重，财官印同宫，第一要诀不是五行多少，而是“谁透、谁当令、谁能冲库而出”。
  - **寅卯月总法** → 木旺当令，木日不怕再旺，怕的是无金成材；金日不怕官杀，怕的是无火制寒。
  - **辰巳午未总法** → 火土递进，最忌一味燥烈。木火土旺时，看是否有水金润泽；水金过弱时，看是否可借比印保身。
  - **申酉月总法** → 金旺肃杀，木火日主要靠印比或调候；金日则多成器象，但过旺反成顽刚。
  - **戌亥月总法** → 戌是火库土燥，亥是初冬水旺。戌月看藏火能不能发，亥月看寒木寒金能不能得火。
  - **月支决定“本气”，节气决定“轻重”** → 同是丑月、戌月、未月，前中后气差别极大，原文多次提醒“分节气浅深轻重言之”。
- 格局成败条件：
  - 成格条件：月令本气与所取用神同路，或虽不同路但有时上、三合、干透引化。
  - 破格条件：月令被冲提、当令之神被合去、库中财官被压伏而无冲起。
  - 救应方法：杂气靠透干、墓库靠冲开、寒湿靠火、燥烈靠水。
- 经典命例拆解：
  - **丑月用杂气**：甲日丑月可同时见辛官、己财、癸印，贵贱全看透出与节气，典型是“杂气不怕多，怕混而无主”。
  - **酉月官禄**：甲酉、乙申若官透而日主有根，多成正官清格；若丁伤官得势，则贵气易破。
- 例外与易错点：
  - “某月一定喜某神”是最粗糙的用法。本章讲的是“月令为纲”，不是“月令包办一切”。
  - 墓库月绝不能只按“土月”看，必须同时看藏干与透出。
- 与前书关联：
  - 与《穷通宝鉴》按月调候完全并轨，可把本章当作《穷通》的格局补注。
  - 与《子平真诠》“月令取格”互证。

## 3. 五行时地分野吉凶（《论五行时地分野吉凶》与《论木》《论火》《论土》《论金》《论水》）

- 原文金句：「东南主生，西北主杀。」「得其方则发，失其方则败。」
- 核心规则详解：
  - **木看东春与金斧** → 木得东方生气则秀，见金而有水火调剂则成器，纯木无制反贫。
  - **火看南方与水济** → 火得南方则明，若无水济则烈；有水过多又灭，关键是“既济”。
  - **土看四季与木疏** → 土要木疏、水润、火温，四季土若再逢重土，多成顽滞。
  - **金看西方与火炼** → 金旺不怕财，怕顽冷；得火方贵，得水方清，火水都过则折。
  - **水看北方与土堤** → 水旺须堤，水弱须源，水土配则贵，水火济则显。
- 格局成败条件：
  - 成在“顺方得地”；败在“逆方失气”。
  - 岁运若把命局送到本气旺方，常有骤发；送到死绝之方，原局再好也易败。
- 经典命例拆解：
  - 卷中大量以“某干遇某方运”断贵贱，核心不是神煞，而是五行在地分野上的得失。
- 例外与易错点：
  - “喜东喜西”不是固定方向学，而是五行随人而变。甲木喜西北，不代表乙木也必同喜。
- 与前书关联：
  - 可与《滴天髓》“旺相休囚死”配合使用，地分野就是大运方位化的旺衰法。

## 4. 五行分体细论（《论甲乙》《论丙丁》《论戊己》《论庚辛》《论壬癸》）

- 原文金句：「甲乙冬生木本枯，若逢金土反宜乎。」「丙丁冬月用当垣，从化都宜不带根。」
- 核心规则详解：
  - **甲乙**：春木太旺宜金；夏木喜水金；秋木喜火通关；冬木喜金土立格。甲重栋梁，乙重成材。
  - **丙丁**：春火怕水多；夏火若成炎上、倒冲、飞晶等格可贵；秋火以财论但须防金水太盛；冬火从化、制杀、印助三路并行。
  - **戊己**：春土官杀旺，宜火印或木官成器；夏土土焦，必须金水；秋土用金水但怕木火乱局；冬土多取财化与从财。
  - **庚辛**：春金得财官，怕比劫与水泛；夏金要水制火；秋金忌过刚，需火炼；冬金喜火土温养。
  - **壬癸**：春水喜财、木局与比助；夏水喜官印与从财；秋水要火土；冬水从旺、飞禄、合禄最灵。
- 格局成败条件：
  - 成格关键是“季节 + 本体 + 取用神”三项一致。
  - 破格关键是“同类太过”或“病药同败”。
- 经典命例拆解：
  - **甲生八月**：官得时而怕卯丁破，说明旺官也怕伤官挑破。
  - **丁亥月遇寅时**：丁壬化木成格，见丁分官、见辰墓官，皆破化。
  - **戊日庚申时**：秋冬水旺而合禄成名，正是“土以金水为功”的典型。
- 例外与易错点：
  - 同一干在不同季节，取用几乎反转，不能背固定口诀断全局。
- 与前书关联：
  - 与《穷通宝鉴》日主分月法高度一致，但此处更强调“格”“运”“变局”。

## 5. 纳音数理取象群（《甲坤黄数五》至《长鬼为殃》）

- 原文金句：「火生土旺，以下专重纳音取用。」「福欲乘之，鬼欲制之，然后可观。」
- 核心规则详解：
  - **甲系数理**：甲人贵真土，喜戊癸、子巳、辰戌丑未成实；最忌寅重逢火、木发伏火，主破业夭横。
  - **乙系数理**：乙以丙辛、巳酉丑成真金为上；火盛无水则“鬼作截福”，即福气被火毁。
  - **丙系数理**：丙喜甲己、己丑、申子辰与金水清局；子位见土为“中央夺福”，寅位逢鬼为“截根无荫”。
  - **丁系数理**：丁喜戊癸与申子辰清局；水旺而无土最清，土长生太重反成“破相孤独”。
  - **戊己系数理**：戊喜曲直真火与申子水源；己喜乙庚、寅午戌火土清局。两者都怕甲木伤土太过。
  - **庚辛系数理**：庚喜戊癸火土与丑未二吉，辛喜丁壬、亥未、寅戌火木生扶。金得火为贵，得水无木则迟滞。
  - **壬癸系数理**：壬喜丙辛、巳酉、卯亥成真木数；癸喜甲己、未寅、亥卯成真火数。两者都怕干鬼重犯。
- 格局成败条件：
  - 成格条件：纳音本数得“子母相生、真数相逢、墓库成实”。
  - 破格条件：干鬼重临、火过旺伤金水、木过旺焚土、无救之鬼临生处。
  - 救应方法：用对应真数、墓库、福神还元，或者以水制火、以火制金、以木疏土。
- 经典命例拆解：
  - 原文此组多是口诀象法，不给完整四柱，但逻辑很清楚：凡“真数 + 成实地 + 清气”者贵，凡“见鬼无救 + 发伏太过”者败。
- 例外与易错点：
  - 这部分绝不能跳过正五行、月令、岁运，纳音只是补象，不是夺权。
  - “见某字即贵”是误读，原文真正看的是成数、成实、还元。
- 与前书关联：
  - 与 `knowledge/三命通会_批1.md` 的纳音部分可无缝衔接，批1重器象，批2重数理成败。

## 6. 十神立名总纲（《论古人立印食官财名义》）

- 原文金句：「学者明于人事，斯可以言造化矣。」「五行展转生克，皆子为父母复仇之义。」
- 核心规则详解：
  - **印绶** → 生我者，像父母、印信、护身之物。
  - **食神** → 我生者，像子孙、福禄、致养之气。
  - **官杀** → 克我者，像法度、压力、拘束与祸害。
  - **妻财** → 我克者，像资产、资源、妻室与掌控对象。
  - **同我与异我** → 比劫不是单纯“同类”，而是争财、分力、扶身、夺禄的双刃结构。
  - **十神关系来自人事类比** → 所以后面所有格局，最终都落到“谁生谁、谁克谁、谁护谁、谁夺谁”。
- 格局成败条件：
  - 成在理解十神是关系，不是标签。
  - 败在把十神口诀硬套，不看阴阳、位置、得令与否。
- 经典命例拆解：
  - 本章无单独八字例，但它是卷五、卷六全部格局的解释钥匙。
- 例外与易错点：
  - “偏官一定凶、伤官一定坏”都不对。卷五后文正是在说如何把凶神化成权柄。
- 与前书关联：
  - 与 `knowledge/十神关系速查.md` 可直接互参，是其理论母体。

## 7. 正官系统（《论正官》及《天福贵人》至《三台拱帝座》）

- 原文金句：「正官为六格之首，止许一位，多则不宜。」「正官格只喜向禄临财。」
- 核心规则详解：
  - **正官取月令为先** → 先看月令官星，再看透干、根气、印财扶助。
  - **官要纯，不要杂** → 正官最忌七杀混入、伤官挑破、比劫分福、贪合忘官。
  - **身弱可借印，身旺可借财** → 印为护官之本，财为生官之源。
  - **时上能成也能败** → 时上官星得财印可晚发；时上引官入死绝、反引伤官归旺地，则早有官而终败。
  - **相关短章要点**：
    - 《天福贵人》：官坐禄地，主科名与翰苑之贵。
    - 《天官贵人》：官坐天乙，贵上加贵，但怕反伤冲克。
    - 《天元坐禄》《岁德正官》《时上正官》：官若坐禄、坐年、坐时，各有“祖荫早发”“晚发贤子”“本位执权”的差异。
    - 《官印禄煞俱全》《禄马官印》《官印禄库》：不是单看官，而是官印禄库成套出现。
    - 《相刑遇贵》《三合遇贵》《月时逢贵》：刑冲未必纯凶，若恰成官局、贵局，反可发权。
    - 《金木间隔》《水火既济》《金火相成》：说明正官格也看调候与五行成器，不只是官星本身。
    - 《官星六合》《真官催官》《官煞会墓》：官有时靠暗合、虚夹、墓中发用而起。
- 格局成败条件：
  - 成格条件：官纯、身健、印财有情、行运不伤官。
  - 破格条件：官杀混、伤官见官、刑冲提纲、官星入墓无救。
  - 救应方法：印护、财生、合去混杀、食制偏煞。
- 经典命例拆解：
  - **甲生酉月见辛酉**：支藏干透，若日主有根、柱无丁伤，是标准清官格。
  - **辛未辛卯庚戌壬午**：时上正官得局，晚发而成学士，说明时官不轻。
  - **丙午己亥戊子壬子（范仲淹命例）**：交互官星、水火既济，属于官格中夹特殊成象。
- 例外与易错点：
  - 官多不是贵，官多身弱常成病；官轻印重也不一定吉，可能虚名。
  - “官见食必败”并非绝对，月令隐禄见食，原文反称三奇。
- 与前书关联：
  - 与《子平真诠》“官星佩印”“逢官看财”完全同法。
  - 与《滴天髓》“官为约身之物”互证。

## 8. 偏官七杀系统（《论偏官》及《天元坐煞》至《官煞去留并官鬼互变杂论》）

- 原文金句：「有制谓之偏官，无制谓之七煞。」「煞无刃不显，刃无煞不威。」
- 核心规则详解：
  - **七杀不是绝凶，关键在制化** → 印化、食制、刃驾、比助，都是把杀从“祸”转成“权”的手段。
  - **身强煞浅宜用煞，身弱煞重先救身**。
  - **财能生杀，也能助祸** → 身强喜财引煞成权，身弱见财则“盗气助鬼”。
  - **相关短章要点**：
    - 《天元坐煞》《时上一位贵》《年上七杀》：坐杀、时杀、年杀，轻重各异，时杀成格最容易显权。
    - 《弃命从煞》：身弱无根、煞旺成局，反以从杀为上。
    - 《时煞归库》：杀入库而可发，须待岁运冲开。
    - 《官藏煞显》《官煞混杂》：官杀同见最怕无去留。
    - 《会煞化印》：卷五最重要的“杀印相生”总论。
    - 《专煞无制》《专禄要制》：煞太专亦要控制，不然刚而无德。
- 格局成败条件：
  - 成格条件：身杀两停，或杀旺而印化、食制、刃驾得中。
  - 破格条件：身弱杀强、财多助鬼、官杀混杂、制杀太过。
  - 救应方法：印绶为第一救，食神为第二救，阳刃比肩为第三救。
- 经典命例拆解：
  - **甲寅、乙卯等身旺木日见庚申辛酉**：若成印局或火制，便是“化鬼为官”。
  - **戊寅甲寅戊寅甲寅**：四柱纯煞，靠长生火印化煞成贵，正是卷六“极返格”的前导案例。
- 例外与易错点：
  - 杀格最怕“只会说食神制杀”，却忘了身弱先扶身。
  - 杀印相生和从杀格不是一回事。前者日主仍要承受并转化，后者是弃命随势。
- 与前书关联：
  - 与《滴天髓》“儿能救母”“刃煞两停威权万里”相合。
  - 与 `knowledge/十神关系速查.md` 的“杀印相生”“羊刃驾杀”呼应。

## 9. 财星系统（《论正财》《论偏财》及相关短章）

- 原文金句：「富而且贵，定因财旺生官。」「偏财元是众人财，最忌干支兄弟来。」
- 核心规则详解：
  - **正财重稳定、库藏、妻室、正产**；**偏财重外财、流动、众财、经营**。
  - **身旺任财** → 财能生官、能立家、能起业。
  - **身弱财多** → 富屋穷人，徒见财源，反被财累。
  - **相关短章要点**：
    - 《岁带正马》《时带正马》：年财多祖业，时财多自立与外来财。
    - 《财旺生官》：原局官不必显透，只要财旺有官气即可因富致贵。
    - 《财临库墓》《日坐天财》《天元坐财》：库财最怕身弱，开库则发。
    - 《时上偏财》：最利白手起家，但极怕兄弟分夺。
    - 《专财》《弃命从财》：无根日主满盘财局，可从财，不可再求比印。
    - 《偏正财合论》：真财、天财、鬼财、库财、类财煞等，都是财星细分法。
- 格局成败条件：
  - 成格条件：身旺、财有源、财不被劫、官印可承。
  - 破格条件：比劫并出、官杀太旺夺财、财多身弱、财被冲库而无根。
  - 救应方法：身弱用比印，财多可借食伤泄秀或官星导气。
- 经典命例拆解：
  - **己巳、癸酉、庚辰、甲申**：日坐天财，主财入库而可聚。
  - **甲辰、戊辰、丙申、丙申**：专财格，财神专气，宜行官旺财旺运。
- 例外与易错点：
  - “偏财比正财大”不是贵贱规则，而是性质不同。偏财厚而不稳，正财稳而不猛。
  - 从财格最怕一点根气回头救身，一救反破格。
- 与前书关联：
  - 与《子平真诠》“财旺生官”“财轻分夺”一致。
  - 与《渊海子平》大量商贾、外财命例可互证。

## 10. 印绶与倒食系统（《论印绶》《鬼化为印》《阳刃化印》《时逢生印》《胞胎逢印绶》《弃印就财》《论倒食》）

- 原文金句：「印赖官生。」「煞能生印，畏行财乡。」
- 核心规则详解：
  - **印是护身本、读书本、祖荫本**。
  - **印以官生最吉，以财破最忌**。
  - **印多身弱为福，印多身旺则滞**。
  - **相关短章要点**：
    - 《鬼化为印》：鬼被木土化去而生印，是“险中生贵”的极高格。
    - 《阳刃化印》：刃遇火印，不再纯以刃论，关键看财水是否来破。
    - 《时逢生印》《胞胎逢印绶》：印落时上、胎元，多主晚福与祖荫。
    - 《弃印就财》：偏印居年时而财旺，身旺可舍印取财。
    - 《论倒食》：偏印夺食，最伤福禄、子息、财路；有制合则不凶，反可成贵。
- 格局成败条件：
  - 成格条件：印清、官生、身不太旺、财不乱破。
  - 破格条件：财破印、梟夺食、印多无泄、印入墓绝。
  - 救应方法：比肩护印、官星生印、食神制煞后再转印。
- 经典命例拆解：
  - **乙丑、癸未、丙子、乙未**：鬼化为印，既济格中鬼转印，为显贵范式。
  - **戊寅、戊午、戊午、戊午**：阳刃化印，寅午会火，刃气转印气，大贵。
- 例外与易错点：
  - 倒食不是一见即凶，关键看是否有辰戌丑未土制、天干合去。
  - 印多不一定文贵，水泛木浮、土重金埋、火燥土烈都属假印。
- 与前书关联：
  - 与《子平真诠》“印绶喜官”“贪财坏印”一致。
  - 与 `knowledge/十神关系速查.md` 的“枭神夺食”“印化杀”互证。

## 11. 杂气、伤官、食神、阳刃、建禄总断（《论杂气》《附论墓运》《论伤官》《论食神》《飞天禄马》《倒冲禄》《天冲食禄》《福星贵人》《食神同窠》《食神带合》《专食合禄》《红鸾天印》《墨池涌泉》《专印合禄》《论阳刃》《论建禄》）

- 原文金句：「杂气财官身旺有冲而发。」「伤官见官，祸患百端。」「食神制煞，吉非常。」「建禄生嫌门户中。」
- 核心规则详解：
  - **杂气**：辰戌丑未最怕压伏，最喜透出与冲刑。库不开，财官不发；开得太过，秀气也散。
  - **伤官**：真正高格是伤尽生财，不见官；若官露而再见伤，祸机大开。火土伤官最怕官，金水木伤官反可见官。
  - **食神**：食神是福神，能制煞、能生财、能养命；最忌偏印夺食。
  - **飞冲暗合诸格**：
    - 《飞天禄马》：庚子、壬子、辛亥、癸亥冬月以子亥多冲出官禄，格高而险。
    - 《倒冲禄》：丙午、丁巳等以午巳多冲出对宫官星，贵而不稳。
    - 《天冲食禄》《专食合禄》《专印合禄》：都在讲“食神、印绶、暗合官禄”如何从虚处取贵。
    - 《福星贵人》《食神同窠》《红鸾天印》《墨池涌泉》：属于食印同来、文名清贵的附格。
  - **阳刃**：刃本身不判贵贱，关键看见杀、见印、见财、见冲。刃无杀不显，刃逢财多凶。
  - **建禄**：月令比劫旺而财官弱，多主难守祖业；若年时透财官食伤，反可由旺身任财官而发。
- 格局成败条件：
  - 成格条件：
    - 杂气要“透或冲”。
    - 伤官要“伤尽、财印得所”。
    - 食神要“身旺、无枭、见财或制杀”。
    - 阳刃要“有杀印制伏”。
    - 建禄要“透财官食伤，不可空旺”。
  - 破格条件：
    - 杂气被戊己压库。
    - 伤官见官又无印财救。
    - 食神见枭。
    - 刃逢财旺、岁运冲刃。
    - 建禄再逢比劫而无财官。
  - 救应方法：伤官配印、食神制杀、刃印相生、建禄取财官食伤泄秀。
- 经典命例拆解：
  - **甲子甲子遥巳禄**：子多冲起巳中官禄，是卷六飞冲取贵的先声。
  - **戊日庚申时专食合禄**：食神旺而暗合官星，是“食神不只生财，也能生官”的典型。
  - **丙申甲午丙戌乙未、庚寅戊寅丙戌甲午**：炎上、食神、伤官、专旺互见，说明不能只用单一格名套断。
- 例外与易错点：
  - 伤官格不是都怕官。金水伤官、木火通明、伤官配印，都可见官。
  - 建禄不是贫格，而是“空旺则贫、有泄有任则贵”。
- 与前书关联：
  - 与《子平真诠》“建禄月劫用财官”“伤官配印”“食神生财”完全打通。
  - 与 `knowledge/十神关系速查.md` 的伤官见官、食神制杀、印绶太重、比肩重重相互印证。

## 12. 飞冲暗合取贵系统（卷六前段：《井栏斜义》至《拱禄拱贵》）

- 核心规则详解：
  - **井栏斜义**：庚日申子辰全局、三庚透出，用水局冲起对宫官星，贵在局纯无火。
  - **壬骑龙背**：壬辰、壬寅等借辰冲戌、寅合辰，暗取财官。辰多主贵，寅多主富。
  - **子遥巳禄、丑遥巳禄、刑合得禄、冲合禄马**：共同逻辑都是“原局不见官，借冲、合、刑、遥把官禄从虚处引出来”。

  - **日禄归时、拱禄拱贵**：归禄比飞冲更稳，拱禄比实见更妙，但怕填实、怕冲空。
- 格局成败条件：
  - 成格条件：原局无明官而暗路清，且有一字合住贵气，不使飞散。
  - 破格条件：官星明露、合神被绊、冲位被填实、月令另成他格。

- 经典命例拆解：
  - **庚申庚子庚辰**：井栏斜义，水局纯、火不露则贵清。
  - **壬辰甲辰壬寅庚子**：壬骑龙背，辰寅并见，贵而能发。
  - **庚子/壬子、辛亥/癸亥**：飞天禄马与本组逻辑相通，都是“虚邀官禄”。
- 例外与易错点：
  - 这类格最怕“半懂不懂乱取暗拱”。只要原局已有财官成格，就应回到财官论，不再强取飞冲。
- 与前书关联：

## 14. 专旺、清浊、消息流转系统（卷六中后段：《曲直》至《五行拘聚》）

- 经典命例拆解：
  - **曲直格**：甲乙亥卯未全，见土金成材则贵；纯木无亥无金土，只是木旺不秀。
  - **炎上格**：丙丁寅午戌全，不见水为前提，若土晦火光则由贵转病。

  - 专旺格最忌“见一点就说成格”。三合、三会、透干、节令、岁运缺一不可。
  - 背禄并不必贫，关键看是否仍有财路承接。
- 与前书关联：
  - 与《滴天髓》“从得真者只论从、从神有吉有凶”直接呼应。

## 15. 合化与象法八门（《合化成局》及十条、《返象》《照象》《鬼象》《伏象》《属象》《类象》《从象》《化象》）

- 原文金句：「干合而不成局则不化，但以本干支取祸福断之。」「类属从化，格判旺衰。」
- 核心规则详解：
  - **合化成局总法**：甲己土、乙庚金、丙辛水、丁壬木、戊癸火，不是“相合就化”，而是要得月气、得旺地、得无伤。
  - **十条合化歌**：是在总法之上，分别补充五合成化后的喜忌，如甲己化土喜辛戊，乙庚化金喜巳甲，丙辛化水喜戊乙等。
  - **返象**：化局失地、化中返本、绝处逢生而反复不定，多主迁动、事业反覆。
  - **照象**：火土高明，上下相照，多主显达；见水则减力。
  - **鬼象**：身鬼衰旺相敌，轻则悍，重则残；制得中反主武权。
  - **伏象**：财官印煞伏于支中，不透不显，须待援引；无援则才而不达。
  - **属象、类象、从象、化象**：分别处理“归方”“成局”“随势”“真化”四种不同逻辑，最怕互相误判。
- 格局成败条件：
  - 成格条件：合化得时得位，或象法与月令、岁运同向。
  - 破格条件：化而不化、返而无依、鬼旺身衰、伏而无援。
  - 救应方法：以月令裁定，以岁运验证，不可只凭天干一合。
- 经典命例拆解：
  - **甲寅己巳甲戌壬申**：甲己化土得申长生，是化象得位。
  - **辛未壬辰丙午癸巳**：丙辛化水失地，属返象，主一生反复。
  - **戊戌戊午丙午戊戌**：照象，火土高明，四柱无伤则显。
- 例外与易错点：
  - 最大误区是把“从”“化”“类”“属”混成一类。四者判法不同：从重在日主无根，化重在干合得令，类重在局成一家，属重在入其方位。
- 与前书关联：
  - 与《断命流程指南》附录C“合化条件与争合妒合”可直接合并使用。

## 16. 正贵总图（卷六后段：《天地德合》至《十干十二年生大贵人例》）

- 原文金句：「好年不如月，好月不如时。」「年月日時多不足凭，天生大贵人必有冥数气运以主之。」
- 核心规则详解：
  - **德合类**：天地德合、君臣庆会、六合双鸳、贵气冲和，重在干支两层都和而不杂。
  - **纯一类**：一气为根、两干不杂、三合聚集、五行俱足、五行一旬、四柱顺布、干支双连、天干连珠、地支连茹，重在“纯”“顺”“通”。
  - **辅贵类**：禄马交驰、赶禄栏马、集福发福、禄库逢财、福会相迎、夹贵夹禄、福神相还，强调贵神互相递生。
  - **包拱类**：引从包承、拱揖阙门、子午双包、四般生一、虚一待用，都是“前后包护、中宫得势”的思路。
  - **象征类**：龙吟虎啸、龙跃天门、虎卧龙阁、云行雨施、清肃宪台、风云庆会、鞍马坐贵、锦衣特赐等，是对高层清贵、武权、台谏、恩宠的象法命名。
  - **方法论收束**：最后《十干十二年生大贵人例》明确提醒，同八字不同命运极多，格局只是骨架，家世、修为、气运、享用深浅都会改变结局。
- 格局成败条件：
  - 成格条件：纯、顺、清、连、递互有情，且不被空亡刑破。
  - 破格条件：一字驳杂、贵神被冲、纳音死绝又逢破、福神不还而凶煞先行。
  - 救应方法：用月令、运程、福神轻重复核，不迷信格名。
- 经典命例拆解：
  - **明太祖 戊辰壬戌丁丑丁未**：贵人黄枢，四季土全而顺阴阳，属帝王格。
  - **徐阶 癸亥壬戌癸未壬子**：引从包承格，前后包护，本气一贯，故能久居台辅。
  - **陈以勤 辛未戊戌丁卯辛亥**：虚中精实，说明伟器之命未必四位全纯，关键是“上下能包，下路能成”。
- 例外与易错点：
  - 这一组最容易被“格名崇拜”带偏。原文末尾已经明说：同八字者，贵贱寿夭仍不同，故不可执例伤理。
  - 不能离开人事修为谈命。享用过当、修德不足，也会折损格局。
- 与前书关联：
  - 与《滴天髓》“造化因人而显”互应。
  - 与《子平真诠》相比，本卷更像高阶格局备忘录，务必回到月令、十神、用神再落地。

## 卷四至卷六总提要

- 卷四的核心价值，在于把“日干本体、月令时气、方运去向、纳音数理”四层同时铺开。它不是单讲神煞，而是在搭一套“为什么同一日主会千差万别”的细分框架。
- 卷五的核心价值，在于把正官、七杀、财、印、伤官、食神、阳刃、建禄全部拉回十神体系，说明“格局不是名词，而是十神关系的动态平衡”。
- 卷六的核心价值，在于把高阶变格、虚邀暗拱、包护纯一、合化象法集中展示。它最适合做“特殊命式校验手册”，不适合脱离常规格局单独滥用。

## 实战落地法

- 第一层：先用《子平真诠》定月令与格局主轴。
- 第二层：再用卷五十神法判断官杀财印食伤谁是真用、谁是假名。
- 第三层：若原局不走常规，再到卷六查飞冲暗合、专旺、从化、贵格。
- 第四层：若仍难分高下，回卷四的方运、纳音、数理、节气轻重校正。
- 第五层：最后以大运流年验证，不可只见格名便下贵断。

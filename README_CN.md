## 🔧 核心功能

### 1️⃣ 八字排盘

- **输入**：公历/农历出生时间 + 性别
- **输出**：四柱八字 + 纳音 + 生肖 + 十神
- **工具**：sxtwl库（寿星万年历，最权威）

### 2️⃣ 断命分析（7步法）

```
第1步: 排盘
  输入: 出生年月日时(公历/农历) + 性别
  输出: 四柱八字 + 纳音 + 生肖

第2步: 太岁检查
  值太岁/冲太岁/刑太岁/害太岁/破太岁
  严重度排序: 冲太岁 > 值太岁+自刑 > 三刑 > 害 > 破

第3步: 神煞
  天乙贵人/文昌/将星/华盖/驿马/羊刃/桃花/十干禄
  /天德/月德/空亡/魁罡

第4步: 日主强弱
  月令旺相休囚死 + 帮身/克泄耗力量对比
  判断: 身旺/身弱/中和

第5步: 用神喜忌
  身旺 → 克泄耗为用(官杀/财星/食伤)
  身弱 → 生扶为用(印星/比劫)
  调候用神: 穷通宝鉴——按日主天干+月份查用神

第6步: 十神关系分析
  贪财坏印 / 印化杀 / 官印相生
  官杀混杂 / 食神制杀 / 伤官见官
  劫财夺财 / 食伤生财 / 财官相生
  比肩重重 / 印绶太重

第7步: 大运+流年
  大运排列 + 起运年龄
  流年与原局天干地支作用
  大运+流年叠加效应
  逐月分析
```

### 3️⃣ 流年预测

- **太岁检查**：值/冲/刑/害/破
- **流年十二神煞**：死符、官符、病符等
- **逐月分析**：流月与原局关系
- **概率估计**："60-70%失业概率"这种具体数字

### 4️⃣ 合婚配对

**合婚检查清单（按优先级）：**

| 层级 | 检查项 | 含义 |
|------|--------|------|
| 第一层 | 日柱关系 | 最关键，夫妻宫 |
| 第二层 | 年柱关系 | 家庭层面 |
| 第三层 | 月柱关系 | 事业/社会层面 |
| 第四层 | 时柱关系 | 子女/晚年 |
| 第五层 | 五行互补 | 你缺她有✓ |
| 第六层 | 深层关系 | 三合/六冲/伏吟 |

### 5️⃣ 其他术数

- **紫微斗数**：十四主星 + 十二宫 + 四化表
- **六爻卜卦**：起卦 + 装卦 + 纳甲 + 六亲断法
- **面相**：五官 + 十二宫 + 气色 + 骨相
- **梅花易数**：起卦方法 + 体用生克 + 八卦万物类象

## 💡 使用示例

### 用户输入
```
男，1990年农历九月廿八，戌时，看一下2026年
```

### AI分析流程
1. **排盘** → 庚午 丁亥 癸未 壬戌
2. **太岁** → 年支午=值太岁+刑太岁 ★★★★★
3. **神煞** → 将星/华盖/无禄/无天乙 + 交互关系
4. **日主** → 癸水身旺
5. **用神** → 土(官杀)/火(财)/木(食伤)
6. **十神关系** → 贪财坏印/官杀混杂/劫财夺财/食神制杀...
7. **流年** → 丙=正财, 午=偏财, 用神到位但太岁叠加
8. **综合** → 每条结论标注依据和出处

## 🔍 技术要点

### 排盘必须用 sxtwl
```python
import sxtwl

# 公历转农历
day = sxtwl.fromSolar(1990, 11, 14)

# 获取四柱
year_gz = day.getYearGZ(False)  # 年柱
month_gz = day.getMonthGZ()     # 月柱
day_gz = day.getDayGZ()         # 日柱
shi_gz = sxtwl.getShiGz(day_gz.tg, 19)  # 时柱（戌时=19点）
```

### 太岁检查顺序
```
冲太岁 > 值太岁+自刑 > 三刑 > 害太岁 > 破太岁
```

### 十神关系比单个十神重要
- 不是只列"偏财""七杀"这种标签
- 要分析组合：贪财坏印/官杀混杂/劫财夺财/食神制杀/伤官见官/印化杀

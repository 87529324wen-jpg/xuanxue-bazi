## 一、喜神仇神闲神系统（用神判断核心补充）

### 五神关系
以用神为中心，五行各神各有角色：

| 角色 | 定义 | 作用 |
|------|------|------|
| **用神** | 命局最需要的五行 | 核心，决定吉凶 |
| **喜神** | 生助用神的五行 | 用神的帮手 |
| **忌神** | 克制用神的五行 | 最大敌人 |
| **仇神** | 生助忌神的五行 | 忌神的帮手（间接敌人） |
| **闲神** | 与用神忌神都无直接关系的五行 | 无关紧要 |

经典：《滴天髓》任铁樵注：
"如用木，木有餘，以火為喜神，以金為忌神，以水為仇神，以土為閒神；
木不足，以水為喜神，以土為忌神，以金為仇神，以火為閒神。
是以用神必得喜神之佐，閒神之助，則用神有勢，不懷忌神矣。"

### 判断方法
1. 先定用神
2. 生用神者 = 喜神
3. 克用神者 = 忌神
4. 生忌神者 = 仇神
5. 其余 = 闲神

**注意：喜神到位的大运/流年 = 最佳时期；忌神到位 = 最差时期。**

## 二、杂气与月劫

### 杂气（辰戌丑未月的特殊性）
辰戌丑未为四季土，藏干复杂（如辰藏戊乙癸），不是单一五行主事。
经典：《子平真诠》有"論雜氣如何取用"专章。
处理方法：看天干透出什么，就以什么定格。若都不透，以本气为主但力量减弱。

### 月劫（月令与日干同五行但不同阴阳）
如甲日生寅月（寅中甲为比肩），丙日生巳月（巳中丙为比肩）。
月劫 = 月令帮身，但不是禄（禄是天干在地支的临官位）。
经典：《子平真诠》："論建祿月劫"专章。
处理：月劫身旺，需食伤泄秀或财官克耗方能发达。

## 三、偏官格（= 七杀格，不同称呼）

偏官即七杀，同一概念的不同名称。
- 古书常用"偏官"（《子平真诠》目录："論偏官"）
- 现代常用"七杀"
- 两者完全等同，不要混淆

**偏官格成格条件：**
月令藏干透出七杀（偏官），且有制（食神/印）或有化（印）。
经典：《子平真诠》第四十章："論偏官取運"

## 四、伤官配印

格局：伤官旺 + 印绶制伤
含义：伤官太旺（叛逆、不服管）→ 用印绶（学识、修养）来制衡
经典：与"伤官佩印"同义，见《子平真诠》伤官章。
与"伤官伤尽"的区别：
- 伤尽 = 无官星，伤官自由发挥成格
- 配印 = 有印制伤，伤官被驯服成格

## 五、财格佩印

格局：财格 + 印绶同透，但位置不相碍
经典：《子平真诠》："或透印而位置妥貼者，財印不相礙也。"
含义：财能生官，印能护身，两者配合得当则富而有学。
条件：财印不能紧贴（紧贴则财克印 = 贪财坏印）。
最佳位置：年干透印，时干透财（中间隔比劫/官星则不相碍）。

## 六、特殊格局（三命通会卷六）

以下格局出现率低，但在经典中有专章论述，分析时如遇到应能识别：

### 壬骑龙背格
日柱壬辰，辰为龙。见寅多则富，辰多则荣。
经典：《神峰通考》："壬骑龙背，見戌無情，寅多則富，辰多則榮。"
忌：见戌（辰戌冲破格）。

### 子遥巳格
甲子日甲子时，无庚辛申酉，子遥合巳中丙戊为财官。
经典：《三命通会》卷八有专章。

### 丑遥巳格
癸丑日癸丑时，丑遥合巳中丙戊为财官。
经典：《三命通会》卷九有专章。

### 刑合格
癸亥/癸卯/癸酉日见甲寅时，寅刑出巳中戊土为官星，戊癸合。
经典：《神峰通考》卷三有专章。

### 合禄格
戊日庚申时，无丙甲卯寅，申合巳中丙戊为禄。
经典：《三命通会》卷八有专章。

### 日德格
甲寅、丙辰、戊辰、庚辰、壬戌五日为日德。
经典：《子平真诠》提及（列为"无理之格"，但《三命通会》有论述）。

### 拱禄格 / 拱贵格
两柱之间虚拱出禄位或贵人位。
经典：《神峰通考》："拱祿拱貴，填實則凶。"
含义：虚拱出来的东西，填实（大运/流年补上）反而破格。

### 化气格
天干合化成功，化神当令，全局顺从化出之五行。
如甲日己月，甲己合化土，地支辰戌丑未全，天干透戊己 → 化土格。
条件：1. 天干合化成功 2. 化神当令 3. 无争合妒合 4. 全局顺从
化气格有五种：化木、化火、化土、化金、化水。
经典：《三命通会》卷六有"化氣"专论。

## 七、流年十二神煞（太岁临位神煞）

每年太岁带一组临位神煞，用于流年细断。以太岁为起点顺排：

| 序号 | 神煞 | 吉凶 | 含义 |
|------|------|------|------|
| 1 | 太岁 + 剑锋 + 伏尸 | ★★★★★ | 值太岁，主压力、血光 |
| 2 | 太阳 + 天空 | ★★ | 阳气虚浮，凡事不实 |
| 3 | 丧门 + 地丧 | ★★★★ | 丧服、损亲 |
| 4 | 勾绞 + 贯索 | ★★★ | 牵绊、口舌 |
| 5 | 官符 + 五鬼 | ★★★ | 官非、小人 |
| 6 | 死符 + 小耗 | ★★★ | 破财、健康差 |
| 7 | 栏杆 + 大耗 | ★★★ | 阻碍、大破财 |
| 8 | 暴败 + 天厄 | ★★★ | 突然失败、意外 |
| 9 | 飞廉 + 白虎 | ★★★★ | 血光、刑伤 |
| 10 | 卷舌 + 福星 | ★★ | 口舌但有福星解 |
| 11 | 天狗 + 吊客 | ★★★★ | 损伤、丧服 |
| 12 | 病符 | ★★★ | 疾病、晦气 |

经典：《神峰通考》卷四完整列出。
**用法：以生年地支查流年太岁所在位置，确定该年带哪些神煞。**

### 吊客
起法：命后三辰。子生见戌，丑生见亥，依此类推。
经典：《神峰通考》："命前三辰為喪門，命後三辰為吊客。"
吉凶：主丧服哭泣，损远亲。

### 官符
起法：太岁前五辰。
经典：《神峰通考》："命值官符官事多。"
吉凶：主官灾、诉讼。官符落空亡则减轻。

### 病符
起法：太岁后一辰。
吉凶：主疾病、晦气。

### 天空
起法：太岁前一位。
吉凶：主虚浮、不实、计划落空。

### 三丘五墓
起法：春见丑、夏见辰、秋见未、冬见戌为三丘；五墓取对宫。
经典：《神峰通考》："三丘五墓得人愁，爹娘妻子盡不周。"
吉凶：主亲人有损。

### 吞陷煞
起法：以年支查特定组合。
经典：《三命通会》卷三有完整口诀。
吉凶：主凶死、意外。

### 的杀（暗金的煞）
经典：《三命通会》卷三："論暗金的煞"
起法：子午卯酉月见巳、寅申巳亥月见酉、辰戌丑未月见丑。
吉凶：主刑伤、血光。

### 攀鞍
起法：驿马后一位。
吉凶：主出行、变动中的帮助。

## 八、五行生克基础（洪丕谟理论框架）

### 五行相生
木生火、火生土、土生金、金生水、水生木

### 五行相克
木克土、土克水、水克火、火克金、金克木

### 天干关系
- 相合：甲己合土、乙庚合金、丙辛合水、丁壬合木、戊癸合火
- 相冲：甲庚冲、乙辛冲、丙壬冲、丁癸冲

### 地支关系
- 六合：子丑合土、寅亥合木、卯戌合火、辰酉合金、巳申合水、午未合火/土
- 六冲：子午冲、丑未冲、寅申冲、卯酉冲、辰戌冲、巳亥冲
- 三合：申子辰水、寅午戌火、巳酉丑金、亥卯未木
- 三会：寅卯辰木、巳午未火、申酉戌金、亥子丑水
- 三刑：寅巳申、丑未戌、子卯
- 六害：子未、丑午、寅巳、卯辰、申亥、酉戌
- 六破：子酉、卯午、辰丑、戌未、申巳、寅亥

## 九、四柱宫位系统

| 柱位 | 人生阶段 | 六亲 | 社会关系 |
|------|----------|------|----------|
| 年柱 | 0-15岁/童年 | 祖上、父母（父母宫） | 根基、出身 |
| 月柱 | 16-30岁/青年 | 兄弟、朋友（兄弟宫） | 事业、社会地位 |
| 日柱 | 31-45岁/中年 | 自己、配偶（夫妻宫） | 婚姻、内心 |
| 时柱 | 46岁+/晚年 | 子女、下属（子女宫） | 归宿、后代 |

**日支 = 夫妻宫（最重要的婚姻指标）**
**月柱 = 事业宫（社会地位的核心）**

## 十一、纳音与年命

年柱纳音 = 命主本命纳音，用于判断"XX命"（如"海中金命"）。
完整六十甲子纳音表见神煞速查表。

### 纳音相生相克
- 纳音相生 → 吉（如海中金生大溪水）
- 纳音相克 → 凶（如炉中火克剑锋金）
- 纳音比和 → 中性

### 年命与太岁纳音关系
- 太岁纳音生年命纳音 → 得太岁帮助
- 年命纳音克太岁纳音 → 能驾驭太岁
- 太岁纳音克年命纳音 → 被太岁压制，凶
